Deleting a Section

To delete a section you have created, access the Sections tool and follow these steps:

1.  If the CLE is not displaying the home page, click the Reset button.

2.  Select the Remove box by the section name that you wish to remove.

3.  To delete these sections, click Remove Sections. Or, to exit without removing the section(s), click Cancel.

If you have chosen to proceed, the CLE displays this message:

4.  To complete the deletion, click Remove. Or, to cancel the removal and return to the home page, click Cancel.
