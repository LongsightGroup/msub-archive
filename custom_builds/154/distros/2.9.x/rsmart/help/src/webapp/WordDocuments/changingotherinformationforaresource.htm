Changing Other Information for a Resource

Using the Resources tool, you may revise the name and administrative details (description, copyright information, availability and access information, etc.) for any resource on your site. To revise information about a resource, access the Resources tool and follow these steps:

1.  If the Resources home page is not displayed, click the Reset button.

2.  Find the name of the resource you want to work with.

3.  Click the Actions button for this resource and select Edit Details from the pulldown list that is displayed. (Note: for links to Google documents, select Edit Properties.) The CLE displays the Edit Details screen for the appropriate type of resource.

4.  Make changes as appropriate, based on its type. For information about appropriate entries for each type of resource, see Adding Resources.

5.  To save your changes, click Update. The CLE makes the changes and displays the Resources home page. Or, to return to the home page without saving these changes, click Cancel.
