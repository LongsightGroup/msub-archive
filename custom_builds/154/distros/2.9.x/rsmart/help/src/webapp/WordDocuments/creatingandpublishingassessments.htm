Creating and Publishing Assessments

There are several different approaches you can take for delivering an assessment. Your workflow will partly depend on the choices you make for the process:

1.  Create the assessment. To create an assessment, you provide a name and then choose from two methods of creation:

•      Create using assessment builder

•      Create using markup text   

At this point, you also have the option to select an “assessment type,” which is a predefined collection of settings (see Creating and Working With Assessment Types). Or, you can wait until later in the process to make your own settings selections.

2.  Add parts/questions. What you do at this stage will depend on the creation method you’re using for the assessment:

•      If you’re using the assessment builder, you can begin adding questions and/or adding parts. You’ll see an initial “Default” part displayed. You can add questions to this part only, or you can add questions to multiple parts. Add as many parts, and questions for parts, as you’d like.

•      If you’re using markup text, add all of your questions to the single text box displayed.

3.  Select/confirm settings. Once you have added parts/questions to the assessment, you can configure settings to determine how it will be delivered, presented, and submitted. Even if you opted to use an assessment type, you’ll probably want to select some settings, such as delivery dates, and you may want to confirm and/or change other settings, too.

4.  Preview and/or publish the assessment. You can preview the assessment to confirm how it will be presented to students. If you’re ready to make the assessment available to students, you can then publish it.        

Keep in mind that the assessment workflow is very flexible—these steps don’t have to occur all at once, or even in this exact order. You might, for example, create an assessment and decide on its settings first, and then later add questions and publish. Or, you may want to add questions, select settings, and then immediately publish. Once you have created an assessment and it has “pending” (unpublished) status, you may edit or add questions/parts, select or change settings, preview, and publish at any point in time.
