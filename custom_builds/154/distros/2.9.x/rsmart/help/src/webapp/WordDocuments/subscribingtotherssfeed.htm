Subscribing to the RSS Feed

If you subscribe to the RSS feed for the Podcasts tool, you will receive podcasts automatically as they are posted. To subscribe, access the tool and choose either of these options from the home page:

•      Copy the URL displayed (between the tool name and the list of podcasts) and paste it into the podcaster of your choice. The podcaster will automatically download podcasts from this tool as they become available.

•      In your browser, create a live bookmark folder for podcasts by clicking the RSS feed icon () and entering a name for the live bookmark. Your browser will automatically update this bookmark folder each time a new file is added to the Podcasts tool.
