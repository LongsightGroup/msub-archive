Importing a Wizard

To import a wizard from your computer, access the site to which you want to import it and follow these steps:

1.  Add the compressed file to Resources from your computer. For instructions, see Adding Resources.

2.  Access the Wizards tool. If the Wizards menu bar is not displayed, click the Reset button.

3.  Click the Import button on the menu bar. The CLE displays the Import Wizard screen.

4.  Click Select files. The CLE displays the Add Attachment screen.

5.  Select the compressed file for the wizard by following the instructions in Selecting an Item from Resources for Use in ePortfolios.. After you click Continue, the CLE displays the Import Wizard screen again.

6.  The name of the file you chose is displayed in the Name box.

7.  Click the Import button. The CLE imports the file from the Resources area and redisplays the Wizards home page. The imported wizard is listed there. It has the same name and instructions as it had on its original site. Or, to return to the home page without importing the file into the Forms tool, click Cancel.

Note: Regardless of the state (published or unpublished) of the wizard on the original site, it is always imported in the unpublished state. For instructions on publishing a wizard, see Previewing and Publishing a Wizard.
