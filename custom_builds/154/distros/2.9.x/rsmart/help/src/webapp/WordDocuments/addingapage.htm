Adding a Page

To add a page to your wiki, you must create a link to it from an existing page. To do so, access the Wiki tool and follow these steps:

1.  Navigate to any page in the wiki from which you want to link to the new page. Alternatively, you may use your home page as the location from which you will create the link.

2.  Enter introductory text for the link (if desired) followed by the name of the new page in brackets.

Note: The following characters cannot be used in a Wiki link or title:
    @    at sign
     :    colon
     #    hash mark
     |     pipe
     /     backslash

3.  Click Save. The CLE displays the revised page, including your introductory text and the new link.

4.  Access the new page by clicking the link for it. The CLE displays the new page.

5.  Edit the page as instructed under Editing a Wiki Page.
