Adding Sections

Note: You may add up to ten sections to a course at one time. However, when adding more than one section at a time, all of the sections must be in the same category. To add sections in different categories, add the sections one at a time.

To add sections to a course, access the Sections tool and follow these steps:

1.  If you do not see the Sections menu bar, click the Reset button.

2.  Click Add Sections on the menu bar. The CLE displays the Add Sections screen.

3.  Using the two pulldown lists, select the number of sections you want to add for a specific category.

Warning: When selecting the number of sections to add, be sure to select the exact number of sections you wish to add. The Sections tool does not add the sections until you have entered information for all of the sections you asked to create.

 

Note: The Sections tool offers the same set of features for use with all categories. So, for example, the same options are available to you for a Lab section as for a Discussion section.

The CLE displays entry forms for the number of sections you specified.

4.  Enter a title for this section in the Name box. The default name is the category name plus a number.

5.  For Section Size, check the appropriate option. If you are limiting the number of students, enter the appropriate number as well.

6.  Check the days of the week this section meets.

7.  In the from box, enter the time this section meets. Enter the time in the format hour:minutes and then check AM or PM. Enter the ending time in the same way in the to box.

8.  In the Location box, enter the building and/or room name or number (up to 20 characters) for this section.

9.  To specify additional meeting times and places for this section, click the Add day(s) with a different meeting time and/or location link. When the CLE displays an additional set of fields, enter data as instructed in steps 6 - 8

10. Repeat steps 4 - 9 for each additional section.

11. To add these sections, click Add Sections. The CLE displays the Sections home page, including the new section(s). It also displays a message stating that the section has been successfully created.

Or, to return to the home page without adding any sections, click Cancel.
