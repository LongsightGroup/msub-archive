Adding Templates

To create a template on which to base evaluations, access the Evaluation System tool and follow these steps:

1.  If you are not on the Evaluation System home page, click the Reset button to display it.

2.  In the menu bar at the top of the screen, click My Templates. The CLE displays the Evaluation Templates screen.

This screen lists templates you have already created in this site and any other site you own along with templates created by others for your use. For each template, the system displays its title, the name of the template’s owner, the date it was last updated, and links for the activities you may perform on the template.

3.  Click Add Template in the upper right quadrant of the screen. The CLE displays the Edit Template Title/Desc. screen.

4.  Enter a title for the template in the Title field. An entry in this field is required

5.  Enter explanatory text in the Description field, if desired.

6.  To proceed, click Save. The CLE displays the Edit Template screen. (Or to return to the Evaluation Templates screen without creating the template, click Cancel.)

7.  If desired, add one or more existing questions and/or headers to the template as follows:

a.  Select the Existing option from the Add Item dropdown list and click Add. The CLE displays the Choose existing items screen.

This screen lists all of the questions and headers available for use in your templates.

b.  To search for a specific item, enter a keyword into the box in the upper right and click Search.

c.  Check the box for each item you would like to add.

d.  To add the selected items to the template, click the Insert Items button at the top or bottom of the list. The CLE displays the Edit Template screen, and the selected items have been added at the bottom of the list. Or, to return to the previous screen without adding items, click Cancel.

8.  If desired, add one or more new questions and/or headers according to the instructions in Adding Questions and Headers, steps 3-9.

9.  If desired, add one or more expert questions as follows:

a.  Select the Expert option from the Add Item dropdown list and click Add. The CLE displays the Expert Items popup screen.

The top part of this screen shows the steps in selecting expert items. The lower part of the screen contains links to categories in which expert questions are available.

b.  Click the name of the desired category. The CLE displays one of two screens:

o  If the category is further subdivided into objectives, the CLE displays the links to the objectives for the category.

          In this case, click the name of the objective for a list of expert questions for that objective.



o  If the category contains no further subdivision, the CLE displays a list of expert questions within the category.

c.  Once the CLE displays a list of questions, check the box for each question you want to add.

d.  To add the selected items to the template, click the Insert Items button. The CLE displays the Edit Template screen, and the selected items have been added at the bottom of the list. Or, to return to that screen without adding items, click Cancel.

10. If desired, create groups of questions that use the same rating scale and display format. To create each group:

a.  Click the white box for a rating scale item you want to group. The CLE displays an asterisk () next to each item with the same rating scale and display style. These are the only items you may add to this group.

b.  Check boxes for each additional item you want to add to the group. You may select one, some, or all of the available items. Once you select two items, the Create Group button at the bottom of the list becomes active.

c.  Click Create Group. The CLE displays the Edit Group screen.

d.  In the Item Header Text box, enter a heading and, if desired, explanatory text that applies to this group of questions. Use the editor’s pulldown lists and buttons to format the text as appropriate. For information on using these options, see Using the Rich Text Editor.

e.  Select appropriate display options and the category for this group of questions.

f.   To complete your work, click Save item. The CLE displays the Edit Template screen. The selected questions are now grouped into a unit.

g.  You may now work with the group in a variety of ways:

•      Drag and drop the items to reorder them within the group.

•      To add a new item of the same type and format, click Add new item. To add an existing item of the same type and format, click Add existing item.

•      To remove an item from the group, click the Ungroup link for that item. To remove an item entirely, click on the “remove” icon () for that item. In the confirmation popup window that is then displayed, click Remove.

•      To hide all items in the group so that only the group header/text is displayed, click Hide child items.

11. Work with all items in the template as needed.

You have these options:

•      To see a preview of any item, click the “preview” icon () for the item.

•      To change an item, click the “edit” icon () for it and proceed as instructed in Adding Questions and Headers, steps 5-9.

•      To remove an item from the template, click the “remove” icon () for it. When the CLE displays a confirmation popup window, click Remove.

•      To rearrange items, either drag and drop them in the preferred order or use the numbered pulldown lists to change the item’s number in the list. To save the order you have established, click Save Order. To revert to the original order, click Revert Order.

•      You may also edit the header/text and settings for a group, as well as preview and/or remove the group. In addition, you may ungroup all items for a group by clicking Ungroup. To add and manage items in a group, click on Show child items if you have hidden the group’s items, and follow the instructions in step 10-g.

Tip: After creating a template, you may create an evaluation based on it by clicking the New Evaluation link to the right of the template title.
