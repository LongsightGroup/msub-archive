Live Virtual Classroom

Adding a Virtual Classroom Session

To add a virtual classroom session in Elluminate, access the Live Virtual Classroom tool and follow these steps:

1.  If the Live Virtual Classroom home page is not displayed, click the Reset button to display it.

2.  In the tool menu near the top of the screen, click Add. The CLE displays the Add Virtual Classroom Session screen.

3.  Enter information as appropriate for this session.

Field Name

Description

Name

A descriptive name for the session. An entry in this field is required.

Capacity

The maximum number of participants for this session. An entry in this field is required.

Start Time

For one-time sessions, the date and time of day at which the session is scheduled to start. For recurring sessions, the time of day only. An entry in this field is required.

End Time

For one-time sessions, the date and time of day at which the session is scheduled to end. For recurring sessions, the time of day only. An entry in this field is required.

Time Zone

The time zone associated with the specified start/end time. An entry in this field is required.

Occurrence

Whether this is a one-time session or one that will recur regularly. An entry in this field is required.

Recurrence Pattern

For recurring sessions only, the day(s) of the week on which it occurs. An entry in this field is required.

Start Date

For recurring sessions only, the date of the first session. An entry in this field is required.

End Date

For recurring sessions only, the date of the last session. An entry in this field is required.

Add to calendar

Adds an entry/entries to the Calendar tool for this/these sessions.

Add meeting date to session/recording names

Includes the date of the meeting in the name of the session recording(s).

 

4.  To save your work, click Finish. The system re-displays the tool home page, and the list of sessions now includes the session you have added. Or, to return to that screen without adding the session, click Cancel.
