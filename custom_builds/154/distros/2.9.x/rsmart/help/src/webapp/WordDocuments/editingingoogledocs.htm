Editing in Google Docs

Note: For instructions on removing CLE access permission to a Google Docs account, see Removing CLE Access to a Google Account.

You may access your Google account from the Resources tool to make changes to your Google documents. To change the content of a Google document to which you have added a link, follow these steps:

1.  If the Resources home page is not displayed, click the Reset button.

2.  Find the link for the Google document you want to change.

3.  Click the Actions button for this resource and select Edit in Google Documents from the dropdown list. The appropriate Google document will be displayed within the Resources tool. (Note: if you are not signed in to your Google account, you will first see the Google Docs sign in page. Once you sign in, the appropriate Google document will be displayed.)

4.  Once you have made and saved changes to the Google document, click the Reset button for the Resources tool. The CLE displays the Resources home page.
