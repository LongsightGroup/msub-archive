Posting Messages

You have three options for posting messages on a topic:

•      Post a simple message in which the text is not formatted and you have no attachments or other special requirements.

•      Post a message in which you have formatted text, attached a file, and/or requested email notification when the message has been posted.

•      Post a message in response to part of another message on this topic.
