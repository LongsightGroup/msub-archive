Site Editor

The Site Editor tool provides information about the site you are currently in. If you are an instructor or site organizer, it also allows you to change the content, access, organization, and structure of the site. You may also use this tool to duplicate the site and to import content from other sites you own.

Note: You may also revise your site via the Site Setup tool. For more information about the Site Setup tool, see Site Setup.

The Site Editor tool contains many useful features:

•      Edit Site Information allows you to describe your site, change the site's appearance, and specify contact information.

•      Edit Tools allows you to select tools for use in your site.

•      Manage Groups allows you to create groups of site participants.

•      Manage Access allows you to make your site available to the public and/or to students or other site participants.

•      Add Participants allows you to add users to your site and assign appropriate roles to them.

•      Edit Class Rosters allows you to add or remove class rosters for your site.

•      Duplicate Site allows you to duplicate this entire site in order to quickly create a new site.

•      Import from Site allows you to selectively reuse materials from other sites you own.

•      Import From Archive File allows you to migrate materials from an LMS, such as Blackboard or Moodle.
