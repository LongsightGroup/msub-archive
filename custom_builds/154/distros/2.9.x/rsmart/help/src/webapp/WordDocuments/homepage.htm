Gradebook

Home Page

To access this tool, click Gradebook in the tools menu on the left side of your screen. The CLE displays the Gradebook home page, which may look similar to the following:

The screen above shows the instructor's view of the Gradebook home page. If you are a student, your Gradebook home page looks different. To see the student view, refer to For Students: Viewing the Gradebook.

The home page is also called the Gradebook Items page. It lists all the items in the gradebook, including items sent from other tools (Tests & Quizzes, Assignments, or Discussion Forums), as well as items with which scores from Forums or Assignments have been associated. If items have been assigned to categories, each item is shown within its category.  

For each gradebook item, columns in the list display the following:

•      Item title

•      Class average 

•      Due date

•      Indicator showing whether the item is released to students

•      Indicator showing whether the item is included in course grade calculations (an item not included in course grade calculations is also displayed in gray, and the average grade for the item is displayed in parenthesis)

•      Sorting icons, for reordering the item (if the item is in a category, you may only reorder within the category)  

•      Source of the item, if it has been sent from another tool (Tests & Quizzes, Assignments, or Discussion Forums)

If the gradebook has categories, the Gradebook Items page also shows the class average for each category, as well as each category weight if the categories are weighted.
