Editing Templates

Note: When you edit a template as explained below, you are changing only the “master” template (that is, the template you base future evaluations on). The system does not apply your changes to the copy of the template used in any evaluation(s) you have already created. To change the copy of the template used in any particular pre-existing evaluation, see general information in Changing Evaluations and instructions in Specifying Settings and Dates.

You are not permitted to edit templates created by others.

To change a question or header, access the Evaluation System tool and follow these steps:

1.  If you are not on the Evaluation System home page, click the Reset button to display it.

2.  In the menu bar at the top of the screen, click My Templates. The CLE displays the Evaluation Templates screen.

3.  Find the template you want to change, and click the Edit link for it. The CLE displays the Edit Template screen.

4.  To change the name or description, click the Edit link to the right of the Template title, make changes as needed, and click Save. The CLE redisplays the Edit Template screen.

5.  To work with the items in the template, proceed as instructed in Adding a Template, steps 7-11.
