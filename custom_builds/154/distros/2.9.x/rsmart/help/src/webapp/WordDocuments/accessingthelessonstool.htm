Accessing the Lessons Tool

To access the Lessons tool, click Lessons in the tools menu on the left side of your screen. The CLE displays the Lessons home page.

 

If you are an instructor, you will see the home page shown here. The instructor's home page is also known as the Authoring Options screen. If you are a student, refer to For Students: Working with Lessons for more information.

The Authoring Options screen lists the modules for this course that have been created so far in Lessons. It also shows the start and end date and time for each module.
