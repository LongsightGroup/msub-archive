Opening or Downloading an Attachment or Other File

To open or download an attachment or file, simply click the file name. Your browser opens a separate window and displays the contents of the file in it. Depending on the type of file, you may be asked to open, save, or cancel in a window like this one:

Note: If your desktop does not have the program installed that is required to open a particular type of attachment, the system displays a window asking you to either select a program with which to open the file or save the file to the desktop.
