Displaying Submission Status

You can quickly check the submission status of assessments listed under Published Assessments on the Tests & Quizzes home page. For each published assessment, the CLE displays the number of students who have submitted—or are in the process of submitting—the assessment.

For additional submission information, select Scores from the Action dropdown menu for the appropriate published assessment. The CLE displays the Total Scores page.

On this page, click Submission Status in the menu bar. The CLE displays the Submission Status screen.

For each participant, you can see the person's name, user name, and role, along with the date he or she submitted the assessment. If the Date entry is blank, this participant has not submitted the assessment.
