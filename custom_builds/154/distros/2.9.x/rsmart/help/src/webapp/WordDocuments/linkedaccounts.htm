Linked Accounts

The Linked Accounts tool lets you remove CLE access permission to an external service provider, such as Google. To add a link in Resources to a Google document, you must first grant permission for the CLE to access your Google account. You can use the Linked Accounts tool to remove this permission. The tool is typically available in My Workspace.

Note: For more information about adding Google Docs links in the Resources tool, see Adding Links to Google Documents.

To access the Linked Accounts tool in My Workspace, click Linked Accounts in the tools menu on the left side of the screen. The CLE displays the tool home page.

If you have granted permission for the CLE to access your Google account, you will see “Google Docs” displayed in the Provider’s Name column.
