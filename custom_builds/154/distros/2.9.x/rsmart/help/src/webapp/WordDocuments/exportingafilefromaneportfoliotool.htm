Exporting a File from an ePortfolio Tool

The CLE allows you to export a compressed file containing a glossary, form, wizard, or other material from a CLE site to your desktop. After exporting the file, you can upload the compressed file from Resources to import it to another site.

Note: Instructions for importing a file vary slightly from tool to tool. For information on importing, see the instructions for the tool you are using.

After you click the Export link or button, the CLE displays a window like this one:

1.  Click Save to Disk.

2.  To export the compressed file named in this window, click OK. The CLE downloads the file to your standard download location and displays the home page for the tool you are working in. Or, to return to that screen without exporting the file, click Cancel.
