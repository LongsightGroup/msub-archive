Using a Template to Create a Portfolio

After you have completed the steps for Creating a Portfolio, you must enter information required by the template you’ve chosen.

Typically, you may be asked to take some or all of these steps:

•      Supply information from online forms.

•      Add attachments or links to web pages that provide supporting evidence.

•      Specify a particular level of a matrix to include in the portfolio.

Note: Depending on the template you’ve chosen, you may have to supply certain information before you will be able to share or preview your portfolio.

Tip: You may return to the portfolios hub screen from this screen at any time. To do so, click the Summary tab near the top of the screen.

Complete the screen(s) by following these steps:

1.  Enter information and attach files and other materials as instructed on the screen. In general:

•      To enter text into a data entry box, either type or copy and paste it.

•      If you have already filled out forms and stored them as form items in Resources, select the form item(s) you want from each dropdown list. To view any item, select it from the pulldown list and click the Edit Selected link. When the system displays the item, you may change it if you like.

•      Click the Create New link to add a form item for use in this portfolio.

•      To attach an item you have stored in Resources, follow the instructions in Selecting an Item from Resources for Use in ePortfolios.

•      To select supporting files and/or URLs, click the resource names in the Available Items box on the left and then click Add. The CLE adds those file names to the Selected Items box on the right.

2.  To save your work, click Save Changes. To undo all the work you have done on this screen, click Undo.

3.  To preview your portfolio at any time, click the Preview the Portfolio link in the top right quadrant of the screen. After previewing it, close the window.

4.  To return to the Portfolio home page at any time, click the Return to List link in the top right quadrant of the screen.
