Adding Participants

To add participants to your site, access the Site Editor tool and follow these steps:

1.  If you do not see the Site Editor menu bar, click the Reset button.

2.  Click Add Participants on the Site Editor menu bar. The CLE displays the Add Participants screen.

3.  To add one or more participants from within your institution, enter each person's user name or email address in the Other Official Participants box. Enter one name or address per line.

4.  To add one or more participants who are not from your institution, enter each person’s email address in the Non-Official Participants box. Enter each address on a separate line (you may also add a name after an address, in this format: email address, last name, first name).

5.  To assign all of these participants to the same role, select Assign all participants to the same role. Otherwise, select Assign each participant a role individually.

6.  To set the status for all of these participants, select either Active or Inactive. If you set the status to “inactive,” the participants will not be able to view the site or receive emails sent to the site's e-mail alias. 

7.  To continue to add these participants, click Continue. Or, to return to the Site Editor page without creating a new participant, click Cancel.

8.  If you clicked Continue, the system displays this screen:

9.  Click to assign the appropriate role to the new participant(s).

10. To continue to add these participants, click Continue. To return to the previous page without making changes, click Back. Or, to return to the Site Editor page without creating a new participant, click Cancel.

11. If you clicked Continue, the system displays this screen:

12. Select an email notification option for the new participants.

13. To continue to add these participants, click Continue. To return to the previous page without making changes, click Back. Or, to return to the Site Editor page without creating a new participant, click Cancel.

If you clicked Continue, the system displays a confirmation screen:

14. To finish adding new participants, click Finish. To return to the previous page without adding the users to your site, click Back. To return to the Site Editor home page without adding the users to your site, click Cancel.

If you clicked Finish, the CLE displays the new participants on the Site Editor home page.

Warning: If your institution’s CLE instance is integrated with a student information system (SIS), the SIS process may override the add/drop operations you perform in the Site Editor tool. Users in the Guest role are usually not affected by this automated add/drop.
