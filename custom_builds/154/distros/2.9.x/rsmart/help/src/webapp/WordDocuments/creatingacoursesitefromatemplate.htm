Creating a Course Site from a Template

To create a course site using a template, access your My Workspace site and select the Site Setup tool. Then, follow these steps:

1.  Click on the Create site from template radio button.

2.  From the dropdown menu, select the appropriate course site template.

3.  If you would like to copy the list of site participants and/or tool content from the site being used as a template, click Copy users and/or Copy content.

4.  Click Go to roster selection. The CLE displays the Course/Section(s) Selection screen.

5.  If your instance of the CLE is integrated with a student information system (SIS), the CLE also displays a list of available rosters that you may add to your site.

6.  In the Subject, Course, and Section boxes, specify subject, course, and section information for the first roster.

7.  If the site supports multiple sections of the course:

•      Select the number of rosters to add from the Add More Rosters? pulldown list. The CLE displays a set of Subject, Course, and Section boxes for each additional roster or select the rosters from the list of available rosters.

•      Enter subject, course, and section information for each roster.

8.  In the Authorizer’s username box, enter the name of the person who must authorize the addition of this course. If you have any comments or special instructions regarding your site setup request, enter them in the Special instructions box.

9.  Click Done. The CLE displays the Site Setup home page, where you will see your new site included in the list of sites.
