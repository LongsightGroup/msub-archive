Creating a Folder

The Messages tool allows you to create folders to organize your messages. To create a folder, access the Messages tool and follow these steps:

1.  If the Messages home page is not displayed, click the Reset button.

2.  Click New Folder in the menu bar near the top of the screen. The CLE displays the Create Folder screen.

3.  In the Folder Title box, enter a name for the folder.

4.  To add the folder, click Add. The CLE displays the Messages home page, and the new folder is listed on the screen. Or to return to the home page without adding a folder, click Cancel.
