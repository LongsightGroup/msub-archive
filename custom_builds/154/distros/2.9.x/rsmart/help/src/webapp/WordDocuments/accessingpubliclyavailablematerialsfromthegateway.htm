Accessing Publicly Available Materials from the Gateway

Users who own web sites (often called worksites or simply sites) in the CLE have the option of making announcements, syllabus items, files, and other resources available to the public. To access any of these publicly available items, follow these steps:

1.  In your browser’s address bar, enter the URL for the institution’s instance of the CLE. The CLE displays the gateway screen.

2.  Below the Log In button, click Search Public Courses and Projects.

The CLE displays the Search for Sites screen.

3.  If desired, use the radio buttons to select a particular type of site. For course sites, you may also select an academic term from the pulldown list.

4.  If desired, enter part of a site name in the Search for box.

5.  Click Search for Sites. The CLE displays a list of sites that match the criteria you entered.

6.  Click the name of the site for which you want to view public information. The CLE displays all publicly available information, files, links, etc., for that site.

7.  To return to the CLE gateway, use the Back button on your browser.

Note: This is one of the few places in the CLE where your browser’s Back button will work as expected. Do not use the Back button after logging into the CLE.
