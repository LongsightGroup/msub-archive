Adding SCORM Content

If your institution provides an optional SCORM player by Rustici or Icodeon, the CLE displays an additional selection in the Add pulldown list so you may add SCORM-compliant content to Resources:  

To add SCORM content, follow these steps:

1.  Select Upload Content Package from the Add pulldown list for the folder to which you want to add the SCORM content. The CLE displays a screen for uploading SCORM or IMS content packages:   

2.  Click Choose File. From your computer, select and open the appropriate file, which is typically a ZIP file.

3.  Click Upload. The CLE adds the file and displays the Resources home page.

You may access the SCORM content in any tool that is integrated with Resources.

Note: The Rustici and Icodeon SCORM players access existing SCORM content. To create the content, you will need the appropriate SCORM authoring tools.
