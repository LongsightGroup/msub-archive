Inserting a Link to an Assignment, Assessment, or Forum

Note: This procedure works with items created in the Assignments, Tests & Quizzes, and Forums tools. It cannot create links to items in other tools.

To link to an assignment, assessment, or forum in the CLE, follow these steps:

1.  In the rich text editor, highlight the text you want to link from.

2.  Click the Sakai Entity Link icon (). Your browser displays its link window.

3.  In the browser’s link window, click the Browse Server button. Your browser displays a list of group sites you own and tools within those sites that you may link to.

 

4.  Find the item you want as follows:

a.  In the left pane, click on a site name to select the site you want to select an item from.

b.  In the right pane, click the appropriate right triangle icon () to see a list of your assignments, assessments, or forums.

c.  To display a list of topics in a discussion forum, click the arrow for the forum.

5.  Click the Add icon () for the item you want to link to. The browser closes the window and, in its link window, it displays URL for the item you chose.

6.  Click OK. The CLE links the text to the item you selected.
