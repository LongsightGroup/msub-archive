Evaluation System

Viewing and Working with Evaluation Results

After an evaluation ends, the system sends an email that provides a link to a report containing evaluation results.

To view the results, follow these steps:

1.  Access the report in any of these ways:

•      To access it from the notification email you received, click the report link provided, or proceed as directed in the Alternate Instructions section at the bottom of the email.

•      Log into the CLE. From any site, access the Evaluation System tool by selecting it from the tools menu on the left side of the screen, and click the View Report link for the appropriate evaluation.

•      If you are already in the CLE and are on the My Evaluations screen, click the Report link for the appropriate evaluation (listed under Closed Evaluations).

•      If you have been given the direct link for the report, paste it into the address bar of your browser. Depending on the settings for the evaluation, you may need to log into the CLE before viewing the report.

•      If you are already in the CLE and are on the My Evaluations screen, click the Report link for the appropriate evaluation (listed under Closed Evaluations).

If the evaluation was assigned to more than one group, the CLE displays the Choose Report Groups screen.

2.  If the system is displaying the Choose Report Groups screen, check the box(es) for the group(s) for which you want to display results and click Submit. The CLE displays the report.

The report shows results for the course/group items first, followed by questions pertaining to the instructor/evaluate. If multiple instructors are being evaluated, the report contains an identical set of questions for each of them.

For each question, you see the total number of responses, the number of responses received by each answer option, and the percentage of the total represented by the answer count for each option.

If the evaluation contains essay/text answer questions, all submitted answers are listed beneath each question.
