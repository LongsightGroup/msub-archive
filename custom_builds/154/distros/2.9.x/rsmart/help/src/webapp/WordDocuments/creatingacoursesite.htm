Creating a Course Site

When you create a new course site, you have the option of using specific types of content from other sites you own. This option is handy when, for example, you want to reuse content from this course that was delivered in a previous term. You may reuse content from these tools:

•      Announcements

•      Assignments

•      Calendar

•      Chat Room

•      Discussion Forums

•      Resources, including podcasts stored in the Podcasts tool

•      Forums

•      Gradebook

•      Gradebook2

•      Lessons

•      News

•      Syllabus

•      Tests & Quizzes

•      Web Content

•      Wiki

When copying content from the Discussion Forums tool on another site, keep the following points in mind:

•      The CLE copies categories, forums, and the organization of both.

•      If forums and categories with the same titles exist in both sites, the imported content is merged with the existing content.

•      The CLE copies forum type designations (Normal, Read only, Reply only, Deny Access) to the new site.

•      Group forums are transferred as 'Normal' because new groups will be set up in new sites.

•      The CLE copies task topics and retains their attachments. Task topics are posted on the new site with the name of the importer.

•      The CLE does not copy the following: replies; Set topic as designations (Normal, Sticky, Announce, Task); and dates for Lock at and Make visible at.

When copying content to the Forums tool, be advised that you can import discussions only from this same tool (that is, Forums) on another site.

Warning: You cannot import to Forums from Message Center (an earlier tool that combined Forums and Messages).

When copying content from the Gradebook or Gradebook2 tool on another site, keep the following points in mind:

•      All student information and scores are removed.

•      Only gradebook items with data stored in the Gradebook or Gradebook2 tool will be included in the export. Items sent from the Discussion Forums, Assignments, or Test & Quizzes tool will not be included.

•      Assignment due dates are included in the export to preserve the order of the assignments. (Assignment order is determined by due date.) Assignments without due dates are appended at the end of the list in alphanumeric order. (Note: in Gradebook2, due dates are preserved, but items are not ordered by due date.)

•      Categories and weights are not included in the export.

After collecting the necessary information, choose one of the following two options for creating your site:

•      Build your own site (default)—If you have a good understanding of the tools available for use in course sites and are willing to choose tools in a later step of the site creation process, proceed with this option, as instructed in Building Your Own Course Site.

•      Create site from template—Choose this option if you are new to the CLE tools or do not want to select tools for this site from a list. Using a template eliminates the need for you to understand the full range of tools by automatically choosing a set of appropriate tools for your new site. Proceed with this option as instructed in Creating a Course Site from a Template.  

Note: Regardless of which option you choose, you may add and delete tools at any time after creating your site.
