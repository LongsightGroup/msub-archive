Selecting an Item from Resources

Note: The instructions below do not apply to adding items in ePortfolio tools. For instructions on adding items from Resources to an ePortfolio tool, see Selecting an Item from Resources for Use in ePortfolios.

Note: If you have not yet added the appropriate file to Resources, add the file or URL as instructed in Adding Files or Adding URLs (Links to Web Sites).

After the CLE displays the Add Attachments screen, you may attach files from your Resources tool. To do so, follow these steps in the “Select a resource” section of the screen:

1.  If the main folder for the appropriate site is not displayed, click the right-facing arrow () for “Show other sites.” The CLE displays folders for all sites you belong to.

2.  Find the appropriate folder and click to open it.

3.  Click the Attach a copy link for the file(s) you want to attach.

The CLE displays the file name in the Items to Attach area near the top of the screen.

4.  After you have selected all items you want to attach, click Continue.
