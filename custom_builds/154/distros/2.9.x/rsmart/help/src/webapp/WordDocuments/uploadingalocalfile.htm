Uploading a Local File

Note: The instructions given below generally apply to adding/uploading a file to the Resources tool as well as adding an attachment.

After the CLE displays the Add Attachments screen, you may attach a file from your computer by browsing and selecting a file name. To do so, follow these steps:

1.  Click Browse...

 

Your computer displays its File Upload window. That window may look something like this:

2.  Click the file you want to upload and click Open. The CLE briefly displays this message:

Processing….message

When the file has been uploaded, in the “Items to attach” area near the top of the screen, the CLE displays the name of the file with an icon to its left.

3.  If desired, view the uploaded file by clicking its name. Or, to remove it, click the Remove link.

4.  After you have selected all items you want to attach, click Continue.
