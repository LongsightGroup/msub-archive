Setting up Grading for Comments Only (No Scoring)

In the Discussion Forums tool, you can set up grading to allow instructors to give private feedback to each participant without entering a grade. This allows them to guide participants in discussions that do not count toward final grades.

To do so, set up grading by forum or topic as noted above. Be sure to set the point value to 0 (zero). The CLE will not allow scoring or count the topic or forum toward grades and will not send the item to the Gradebook and Gradebook2 tools. Your comments will be available to participants privately, as usual.
