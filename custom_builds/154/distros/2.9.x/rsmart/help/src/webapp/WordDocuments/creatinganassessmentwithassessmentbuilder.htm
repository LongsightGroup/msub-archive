Creating an Assessment with Assessment Builder

To create an assessment using assessment builder, access the home page of the Tests & Quizzes tool. Then, follow these steps:

1.  Enter a title for the assessment in the Title box.

2.  If not selected, click on the radio button for Create using assessment builder. Optionally, select an assessment type, which is a collection of predefined settings, from the Choose assessment type dropdown menu.  

Warning: If you are selecting an assessment type, be sure to review the settings for the type, to confirm they match your needs. Once you select an assessment type, you cannot change it. For more information, see Creating and Working With Assessment Types.

3.  Click the Create button. The CLE displays the Questions page.

On this page, you can begin adding questions (as instructed in Adding Questions to an Assessment) and parts (as instructed in Adding Parts to an Assessment).

You can also continue on to select/change settings for the assessment, as well as preview and publish the assessment. Or, you can click on the reset button and return to the home page, where you’ll see the assessment listed under Pending Assessments.
