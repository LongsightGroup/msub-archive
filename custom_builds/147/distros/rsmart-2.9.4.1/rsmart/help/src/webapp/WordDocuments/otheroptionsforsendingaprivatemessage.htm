Other Options for Sending a Private Message

You have other options for sending a private message:

•      To create a private message in response to a specific discussion message, access that message and click PM.

•      To send a private message from your Inbox, click newpm. When the CLE displays the New Private Message display, select the appropriate users(s) from the To user box.

•      To reply to a private message, click the Quote button on the message display. The system will copy the original text of the message into the Message body. Add your response after that text.
