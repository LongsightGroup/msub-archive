Exporting Meetings

You can export meeting information in Excel (.xls) spreadsheet format, in one of two ways:

•      On the Sign-up tool home page, click Export in the tool menu bar. The CLE displays the Export Meetings page.

Check boxes for the appropriate meetings. You can select one or multiple meetings for export, or select all meetings by checking the All box or the Check All link. Once you’ve selected meetings, click on the Export as Excel button.

•      On the Sign-up tool home page, click on a specific meeting. The CLE displays the Meeting Details page for that meeting. Click on Export as Excel in the menu bar.

With either approach, an EventsWorksheet.xls file is downloaded to your computer. The organization of the file will depend on whether you have exported a single or multiple meetings.

If you exported a single meeting, the file will include the following:

•      A tab with meeting information, such as time slots, participant sign-up and wait list, and participant comments.

•      An Attendees' Schedules tab that includes each attendee's name, user ID, email, time slot, and other information.

If you exported multiple meetings, the file will include the following:

•      An Attendee's Schedules tab that includes the same information as the export for a single meeting, but for all meetings exported.

•      A separate tab for each meeting that includes time slots, participant sign-ups and wait lists, participant comments, and other information.

The file will also include an Events Overview tab. In this tab, you’ll see the same information for the exported meetings as you would find on the Sign-up home page, as well as links to the individual meeting information tabs.

Note: If you do not have the appropriate permissions to be a meeting organizer, your export files will not include Attendee's Schedules tabs. You’ll see general meeting information, your sign-up and waitlist information, and comments you added. If the meeting organizer allowed the display of other users signed up for a meeting, you’ll see sign-up and waitlist information for those users as well.
