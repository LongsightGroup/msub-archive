Using the Rich Text Editor

Many screens in the CLE contain large text-entry boxes along with many icons and dropdown menus (located above the actual text entry area). These icons and menus make up what is known as a rich text editor. In the CLE, this editor is the CKEditor.

In some tools, you have the option of showing or hiding the rich text editor. If this option is available, the CLE displays a Show/Hide Rich-Text Editor link above the box. To show or hide the icons and pulldown lists, click the link.

The table below lists the actions you can perform and indicates the icon or dropdown menu to use for each.

Note: Some of the buttons (and their functions) in the editor are not available in all tools in the CLE. In most cases, if you see a button, you can use it.
