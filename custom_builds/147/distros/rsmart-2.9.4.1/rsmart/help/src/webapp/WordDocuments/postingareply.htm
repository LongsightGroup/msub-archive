Posting a Reply

To post a reply to a message, follow these steps:

1.  Access the Reply to Forum Conversation screen for the message. You can do so in one of three ways:

•      From the list of messages in a conversation, click the “reply” link () for the appropriate message.

•      Click on the subject for the appropriate message, so the CLE displays only that message. Then, click on Reply to This Message.

•      If you want to reply to the first message in a conversation, you can also click on Reply to Initial Message, either when viewing a particular message or viewing all messages in a conversation.

In each case, the CLE then displays the Reply to Forum Conversation screen.

 

2.  If desired, change Reply Title entry. An entry in this box is required.

3.  If desired, insert the text of the message to which you are replying by clicking on Insert original text. The CLE copies the text of that message and the name of its creator and creation date and time into the Message box.

4.  In the Message box, enter your reply. Use icons and menus to format text and include other content, as instructed in Using the Rich Text Editor.

5.  To add one or more attachments to this message, click the Add attachments button and follow the instructions in Adding an Attachment.

6.  To post the message, click the Post button. The CLE displays all messages in the conversation again, with the new message included.

If you’re posting a message to a conversation in a moderated topic, you’ll see an additional message on the Reply to Forum Conversation screen, reminding you that only you and the moderator will see your message until it is approved.
