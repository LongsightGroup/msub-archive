Managing Groups Created for a Meeting

When you add a meeting in the Sign-up tool, one option is to have groups created automatically for each time slot, as explained in step 7f of Adding a Meeting. Once the meeting is published, you can view the groups in the Site Editor tool:

•      Each group title will include the meeting title and the appropriate time slot number. The first time slot of a meeting called “Peer Assessment,” for example, would have the following group title:

SIGNUP_Peer Assessment -slot 1

•      In Site Editor, you can view group membership for time slots by clicking Groups on the tool home page. From the list of site groups, click the Membership link for the appropriate group.

•      To edit one of the groups in Site Editor, click Manage Groups in the tool menu bar and click the Edit link for the appropriate group. In the Description field, you’ll see an automatically generated message indicating the group was created via the Sign-up tool. You may change this description, as well as the group title. You may also add or remove participants for the group.

If you assigned site participants to the meeting, as explained in step 8 of Adding a Meeting, the groups will already be populated with the appropriate participants. If participants sign up for the meeting, however, you must synchronize each time slot so the corresponding group is populated with the attendees for that slot.

Warning: If you delete a time slot group manually via the Site Editor tool, group membership synchronization will not be successful, and the CLE will display an alert message. To restore a deleted time slot group, click Modify on the Meeting Details page for the meeting and then click the Publish Modification button. You may then proceed with synchronization.
