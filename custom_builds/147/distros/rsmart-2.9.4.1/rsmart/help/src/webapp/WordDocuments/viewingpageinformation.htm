Viewing Page Information

You may view information about any wiki page on your site. To do so, access the Wiki tool and follow these steps:

1.  Access the wiki page you want information on.

2.  Click the Info button on the menu bar.

3.  The CLE displays the Info screen for the page you were on.

This screen provides the information summarized in the table below.

Display

Meaning

Page Permissions

Details of the permissions given to each user role for working on this page. For more information on changing permissions, see Changing Permissions for Using the Wiki Tool.

Incoming

A list of all pages this page links to

Outgoing

A list of all pages that link to this page

Notification Preferences

Your preferences for being notified if you have put a watch on the page. To access your options, click the Edit Notification Preferences link in the second column of this row.

Feeds

Links that allow you to display a printer friendly version of the page, a non-editable HTML (public view) version, an MS Word version (.doc), a .pdf version, and an RSS feed of recent changes to the page.

Owner

The name of the page owner (usually the person who created the page)

Global Name

The full name of the page name (to be entered when linking to the page from outside the wiki)

Permission Section

The site permissions realm used for determining site permissions for accessing and working with the wiki (usually this is the default realm for the site containing the wiki)

Id

The unique ID for this wiki page

Last Edited

The date and time the page was last edited

SHA-1

The SHA-1 Digest for the page.

 

Many of the column and row headings on this screen are followed by a small question mark. For more information about the heading and the content for it, move your cursor over the heading.

4.  To return to the wiki page from which you came, click Recently visited and select the page.
