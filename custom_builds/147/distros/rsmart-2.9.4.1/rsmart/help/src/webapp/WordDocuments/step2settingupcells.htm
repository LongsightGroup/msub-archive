Step 2: Setting up Cells

To set up cells for a matrix, access the Matrices tool and follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  Find the matrix you want to work on and click the Edit link for it. The CLE displays the Edit Matrix screen.

3.  Click the desired cell.

The CLE displays the Edit Cell screen.

The CLE has automatically entered a title for this cell. The default title includes the name of both the column and row that intersect at this cell.

4.  Change the title of the cell, if needed. An entry in the Title box is required.

5.  Enter a description (objectives, etc.) of the cell, if needed.

6.  In the Initial Status box, select either Ready or Locked to indicate whether this cell is to be available to participants as soon as the matrix is published.

7.  To apply a style to this cell, see Step 1: Creating the Framework, step 4.

8.  If you do not want participants adding attachments, click the box for Disable Participant Attachments.

9.  Add the following as needed: instructions, rationale, examples of work relevant to this cell (including uploaded files), rubrics, and expectations.

10. In the Participant Forms section, you may opt to override the default forms for collecting participant work and/or reflections on their work. To do so, uncheck the appropriate box and, in the dropdown menu that is then displayed, select the appropriate form.

11. In the Feedback section, you may opt to override the default form in which reviewers will provide formative feedback to participants. To do so, uncheck the box for Use default form selected in matrix properties and, in the dropdown menu that is then displayed, select the appropriate form. In this section, you may also:

•      Override the default list of selected reviewers by unchecking the box for Use default reviewers selected in matrix properties and then clicking on the Select Reviewers link to select other reviewers.

•      Select whether participants may send feedback invitations to additional reviewers.

12. In the Evaluation section, you may opt to override the default form in which evaluators provide evaluations to participants. To do so, uncheck the box for Use default form selected in matrix properties and, in the dropdown menu that is then displayed, select the appropriate form. In this section, you may also:

•      Opt to hide evaluations from participants.

•      Override the default list of selected evaluators by unchecking the box for Use default evaluators selected in matrix properties and then clicking on the Select Evaluators link to select other evaluators.

Tip: For information on adding forms and supporting information and material to a cell, see Step 2: Adding Wizard Support.    

13. To save your work, click Save Changes. The CLE displays the Edit Matrix screen. Or, to return to this screen without saving your entries, click Cancel.

14. To set up other cells in the matrix, repeat steps 3– 9.
