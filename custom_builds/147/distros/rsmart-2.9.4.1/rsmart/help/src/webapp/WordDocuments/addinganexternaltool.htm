Site Editor

Adding an External Tool

In the Site Editor, you can add a CLE tool that has been pre-configured to provide access to an external solution utilizing LTI, an independent standard for web-based applications developed by IMS. When site members select this tool, they’ll access the external solution, either within the site or in a separate tab/window.

At most institutions, a CLE functional administrator will configure the tool for integration with an external solution, which may require an account with a third-party vendor. If you don’t see any pre-configured external tools and you have questions about their availability in your CLE instance, contact your institution’s support staff.

Tip: You can also provide access to an external solution with the Basic LTI tool.

To add a pre-configured external tool, follow these steps.

1.  From the Site Editor home page, click External Tools in the menu bar. The External Tools (IMS LTI) page displays, with a list of pre-configured external solutions.

2.  Click the Tool Configuration button. The Tool Configuration page displays. On this page, you’ll see any external tools already added to your site.

3.  Click the Create Tool Configuration button. From the dropdown menu, select the appropriate pre-configured tool.

4.  In the Tool Configuration Title field that displays, enter a name for the tool. You must complete this field.

Depending on the tool’s configuration, you may see additional settings:

a.  In the Frame Height field, optionally enter a value to change the default height (in pixels) of the frame in which the external solution displays in the CLE.

b.  Check the Launch in Popup box if you want the external solution to display to users in a separate popup window.

c.  Check the Debug Launch box if you want the CLE to display launch data before the external solution launches.

d.  In the Custom Parameters box, enter additional parameters for the external solution, as appropriate.

5.  Click Save. A confirmation message will be displayed, and you’ll see the tool listed on the Tool Configuration page.

6.  Under Site Navigation for the tool, click Add Site Link.

7.  In the Button Text field, optionally change the name that will display in the tools menu.

8.  Click Add.

You’ll see a confirmation displayed, along with a message that the navigation (tools menu) is being reset via browser window refresh. You should then be able to click on the tool name to access the external solution.
