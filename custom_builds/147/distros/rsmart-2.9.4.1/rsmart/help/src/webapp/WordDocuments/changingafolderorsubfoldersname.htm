Changing a Folder or Subfolder’s Name

To rename a folder or subfolder, access the Messages tool and follow these steps:

1.  If the Messages home page is not displayed, click the Reset button.

2.  Find the folder or subfolder you want to rename and click the Folder Settings link for it.

3.  When the CLE displays the Folder Settings screen, click the Rename Folder button. The CLE displays the Rename Messages Folder screen.

4.  Change the name in the Folder Title box. An entry in this box is required.

5.  To save your changes, click Save Settings. The CLE displays the Folder Settings screen again, and the changed folder name is shown there. Or to return to that screen without making changes, click Cancel.
