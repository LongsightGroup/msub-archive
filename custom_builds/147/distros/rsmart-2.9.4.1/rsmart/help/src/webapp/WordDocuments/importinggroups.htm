Importing Groups

You can create groups by importing a .csv file that contains group titles and memberships. To do so, follow these steps:

1.  From the Site Editor home page, click on Manage Groups in the tool menu bar. The CLE displays the Group List Page.

2.  Click on Import from file.

3.  Click the Browse button and select the appropriate .csv file from your computer.

For the import to be successful, the file should:

•      Be formatted with two columns--the first for group title, the second for user ID.

•      Contain no column headers

4.  Click Continue. The CLE displays the group titles and membership found in the file, and also indicates if a group or group membership already exists. If your file includes new members for an existing group, they will be added to the group.

Note: The CLE may display messages indicating your file is not in the proper format, or that users in the file are either not valid or not site members. Once you have made the necessary corrections, be sure to reset the Site Editor tool before attempting to import the file again.

5.  Click Import groups.

You will then see the new groups displayed on the Group List page.
