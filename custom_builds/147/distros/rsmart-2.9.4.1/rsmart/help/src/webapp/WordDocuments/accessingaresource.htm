Accessing a Resource

To access a resource, access the Resources tool and follow these steps:

1.  If the Resources home page is not displayed, click the Reset button.

2.  Locate the title of the resource you want and click it.

•      If you clicked on the name of a URL, an HTML page, or a simple text document, your browser displays the selected resource in a new window.

•      If you clicked on the name of an uploaded file, it will either display in a separate tab/window or be downloaded to your computer, depending on the file type.

•      If you clicked on the name of a folder, the CLE displays an isolated view of that folder and its contents. To return to the Resources home page, click the Reset button.

•      If you clicked on a link to a Google document, your access to the document depends on the export format chosen when the link was added. The browser will display an HTML file in a separate page in Resources. For other formats, a file is downloaded to your computer.
