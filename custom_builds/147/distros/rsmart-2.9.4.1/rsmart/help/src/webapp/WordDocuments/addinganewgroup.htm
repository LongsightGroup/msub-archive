Adding a New Group

To add a new group, follow these steps:

1.  After accessing the Group List screen, click New on the Site Editor menu bar. The CLE displays the Create New Group screen.

2.  Enter a title and description for the group.

3.  In the left box of the Membership area, highlight the participants you want to add to the group and click Add to Group.

4.  If you accidentally add a name that does not belong in the group, remove the name from the Group Member List on the right side of the screen by highlighting it and clicking Remove.

5.  To save your changes, click Update. The CLE displays the Group List screen again. Or, to return to the Group List screen without creating a group, click Cancel.

6.  To return to the Site Editor home page, click the Reset button.
