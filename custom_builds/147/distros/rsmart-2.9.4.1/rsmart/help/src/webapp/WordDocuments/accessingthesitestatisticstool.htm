Site Statistics

Accessing the Site Statistics Tool

To access the Site Statistics tool, click Site Statistics in the tools menu on the left side of the screen. The CLE displays the Site Statistics home page.

This page is also known as the Overview screen. It presents three different summaries of information: Visits, Activity, and Resources. Each summary provides key statistics for quick access. By default, the CLE displays statistics gathered in the last seven days. 

The table below explains the statistics that the CLE displays in these summaries.

 

Statistic

Description

Visits

Number of times the site was accessed.

 

Users who have visited site

Number of different users who accessed the site.

 

Site members

Total number of users enrolled in the site.

 

Members who have visited site

Number of enrolled users who accessed the site; the CLE also displays this number relative to the total number of site members, expressed as a percentage.

 

Members who not have visited site

Number of enrolled users who did not access the site; the CLE also displays this number relative to the total number of site members, expressed as a percentage.

  

Events

Number of actions that users performed on the site. A complete list of activities tracked by the Site Statistics tool is available on RSN.

 

Most active tool

The tool with which the most actions were performed; the CLE also displays this tool's activity as a percentage of total activity on the site.

 

Most active user

The user who performed the most actions on the site; the CLE also displays this user's activity as a percentage of total activity on the site.

 

Files

Number of new files (resource items) created by users on the site.

 

Files opened

Number of files (resource items) that were opened by users on the site; the CLE also displays this number relative to the total number of new files created, expressed as a percentage.

 

Most opened file

File (resource item) that was opened the most times by users on the site.

 

User who has opened the most files

Site participant who opened the most files (resource items) on the site.

 

You may also click on a statistic to access a detailed report (with the exception of "Enrolled users in site").
