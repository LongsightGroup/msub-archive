Viewing Grades for the Course to Date

When viewing course grade to date for students, keep in mind the following:

•      In the Gradebook tool, each student’s course grade is based on the points earned to date relative to the total possible points for all graded items. If the entry for a particular item is null (that is, if no grade has been entered), the point value for it is not included in the total possible points for the student’s final grade.

•      The average course grade is the average of all course grades (sum of the % course grades / total # course grades). Only students who have received grades are included in the calculation. Students whose work has not yet been graded are ignored.

To view all course grades for students, access the Gradebook tool. On the Gradebook home page, click Course Grades in the tool menu bar. The CLE displays the Course Grades screen.

This screen shows the total points possible for the course, as well as the average course grade. For each student, it also displays the following:

•      Total points earned to date (note: total points do not display if your gradebook is set up for weighted categories)

•      Course grade, as a percentage, along with the corresponding letter grade according to the grade type and scale for your gradebook (note: you can change the grade type and scale, as instructed in Setting Grade Display and Conversion Options)       

•      A Grade Override box, where you can enter a grade that overrides the course grade, as instructed in Overriding an Auto-calculated Course Grade.

You can filter the view of course grades by section or group, and you can also find a particular student by entering that student’s name in the displayed box and clicking the Find button.

Tip: To have the CLE automatically enter zeros for all null values in the gradebook and calculate the course grade, click the Set Ungraded Items to Zero button.
