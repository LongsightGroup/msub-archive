Deleting a Site

Note: In the CLE, sites are “softly” deleted, which means they remain in the system for a period of time before being permanently removed. A deleted site cannot be accessed by site participants, but it can be restored by a System Administrator if it has not yet been purged from the system. Keep in mind that institutions set their own schedules for permanently removing deleted sites.

You can use the Site Setup tool to delete sites. To do so, access the tool and follow these steps:

1.  On the Site Setup home page, check the box(es) for the site(s) you want to delete.

2.  Click Delete in the tool menu bar. The CLE displays a warning message.

3.  To delete the site(s), click Mark for deletion.

The CLE again displays the Site Setup home page, where you will no longer see the site(s) listed.
