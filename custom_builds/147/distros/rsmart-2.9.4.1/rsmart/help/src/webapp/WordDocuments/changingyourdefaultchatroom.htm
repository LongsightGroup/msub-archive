Changing Your Default Chat Room

If more than one chat room has been set up for your site, you may change your default chat room at any time. To do so, access the Chat Room tool and follow these steps:

1.  If the Chat Room home page is not displayed, click the Reset button.

2.  Click the Change Room button on the Chat Room menu bar. The CLE displays the Manage Rooms screen.

3.  Find the name of the chat room you want to set as the default, and click the Set as Default link under this name. The CLE makes this room the new default and displays the Set as Default link under the name of the other chat rooms.

4.  To display the new default chat room, click the Back to room button.
