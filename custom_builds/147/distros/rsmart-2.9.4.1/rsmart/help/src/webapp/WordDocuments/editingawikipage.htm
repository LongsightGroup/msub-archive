Editing a Wiki Page

To edit the contents of a page in your wiki, access the Wiki tool and follow these steps:

1.  Click the Edit button in the menu bar.

The CLE displays the Edit screen.

The white box contains the text for the page, along with the punctuation required in wiki markup language to produce its current formatting. To the right of this box, the CLE displays help tips that introduce the punctuation used for text formatting in wiki markup language. This region of the screen also provides a link at the bottom for more help with wiki formatting.

You have these options for changing the display:

•      To expand the content box and hide the wiki tips, click Hide hints or the  icon above the Wiki Tips header.

•      To shrink the content box and redisplay the help tips, click Show hints or the  icon again.

The formatting bar above the text box contains several icons and pulldown lists that allow you to enter formatting without using special punctuation. The following table indicates the purpose of each icon and how to use it. The icons are listed in order from left to right as they are displayed on the screen.

After clicking either icon, click the Select link for the file(s) you need and then click Continue. Note that you may attach any type of resource that you can create in Resources. Therefore, you can attach any type of file you can upload, an HTML file, or a text file that you create. If you update an image in Resources and if that image is also attached to your wiki, all copies of that image in Wiki will automatically be updated too. Similarly, if you update an attached file in Resources, the attachment will automatically be updated for wiki display, as well.

 

2.  Enter text and other content into the white text box and format it as desired.

Note: The following characters cannot be used in a Wiki link or page title:
    @    at sign
     :      colon
     #     hash mark
     |      pipe
     /      backslash

3.  To preview your changes before saving, click the Preview button near the top of the screen. To save your work, click the Save button near the bottom of the screen. The CLE saves your changes and displays the changed page. Or, to exit without saving your changes, click Cancel.
