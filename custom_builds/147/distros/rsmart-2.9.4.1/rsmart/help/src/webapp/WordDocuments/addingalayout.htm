Portfolio Layouts

Adding a Layout

Warning: If you are a system administrator, be advised that any layout that you add to the PortfolioAdmin site automatically becomes available to all sites on your CLE instance.

To add a portfolio layout to the site, follow these steps:

1.  Add the files for this layout to Resources. For information on adding resources, see Adding Files.

2.  Access the Portfolio Layouts tool.

3.  If the Portfolio Layouts home page is not displayed, click the Reset button.

4.  In the Portfolio Layouts menu bar near the top of the screen, click the Add button. The CLE displays the Add Layout screen.

5.  In the Display Name box, enter a meaningful name for the layout. This is the name that will be displayed to site participants when they create portfolios. This entry is required. If desired, enter a description of this layout in the Description box.

6.  Click the Select File link for the XHTML Layout File box and select the appropriate .xhtml file for the layout you are creating. This entry is required. For more information, see Adding an Attachment.

7.  To make a preview of the layout available to users, click the Select File link for the Preview Image box and select the appropriate file. For more information, see Adding an Attachment.

8.  To add this layout to the site, click the Add Layout button. The CLE displays the Portfolio Layouts home page. The new layout is shown in the list. Or, to return to this page without adding the new layout, click Cancel.
