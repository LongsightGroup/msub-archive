Deleting Questions and Headers

Note: When you delete an item, it is not available for use in future templates, but it continues to be available in templates that already contain it. Consequently, the deleted item still exists in all evaluations based on templates that you made that contained the item, even though the item is no longer available for use in new templates.

To remove a question or header from your evaluation items list, access the Evaluation System tool and follow these steps:

1.  If you are not on the Evaluation System home page, click the Reset button to display it.

2.  In the menu bar at the top of the screen, click My Items. The CLE displays the Items screen.

3.  Find the item you want to delete, and click the Remove link for it. The CLE displays a confirmation popup screen.

4.  To proceed with the deletion, click Remove Item. The popup screen closes, the CLE displays a message that the item has been removed, and the item is no longer listed. Or, to close the popup screen without removing the item, click the “close” icon ().
