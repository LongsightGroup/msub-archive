Performing Common Tasks

Adding an Attachment

Many tools in the CLE allow you to add an attachment. In these tools, you may add any number of attachments in a wide variety of different formats. The attachments may be any combination of files on your desktop, links to web sites, or items stored in Resources.

To add an attachment in a tool you’re using, follow these steps:

1.  Click the Add Attachments button. The CLE displays the Add Attachment page.

2.  Select an item to attach. You can do so in one of three ways:

•      To upload a file from your computer, click Browse and open the appropriate file.   

•      To attach a link, enter the appropriate URL in the URL (link to web site) box. 

•      To select an existing item in Resoures, click the Attach a copy link for the appropriate item. If you want to select a Resources item from another site, click Show other sites.

With each approach, the attached item will display under Items to attach at the top of the Add Attachment page.

3.  Click the Continue button.

You will then return to the appropriate page in the tool you’re using, and the attachment(s) will be displayed.

Note: The maximum size of an attachment is set by your system administrator. The default is 20 MB.
