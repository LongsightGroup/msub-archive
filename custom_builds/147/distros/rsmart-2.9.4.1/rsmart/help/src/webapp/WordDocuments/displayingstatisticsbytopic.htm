Displaying Statistics by Topic

To access participation statistics by topic, follow these steps:

1.  From the Forums home page, click on Statistics & Grading in the tool menu bar.

2.  Click on the Statistics & Grading by Topic link. The CLE displays a list of all topics.

You’ll see the date each topic was created, as well as the forum to which each topic belongs. You'll also see the total number of messages for each topic. You can sort by forum or topic title, date, or number of messages.

3.  To access more information about a topic, click on its title in the Topic Title column.

The CLE displays the number of messages each student has authored and read for the topic, as well as the percentage of total messages read. If the topic is already associated with a gradebook item, columns for entering scores and comments is displayed; otherwise, a Grade link displays for each student. For more information, see Grading Participation in Discussions.

Tip: You can also access user information for all messages and topics in a particular forum by clicking on its title in the Forum Title column.

4.  View statistics for a student. You can do so in one of two ways.

•      Click on the student’s name. The CLE displays a popup window with all of the student’s authored messages for the topic.

You can sort messages by date, and you can also print all the messages, by clicking the print icon in the top left corner. To close the window, click the Close Window button.

•      Click on the Details link for the student. The CLE displays all messages the student has authored and read for the topic.

For each message, you'll see the topic and forum to which it belongs, as well as its date, subject, and word count. You will also see a Grade link for each message.

For more information about viewing statistics for a particular student, see Displaying Statistics by User.
