Editing Evaluations

At any time before the status is set to “complete,” you may change evaluations you have entered for wizard pages and matrix cells. To do so, access the Evaluations tool and follow these steps:

1.  If the Evaluations home page is not displayed, click the Reset button.

2.  In the list of items awaiting evaluation, find the appropriate matrix or wizard cell or page and click its name. The CLE displays it.

3.  Use the dropdown list to select the name of the person whose matrix or wizard content you want. The CLE displays a read-only copy of the requested material.

4.  Find the evaluation you want to change and click the Edit link for it.

5.  Change the text as needed and click Save Changes.

6.  To return to the Evaluations home page, click Back to Evaluations.
