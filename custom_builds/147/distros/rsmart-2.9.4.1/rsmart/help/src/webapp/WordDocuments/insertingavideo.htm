Inserting a Video

In the rich text editor, you can insert video in .flv, .wma, .avi, or .mov format. To insert a video, follow these steps.

1.  Position the cursor where you want the video to be displayed.

1.  Click on the the Insert/Edit Movie icon (). The Movie Properties window opens.

2.  Choose one of these options:

•      To link to a video in an external URL (one not in the CLE), type or paste the URL into the URL box.

•      To link to a video stored in Resources in any site you own, click Browse Server. In the Server Browser window that opens, select the appropriate site, and then select the appropriate video by clicking on its name.

Tip: From the Server Browser window, you may upload a new video to Resources, as well as add a new folder, by clicking the Upload File icon () or the New Folder icon ().

3.  Adjust settings as appropriate:

•      To change the size of the video, enter numbers in the Width and Height boxes.

•      To set the video to begin playing automatically when users access it, click the Auto Play checkbox.

4.  Click OK. In the editor, you’ll see the inserted video.
