Creating a Matching Question

Note: You can add a variety of formatting to all text you provide in a question, by clicking on the Show/Hide Rich-Text Editor link for the appropriate text box and using the rich text editor that is displayed. 

For this type of question, students see a list of entries. For each entry, they must select a match from a dropdown menu.

You can create a question in which each entry has a unique match, or multiple entries have the same match. A question can also include “distractor” entries that have no match. For these entries, students must select “None of the Above” to answer correctly.

When you create a matching question, each entry is a “choice.” A choice is paired with the appropriate menu selection, or “match.” It can also be a distractor with no pairing. You may add multiple choice/match pairs, as well as multiple distractors.

To create a matching question, select Matching from the Insert New Question dropdown menu on the Question page, and follow these steps:

1.  In the Answer Point Value box, enter the point value you want to assign to this question. For an ungraded question, enter 0 (zero). Incorrect answers are worth negative points.

2.  In the Question Text box, enter the question and/or instructions. To add one or more attachments, click Add Attachments and follow the instructions in Adding an Attachment.

3.  Under Add/Edit Pairing and Optional Feedback, add the appropriate choice/match pairs and distractors.

 

a.  To add a new choice/match pair or a new distractor, first enter a choice in the Choice box.  

b.  If you’re adding a new choice/match pair, provide the appropriate match for the choice. From the Match dropdown menu, select a match from an existing pair, or select “new” to enter an additional match. If you’re adding a distractor, select “distractor” from the menu.  

c.  Click the Save Pairing button. The CLE saves the pair or distractor you have created.

d.  Add more pairs/distractors as necessary, by repeating steps a - c. You will then see all of your pairs/distractors listed under the Add/Edit Pairing and Optional Feedback header.

4.  To change a pair/distractor, click Edit and make changes as needed.

5.  To delete a pair/distractor, click Remove. The CLE removes the pair immediately. You will not see a warning or confirmation message first.

6.  If you want to assign the question to another part in the assessment, select the appropriate part from the Assign to Part dropdown menu.

7.  To assign this question to a question pool, select the name of the desired pool from the Assign to question pool dropdown menu.

8.  To provide question-level feedback for correct and/or incorrect answers, enter text in the boxes under Correct/Incorrect Answer Feedback.

9.  Click Save. The CLE once again displays the Questions page, where you’ll find the new question in the appropriate part and in the appropriate order. To return to the Questions page without saving the question, click Cancel.
