Creating a Project Site For Connections

To create a project site and include connections as members of the site, follow these steps:

1.  In the Profile menu bar, click Connections.

2.  On the displayed page, click the Create worksite button. The following box displays:

3.  From the connections in the Available list, select the appropriate users. Then, click on the right-facing arrow () to move them to the Selected list. To move users back to the Available list, select them and click the left-facing arrow (). You may also use the other two arrows to reorder the Selected list. 

4.  In the Worksite name field, enter a name for the project site.

5.  Click the Create button.

After refreshing your browser, you should be able to access the new site.
