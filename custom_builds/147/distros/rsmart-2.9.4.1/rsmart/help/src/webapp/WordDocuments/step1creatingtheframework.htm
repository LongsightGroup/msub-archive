Matrices > Creating a Matrix

Step 1: Creating the Framework

To create the framework for a matrix, access the Matrices tool and follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  In the Matrices menu bar near the top of the screen, click the Add button. The CLE displays the Add Matrix screen.

3.  In the appropriate boxes, enter a title for the matrix and a description of its purpose. Only the Title entry is required.

Warning: Choose the title carefully. After publishing a matrix, you cannot change its title.

4.  Choose one of the following options:

•      To apply the CLE’s default style to this matrix, skip this step and proceed directly to step 6.

•      To apply another style to this matrix, click Select Style. The CLE displays the Manage Styles screen. This screen lists the styles available for use in your matrices. For information on styles, see Styles.

5.  Click the Select link for the style you want to apply to this matrix. The CLE redisplays the Add Matrix screen. The name of the style you chose is displayed in the Style box. Or, to return to that screen without selecting a style, click Go Back.

6.  To change the default colors indicating status in this matrix, select the colors you prefer in the other boxes in the Colors and Style section.

Status

Meaning

Ready

This cell is available for site participant work.

Pending

The participant has submitted this cell for evaluation.

Completed

This cell has been evaluated.

Locked

This cell is not available for site participant work at this time.

7.  To refer to columns and rows in this matrix by other terms, enter the preferred labels in the boxes in the Structure section.

8.  To add a column, in the Structure section click Add Column. The CLE displays the Add/Edit Column screen.

9.  In the Name box, enter a name for display in the column header. Keep in mind that shorter names for columns allow for better use of screen space.

10. To select a background color for this column other than the default (gray), click the color palette icon for the Background Color box. When the CLE displays color choices, click the background color you want.

11. To select a color for the text in this column other than the default (black), click the color palette icon for the Font Color box. When the CLE displays color choices, click the text color you want.

12. To save your work on this column, click Update. The CLE displays the Add Matrix screen, including information for the column you have just created.

Or to return to the Add Matrix screen without adding this column, click Cancel.

13. Repeat steps 8 - 12 as needed to create the remaining columns for your matrix.

Warning: Think carefully about the number of columns in the matrix. After publishing a matrix, you cannot change the number of columns it contains.

14. To add a row, click Add Row. The CLE displays the Add/Edit Row screen.

15. In the Name box, enter a name for display in the row header.

16. To select a background color for this row other than the default (gray), click the color palette icon for the Background Color box. When the CLE displays color choices, click the background color you want.

17. To select a color for the text in this row other than the default (black), click the color palette icon for the Font Color box. When the CLE displays color choices, click the text color you want.

18. To save your work on this row, click Update. The CLE displays the Add Matrix screen, including information for the row you have just created.

Or to return to the Add Matrix screen without adding this row, click Cancel.

19. Repeat steps 14 - 18 as needed to create the remaining rows for the matrix.

Warning: Think carefully about the number of rows in the matrix. After publishing a matrix, you cannot change the number of rows it contains.

20. In the Progression section, click the appropriate radio button to select the way you want participants to work through the cells in the matrix.

21. In the Participant Forms section, add default forms to all cells for collecting participant work and reflections on their work. You may later change these forms for a particular cell, as instructed in Step 2: Setting up Cells.   

22. In the Feedback section, click buttons to specify how to restrict feedback options at both the cell level and the level of attached evidence items. In this section, you may also:

•      Add a form in which reviewers will provide formative feedback to participants. You may later change this form for a particular cell, as instructed in Step 2: Setting up Cells.

•      Add reviewers.

•      Select whether participants may send feedback invitations to additional reviewers.

23. In the Evaluation section, you may add a form in which evaluators will provide their evaluations to participants. You may later change this form for a particular cell, as instructed in Step 2: Setting up Cells. In this section, you may also:

•      Opt to hide evaluations from participants.

•      Add evaluators.

24. To save your work, click Create Matrix. Or, to exit without adding a new matrix, click Cancel.

•      The matrix title you specified is displayed in the top left corner of the table. If you entered a description in that step, it is displayed above the table.
