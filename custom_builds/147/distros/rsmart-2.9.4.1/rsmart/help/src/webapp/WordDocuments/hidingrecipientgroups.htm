Hiding Recipient Groups

As a site owner, you may opt to prevent users from sending messages to certain groups, by hiding those groups in the “To” field that displays when a user composes a message.

To hide recipient groups, access the Messages tool and follow these steps:

1.  On the Messages home page, click the Settings link in the menu bar. The CLE displays the Messages – Settings screen.

2.  From the Add Group dropdown menu, select the appropriate group. You'll see the group name displayed below the menu.

To hide additional groups, select again from the menu. Click the “x” icon for a group if you decide you don’t want it hidden.

3.  To save your settings, click the Save Settings button. Or to exit without changing the settings, click Cancel. The CLE displays the Messages home page.

Tip: You can set permissions so that users in certain site roles can see and select hidden groups when composing messages. See Changing Permissions for the Messages Tool.
