Web Content

The Web Content tool lets you link to a web page that can then be accessed from within the CLE. The web page can be from a web site external to the CLE, or it can be an HTML page you’ve added to a site in the CLE with the Resources tool (as instructed in Adding an HTML Page).

You can configure the tool so the web page displays either in the CLE site window or in a separate tab/window. Depending on how the CLE is configured, you may also opt to have the tool open automatically to the full width of the CLE window, with the site navigation (tool) menu on the left side of the screen minimized (see Collapsing the Tool Menu).    

There is no limit to the number of Web Content tools that can be added to a site. You can give each tool a unique name that displays in the site tool menu, as well as provide a different name that displays at the top of the tool home page. This tool is available in all site types in the CLE.
