Working on and Submitting an Assignment

To access and submit work for an assignment in the Assignments tool, select the tool and follow these steps:

1.  From the list of assignments, click on the appropriate assignment. The CLE displays the submission page for that assignment.

2.  Read the instructions carefully.

3.  Submit your work. Depending on the submission requirements set by your instructor, you may have one or more options:

a.  If an Assignment Text section is displayed, you may submit your work in the rich text editor. Use the tool menu in the editor to format text and insert content or links, as appropriate.

b.  If you are allowed to add attachments, scroll down to the Attachments section. You may add attachments in one of two ways:

•      Upload a file from your computer. Click Browse and select the appropriate item. To upload more files, click the “+” () icon. You may upload up to five files for one submission, but you must upload each file separately. To remove an additional upload selection, click the appropriate “-” () icon.

•      Attach a copy of an existing file. Click or select files from workspace or site.

 

Under Select a resource, find the appropriate file in My Workspace or another site. Click the Attach a copy link for the file. Then, click Continue. You’ll be taken back to the submission page.

c.  If you are only allowed to upload a single file, click Browse and select the appropriate file from your computer. You will not be able to upload additional files or select existing resources. 

Tip: If an assignment has not been submitted or has been saved as a draft, you may remove an attachment by clicking the Remove link for it.

4.  If your instructor requires you to submit an honor pledge, check the box for the pledge. (Note: If you attempt to submit the assignment and this box is not checked, the CLE displays an alert, and you will not be able to submit the assignment until the pledge is checked.)

5.  To preview your work, click Preview. The CLE displays the preview page.

To make further changes to the assignment, click Edit. The CLE again displays the submission page.

6.  Choose what you want to do with your submission. You can take the following actions on either the preview or the submission page:

•      Click Save Draft if you want to save your work but not yet submit. The CLE displays the Saved Assignment Confirmation page. On this page, click Back to list to return to the assignments list, where you’ll see draft-in progress status for the assignment.

•      Click Submit if you want to submit the assignment. The CLE displays the Submission Confirmation page. On this page, click Back to list to return to the assignments list, where you’ll see submitted status for the assignment, along with the date of submission.

On the submission page, you may also click Cancel. If you do, you will return to the assignments list, and your work will not be saved.

Note: Do not submit your assignment until you're sure you're finished. Once you submit, you cannot add text or attachments unless your instructor returns the assignment to you. If you’re ready to submit, be sure to click the Submit button. Otherwise, your assignment will not be submitted to your instructor.
