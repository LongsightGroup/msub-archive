Basic LTI

Note: This tool is often used to integrate the CLE with third-party solutions that require an account with the appropriate vendor. In addition, the tool is often configured by a CLE functional administrator and not individual site owners. If you’re a site owner and you plan on using this tool, contact the appropriate support staff at your institution for information and guidance.

The Basic LTI tool provides access to external solutions utilizing LTI (Learning Tools Interoperability), an independent standard for web-based applications developed by the IMS Global Learning Consortium. This tool can be configured to launch an external solution from within the CLE.

When users select the tool, they’ll access the external solution, either within the site or in a separate tab/window. The tool is typically available in course sites.

As with the Web Content tool, you may add multiple instances of Basic LTI to a site and provide a different name for each instance of the tool. You may also:

•      Configure the tool by entering a URL for the external solution, or by pasting an XML descriptor that includes the URL and other configuration information.

•      Provide both “button text” (label in tool menu) and tool title (displayed at top of tool page).

•      Route scores from the external solution to Gradebook/Gradebook2.

•      Opt to have the external solution open in a new tab/window when users access the tool.

•      Opt to have a site’s tool menu collapse automatically when users select the tool, if the external solution is opening within a site. (Note: depending on how your CLE instance is configured, this feature may not be applicable. For more information, see Collapsing the Tool Menu.)

•      Set the tool frame height if the external solution is opening within a site.

•      Opt to have launch data displayed before the external solution launches.

•      Control whether user names and/or email addresses are sent to an external solution.

•      Provide custom launch parameters.

Tip: You can also add a Basic LTI tool via the External Tools feature in the Site Editor tool. For more information, see Adding an External Tool.
