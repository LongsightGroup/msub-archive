Deleting an Event

To delete an event, access the Calendar tool and follow these steps:

1.  If you are not on the Calendar home page, click the Reset button.

2.  Open the event that you want to delete. Follow instructions in Viewing Calendar Event Detail.

3.  Click Remove event.

 

4.  After the CLE displays a confirmation message, click Remove event again.
The CLE displays the Calendar home page.
