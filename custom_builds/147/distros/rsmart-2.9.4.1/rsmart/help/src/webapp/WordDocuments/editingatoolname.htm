Editing a Tool Name

Note: Depending on how your CLE instance has been configured, you may not have this capability.

To edit a tool name using Page Order, follow these steps:

1.  On the Site Editor home page, click on Page Order in the menu bar.

2.  Find the appropriate tool in the Page Order list.

3.  Click on the “edit” icon () for the tool. You’ll see a new box displayed.

4.  In the Title field, change the tool name as appropriate. If you are editing the Web Content tool, you’ll see an additional URL field for adding or changing the target URL.

5.  Click the “checkmark” icon () to save your change.

6.  Click the Save button. The Site Editor home page is once again displayed, and you will see the new name for the tool displayed in the tools menu.
