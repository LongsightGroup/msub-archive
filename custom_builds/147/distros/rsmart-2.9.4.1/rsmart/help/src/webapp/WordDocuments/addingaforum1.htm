Adding a Forum

To add a forum, access the Forums tool and follow these steps:

1.  In the tool menu bar, click the New Forum button.

The CLE displays the Forum Settings page.

2.  In the Forum Title field, enter a title for the new forum. A title is required. The entry may be up to 75 characters long.  

3.  To provide a short description that will be displayed under the forum title on the Forums home page, enter appropriate text (up to 255 characters) in the Short Description box.

4.  To provide more information that can be accessed on the Forums home page, enter the appropriate text in the Description box. Use the options in the rich text editor menu bar to format the text as needed. Click Add attachments to attach files, as instructed in Adding an Attachment.

5.  In the Forum Posting section, make selections for the following settings:

a.  Check the box for Lock forum (disable forum postings) if you want this setting selected by default for all topics added to the forum. Note that you may uncheck the setting for any topic. If the setting is selected for a topic, users will not be able to post messages.

b.  Check the box for Moderate topics in forum if you want this setting selected by default for all topics added to the forum. In the Permissions section, you may then specify which site roles have permission to moderate messages (the permission will be inactive if you do not select this setting). Note that you may uncheck the setting for any topic.

c.  Check the box for Require users to post before reading if you want this setting selected by default for all topics added to the forum. Note that you may uncheck the setting for any topic. 6.  In the Availability section, select when the forum will be displayed to site participants:

•      If you want the forum to be displayed immediately and not be hidden at a later date, click Show - display immediately with no time restriction.

•      If you want to set a date for when the forum is displayed and hidden, click Specify Dates. Then, enter the appropriate dates in the Open Date and Close Date fields that are displayed.    

7.  In the Mark Threads as Read section, check the box for Automatically Mark All Messages in a Thread as Read if you want this setting selected by default for all topics added to the forum. If this setting is selected for a topic, all messages in a thread will be automatically marked as read when a user displays them. If the setting is not selected, a user must manually mark each message as read.

8.  In the Permissions section, make any necessary changes to the default permissions for the forum, as instructed in Changing Permissions for Forums and Topics. You can adjust permissions for all roles in your site, as well as particular groups or sections you’ve created. Keep in mind that topics added to the forum will inherit the forum’s permissions, but you may change the permissions for a specific topic. Click on the Permissions heading to expand/collapse this section.

9.  If desired, associate this forum with an existing item in Gradebook or Gradebook2 by selecting from the Gradebook Item dropdown menu. All topics you subsequently add to the forum, as well as all messages for those topics, will have this item selected.

Tip: You can always select a different associated gradebook item when selecting settings for a topic, as well as when grading (as instructed in Grading Participation in Discussions).

10. Choose one of these options:

•      To save this forum in draft form, click Save Draft. The CLE displays the Forums home page. The forum is listed there, along with the short description. The forum title is preceded by the word DRAFT. This forum is not yet displayed to site participants because it is in draft form.

•      To save this forum and add a topic to it, click Save Settings & Add Topic. The CLE displays the Topics Setting screen. For instructions on adding a topic, see Adding a Discussion Topic, steps 2 – 10.

•      To return to the Forums home page without creating this forum, click Cancel.

Tip: Site participants cannot view or access a forum until it has at least one topic in it. Even so, you do not necessarily have to add the first topic when you create the forum. You may add a topic when you create the forum or at any future time.
