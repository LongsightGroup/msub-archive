jsMath Control Panel

To access the jsMath control panel, click the jsMath button, which is located in the bottom right corner of the screen in the Wiki tool. The CLE displays the control panel:

The control panel includes links to a variety of online resources. It also has a set of buttons that let you manage the printing and displaying of mathematics.

For more information about using the control panel, click the help link in the panel's upper right corner. To collapse the control panel, click Done.

If you click the Options button in the control panel, the CLE displays the jsMath options panel:

  

This panel provides a variety of options for working with jsMath. To learn more about using this panel, click the help link next to its name.

Use the Save settings for dropdown menu to select the duration of your saved settings. Your choices range from "this session only" (default) to much longer time periods. 

Click Done to save changes and collapse the panel, and click Back to return to the control panel.
