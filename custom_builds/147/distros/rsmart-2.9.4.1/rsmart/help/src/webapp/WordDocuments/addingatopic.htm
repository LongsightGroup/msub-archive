Adding a Topic

To add a topic to a discussion forum, access the Forums tool and follow these steps:

1.  Find the forum to which you want to add the topic and click the New Topic link.

The CLE displays the Topic Settings page.

2.  In the Topic Title field, enter a name for the new topic. This box is required.

3.  Provide information about the topic:

•      To provide a short description (up to 255 characters) for display under the topic title on the Forums home page, enter text in the Short Description box.

•      To provide more information that can be accessed on the home page, enter text in the Description box. Use options in the rich text editor menu bar to format the text as needed. Click Add attachments to attach files, as instructed in Adding an Attachment.

5.  In the Topic Posting section, make selections for the following settings:

a.  Check the box for Lock Topic (disable topic postings) if you want to prevent users from posting messages for the topic.

b.  Check the box for Moderate Topic if you want to allow users to moderate messages for the topic. In the Permissions section of this page, you may then specify which site roles have permission to moderate messages (the permission will be inactive if you do not select this setting).

c.  Check the box for Require users to post before reading if you want users to post a message in order to view other posts for the topic. When users initially access the topic, they will see the following message:

 

6.  In the Availability section, select when the topic will be displayed to site participants:

•      If you want the topic to be displayed immediately and not be hidden at a later date, click Show - display immediately with no time restriction.

•      If you want to set a date for when the topic is displayed and hidden, click Specify Dates. Then, enter the appropriate dates in the Open Date and Close Date fields that are displayed.    

7.  In the Mark Threads as Read section, check the box for Automatically Mark All Messages in a Thread as Read if you want all messages in a thread  automatically marked as read when a user displays them. If the setting is not selected, a user must manually mark each message as read.

9.  If desired, associate this topic with an existing item in Gradebook or Gradebook2 by selecting from the Gradebook Item dropdown menu. All messages posted for the topic will also have this gradebook item selected.

Tip: You can always select a different associated gradebook item when grading, as instructed in Grading Participation in Discussions.

10. In the Automatically Create Topics section, choose from the following settings:

•      If you want to add only one instance of the topic, click on the radio button for Create one topic.   

•      To add multiple instances of this topic for specific groups/sections, click on the radio button for Automatically create multiple topics for groups. Then, select the appropriate groups/sections from the displayed list. An instance of the topic will be created for each selected group/section, and the default permissions will be automatically adjusted for each instance so only users in that group/section may access the topic. If you select this option, proceed to step 12.

11. In the Permissions section, make any necessary changes to the default permissions for the topic, as instructed in Changing Permissions for a Specific Topic. Click on the Permissions heading to expand/collapse this section. You can adjust permissions for all roles in your site, as well as particular groups or sections you’ve created. (Note: if you opt to have multiple instances of a topic automatically created for groups/sections, as instructed in the previous step, access each topic instance to make further changes to permissions.)  

12. Choose one of these options:

•      To save this topic and make it available to site participants, click Save. The CLE displays the Forums home page. The topic is listed there, along with its short description. When site participants access the forum, the CLE displays this topic to them.

•      To save this topic in draft form, click Save Draft. The CLE displays the Forums home page. The topic is listed there, along with its short description. The topic title is preceded by the word DRAFT. This draft is not displayed to site participants.

•      To save this topic and add another topic to the same forum, click Save Settings & Add Topic. The CLE displays the Topic Settings page again.

•      To return to the Forums home page without creating this topic, click Cancel.
