Inserting a Link to an item in Assignments, Forums or Tests & Quizzes

Note: This procedure works with items created in the Assignments, Forums, and Tests & Quizzes tools. It cannot create links to items in other tools. Keep in mind that you can only link to published items that are available to users.  

To link to an assignment, assessment, or topic/forum in the CLE, follow these steps:

1.  In the rich text editor, highlight the text you want to link from.

2.  Click the Link icon (). The Link window displays.

3.  Click the Browse Server button. In the Server Browser window that displays, you’ll see all the sites you own, as well as the tools that have items you can link to in those sites.

4.  In the left pane, select the appropriate site.

5.  In the right pane, click on the “+” icon for the appropriate tool. You’ll see all the items you can link to for that tool. If you select the Forums tool, for example, you’ll see all available forums and topics:

6.  Click on the appropriate item. In the Link window, you’ll see the link path in the URL box.

7.  Optionally, select the Target or Advanced tab to change or add settings.   

8.  Click OK. The CLE links the text to the item you selected.
