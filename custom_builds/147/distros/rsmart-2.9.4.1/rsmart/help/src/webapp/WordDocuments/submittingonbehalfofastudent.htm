Submitting on Behalf of a Student

In some cases, you may want to submit an assignment on behalf of a student. To do so, access the Assignments tool and follow these steps:

1.  On the Assignments home page, select Assignment List by Student from the View dropdown menu. The CLE displays the Assignment List by Student page.

2.  Click on the triangle icon for the appropriate student. The CLE displays a list of assignments for that student.

3.  For the appropriate assignment, click on the Submit as Student link.

4.  The CLE displays the Submit Example Assignment page, which provides information about the assignment and also indicates the student for which you are submitting.

5.  Complete and submit the assignment, as instructed in Working on and Submitting an Assignment.

On the Assignment List by Student page, the CLE will indicate that you submitted the assignment.     

Note: To submit an assignment as a student, you must grant the Submit to assignment(s) permission for the instructor role, as explained in Assigning Permissions for Using the Assignments Tool. Keep in mind that once you grant this permission, your name will be included with the list of students on the Submissions page for each assignment.
