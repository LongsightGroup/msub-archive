Adding URLs (Links to Web Sites)

To add one or more links to web sites, follow these steps:

1.  Select Add Web Links (URLs) from the Add pulldown list for the folder to which you want to add the link. The CLE displays the Add Web Links (URLs) screen.

2.  In the Web Address (URL) box, enter the URL of the web site to which you are linking.

3.  In the Website Name box, enter a name for the web site to which you are linking. The CLE will use this name for the web link. Note: if you leave this box blank, the CLE will use the web site URL for the name of the web link.

4.  Click the Add details for this item link. The CLE displays additional boxes on the screen.

   

5.  Referring to the table below, enter data in all appropriate boxes.

Item

Description

Description

Enter a description of this link.

Availability and Access

In the first subsection, click a radio button to specify whether this resource is to be available to site members only, to the public, or to members of selected groups.

•      For information on the implications of setting the publicly viewable option, see Availability and Access in the table for Adding FoldersAdding Folders .

•      If you choose the “groups” option, the CLE displays a list from which you select the appropriate groups.

In the next subsection, click a radio button to indicate one of these options:

•      The link is to be available on specific dates (use the pulldown lists or calendars to select a start date and time, an end date and time, or both).

•      The link is to be hidden from everyone other than the site organizer(s).

You may also provide metadata by clicking on Optional properties and entering information in the appropriate boxes.

6.  To add another link, click the Add Another Web Link link near the bottom of the screen and repeat steps 2 - 4 as needed.

7.  Select from the Email Notification pulldown list to indicate whether site participants are to be notified of the addition(s). For basic information on email notifications, see Specifying Email Notification Options.

8.  To add the file(s), click Web Links Now. The CLE adds the link(s) and displays the Resources home page. Or, to return to the Resources home page without adding the link(s), click Cancel.
