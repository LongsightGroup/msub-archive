Returning to a Site or Tool Home Page

No matter where you are in the CLE, you can quickly return to the home page for the site you are working in and to the home page for the tool you are working in:

•      To return to the home page for the current site, simply click Home in the tools menu on the left side of the screen.

•      To return to the home page for the tool you are currently using, click the tool name in the breadcrumb trail or click the blue Reset button () displayed to the left of the tool name near the top of the screen.
