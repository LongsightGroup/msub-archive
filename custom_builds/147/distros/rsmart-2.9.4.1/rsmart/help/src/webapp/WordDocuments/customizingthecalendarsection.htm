Customizing the Calendar Section

To customize the Calendar section, access the site home page and follow these steps:

4.  Click the Options button in the Calendar section. The CLE displays the following options:

5.  To change your calendar view, select from the View dropdown menu. You can choose to display by month (the default) or by week.

6.  Define priority for calendar events. To make an event a higher priority, select the event and click the Move up button. To make an event a lower priority, select the event and click the Move down button.

7.  Select colors for priorities by clicking on the “palette” icon () for each priority. In the Color Picker window that opens, use the dropdown menu to select a palette, and then click on the appropriate color. The HTML color code will display in the corresponding field. 

8.  To save your changes, click Update.
