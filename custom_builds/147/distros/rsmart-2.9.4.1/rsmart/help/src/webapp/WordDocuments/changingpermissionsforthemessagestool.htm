Changing Permissions for the Messages Tool

As a site owner, you may change the permission settings that determine how a user can select recipients when composing a message. Depending on these settings, a user might be able to choose an “all participants” option to select all participants in the site, or select all participants in a particular group or site role.

To change permissions in the Messages tool, access the tool and follow these steps:

1.  On the Messages home page, click on Permissions in the menu bar. The CLE displays the Permissions screen.

2.  For each site role, click appropriate check boxes to set permissions. The permissions control selection options that display in the “To” field when composing a message.

Permission

Description

Allow the "All Participants" option in the "To" field

Users can select “All Participants” to send a message to all participants in the site.

 

Allow Groups in the "To field

Users can select particular groups to send a message to all participants in those groups.

 

Allow Roles in the "To" field

Users can select particular site roles to send a message to all site participants in those roles.

 

View Hidden Groups in the "To" field

Users can see and select groups in the “To” field even if hidden, as instructed in Hiding Recipient Groups.

 

To undo any changes you’ve made prior to saving, click Undo changes.

3.  After you have changed permissions as needed, click Save. If you do not want to save your changes, click Cancel. In either case, the CLE then displays the home page.
