Inserting a Link to an External Site

To insert a link to a web site external to the CLE, first copy the URL for the site, if necessary. Then, follow these steps:

1.  In the rich text editor, highlight the text where you want to insert the link. Or, simply position the cursor in the appropriate location, and the URL will be inserted there.

2.  Click the Link icon (). The Link window displays.

3.  If necessary, use the Protocol dropdown menu to select a protocol (such as http:// or https://). In most cases, the appropriate protocol will be set automatically when you paste the URL.   

4.  Paste or enter the URL into the URL field.

5.  Optionally, click on the Target tab to change how the link opens, or click on the Advanced tab for additional link configuration options. (Note: most of these options require advanced HTML experience.)  

6.  Click OK to insert the link.
