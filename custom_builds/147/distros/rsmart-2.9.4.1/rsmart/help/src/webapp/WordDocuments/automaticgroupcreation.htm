Automatic Group Creation

You can opt to have groups created automatically, with users assigned to groups based on their site roles. To create groups with the “auto groups” option, follow these steps:

1.  From the Site Editor home page, click on Manage Groups in the tool menu bar. The CLE displays the Group List Page.

2.  Click the Auto Groups button. The CLE displays the Create New Group(s) page, with a list of all roles in your site.

3.  Select a role or roles for which you want to create groups:

•      If you want to the CLE to create several groups based on site role, select multiple roles. Each group will have the name of the selected role and include all site members in that role.

•      If you want the CLE to create a single group based on a role, select the appropriate role. The CLE displays additional options:

Be sure that the radio button is selected for Create a separate group for each role selected. The group will have the name of the selected role and include all site members in that role.

•      If you want the CLE to create subgroups for a particular role, with users randomly assigned to each subgroup, select the appropriate role. From the additional options displayed, click on the radio button for Create random groups from members with selected role(s). The CLE displays additional settings:

Choose how you want users in the role to be divided into groups, by selecting either Split by number of groups needed or Split by number of users needed per group. Enter the appropriate value for the number of groups or number of users in each group.

Enter a name for the subgroups in the Group Title field. Each subgroup will have this name followed by a number—e.g., “Subgroup-1,” Subgroup-2,” et cetera.

4.  Click the Update button. The CLE displays the Group List page, and you’ll see the newly created groups in the list of groups for your site.
