Working with a Single Citation

To work with a single citation in a list, access the Resources tool and follow these steps:

1.  If the Resources home page is not displayed, click the Reset button.

2.  Find the name of a list whose citations you want to change.

3.  Click the Actions button for this resource and select Edit Citation List from the pulldown list that is displayed. The CLE displays the Edit Citation screen.

4.  Find the citation you want to work with and choose from among these options:

•      To retrieve the citation, click the Get It! Link.

•      To view the information available for the citation, click the View Citation link. Then either return to the citation list by clicking Back or edit the citation by clicking Edit Citation; when the CLE displays the Edit Citation screen (which is nearly identical to the Add Citation screen), change data as needed and then click Save Changes.

•      To edit information for this citation, click the Edit link. When the CLE displays the Edit Citation screen, change data as needed and then click Save Changes.

•      To delete a single citation, click the remove link. The CLE removes the citation from the list.

Warning: Be sure you have selected the proper citation before clicking remove. The CLE does not display a confirmation screen before deleting the citation. Instead, it deletes the citation from the list immediately.

5.  After completing your work, click the Done Editing List button. The CLE displays the Resources home page.
