Editing a Basic LTI Tool

To edit a BLTI tool, select the tool and click on the edit icon () in the far right corner. Then, make all necessary changes to the tool’s configuration. For more information, see Configuring a Basic LTI Tool.

If you want to clear all of the tool’s existing settings, click Clear Stored Preferences.

Once you have made all necessary changes, click Update Options to save them. Or, if you don’t want to save the changes, click Cancel.
