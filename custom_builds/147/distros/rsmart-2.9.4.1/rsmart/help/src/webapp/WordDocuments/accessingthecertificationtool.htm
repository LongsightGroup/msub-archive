Accessing the Certification Tool

To access the Certification tool, select it from the tools menu on the left side of the screen. The CLE displays the tool’s home page, which may look similar to this:

On the home page, you’ll see a list of the certificates you’ve created. For each certificate, the CLE displays the following information:

•      Name

•      Description (if provided)

•      Status (active or unpublished)

•      Date and time created

•      Status URL (for active certificates)

A checkbox will be displayed for each certificate. To edit or delete a certificate, click on the checkbox and select the appropriate function in the tool menu bar.

For students, the CLE displays a Check Status link for each active certificate. When a student who has met all necessary criteria clicks on this link, it will subsequently display as Print Certificate. Students do not see a menu bar displayed on the home page.
