Customizing the Recent Chat Messages Section

To customize the Recent Chat Messages section, access the site home page and follow these steps:

1.  Click Options in the Recent Chat Messages section. You’ll see the following options:

2.  Referring to the table below, make changes as needed.

Option

Description

Number of days in the past

Enter the number of days to go back to retrieve chat messages.

Number of Chat messages

Enter the maximum number of chat messages to display.

Number of characters per Chat message

Enter the maximum number of characters to display in each chat message.

3.  To save your changes, click Update.
