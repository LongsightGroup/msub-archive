Site Editor

Using Page Order

Depending on how your instance of the CLE is configured, you may have access to the Page Order feature in the Site Editor tool. If you're a site owner, this feature lets you:

•      Change the order of tools in the tools menu

•      Edit the names of tools (note: depending on how your CLE instance is configured, you may not have this capability)

•      Remove tools

•      Hide tools, so they do not display in the tools menu but their content can still be accessed by users

•      Disable tools, so they do not display in the tools menu and their content cannot be accessed by users

To access this feature, select the Site Editor tool. From the tool menu bar, click on the Page Order link.

The CLE displays the Page Order screen.

On this screen, the CLE displays a list of all the tools currently included in your site. You'll see icons for editing, hiding/showing, disabling/enabling, and removing tools.

Note: Icons for removing the Home and Site Editor tools will not be displayed, since you may not remove these tools from your site.
