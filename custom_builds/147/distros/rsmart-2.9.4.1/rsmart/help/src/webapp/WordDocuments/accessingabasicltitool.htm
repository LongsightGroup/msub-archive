Accessing a Basic LTI Tool

Site members can access a BLTI tool by selecting the tool in the site’s tool menu. If you’re a site owner, be sure that other users know the name you have given to the tool. When users select the tool, they’ll be able to access an application outside the CLE. Depending on how the tool is configured, they will have access:

•      Within the site

•      In a new browser tab or window

Keep in mind that the external application may require login information to be provided by users.
