Adding Parts to an Assessment

An assessment can include as many parts (i.e., sections) as you’d like, but it must include at least one part. When you create an assessment, it automatically includes a part, called “Default.” You can add questions to this part, and you can also add questions to additional parts you create.

To add a part to an assessment, follow these steps:

1.  Access the Questions page for the assessment. You’ll see this page displayed when you first create an assessment using assessment builder. For any unpublished assessment, you can also access it from the Tests & Quizzes home page, by first clicking on the Working Copies tab and then selecting Edit from the Action dropdown menu for the assessment.  

2.  On the Questions page, click Add Part.

The CLE displays the Add/Edit Part screen.

3.  Enter a name for this part of the assessment in the Title box.

4.  If desired, use the Information box and enter instructions or other information for this part of the assessment.

5.  In the Type section, click to indicate whether you will author one question at a time or draw from a pool of questions. If you click Random draw from question pool:

•      Select the appropriate pool from the dropdown menu and specify the number of questions to be drawn from it.

•      If desired, override the point value assigned to these questions in the pool by entering the point value you want to assign for the assessment you are working with.

•      Click a radio button to specify the type of randomization.

6.  If you did not select random draw, proceed to the Question ordering section and click to select the order of questions.

7.  In the Metadata section, enter specific objectives, keywords, and rubrics to be used for search and categorization.

8.  To save your entries, click Save. The CLE displays the Questions screen. Or, to return to that screen without saving your entries, click Cancel.

If you clicked Save, you’ll see the part you created on the Questions page.
