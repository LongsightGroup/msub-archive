Introducing the CLE

Tools in the CLE

Dozens of tools are available for use in the CLE, and each of these tools may be used in many sites. In any given site, all available tools are listed in a menu on the left side of the screen.

Each type of site—course, project, and portfolio—has the combination of available tools best suited to the activities that are likely to take place in that kind of site. For more information, see Tools Available in My Workspace, Tools in Course Sites, Tools in Project Sites, and Tools in Portfolio Sites.
