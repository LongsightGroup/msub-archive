Editing and Deleting Feedback in Matrices

If you are a reviewer, you may change or delete feedback you have already left. To do so, access the Matrices tool and follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  Find the matrix you want to review and click its name. The CLE displays the matrix.

3.  Use the dropdown list to select the name of the person whose matrix you want. The CLE displays a read-only copy of the matrix for the user you specified.

4.  Display the contents of the appropriate cell by clicking on it.

5.  Find the feedback you want to change or delete.

6.  To change it, click the Edit link for this feedback, change as needed, and click Save Changes.

7.  To delete the feedback, click the Remove link for it. The CLE deletes the feedback immediately.

Warning: Be sure you want to delete this feedback before clicking Remove. The CLE does not display a warning message before deleting it.

8.  To return to the matrix display, click Back to the Matrix.
