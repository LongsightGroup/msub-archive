Creating a Course Site from a Template

To create a course site using a template, access My Workspace and select the Site Setup tool. Then, follow these steps:

1.  From the Site Setup home page, click New in the menu bar. The CLE displays the Site Type page.

Note: If you do not see New in the Site Setup menu bar, you do not have permission to create new sites.

2.  On this page, click on the Create site from template radio button. The CLE displays the site titles of available templates and, if included in a template, a site description.

3.  From the dropdown menu, select the appropriate course site template. From the additional dropdown menu that displays, select a term for the site.

4.  To copy the list of site participants from the site template, check the box for  Copy users. By default, the Copy content box is checked. If you do not want to copy tool content from the site template, uncheck this box.

5.  If you want the site to be published once you finish creating it, check the Publish Site box. If you don’t check this box, the site will be unpublished once you create it.

6.  Click Continue. The CLE displays the Course/Section(s) Information page.

7.  Add a roster or rosters as appropriate. Depending on how your CLE instance has been configured, you may have three roster options:

•      To select from a list of rosters provided by your institution, proceed to step 7a.

•      To find and select other available rosters from your institution, proceed to step 7b.

•      To specify your own roster information, proceed to step 7c.

a.  Select from a list of provided rosters by checking the appropriate box(es).  

Alternatively, click Add course(s) and/or section(s) not listed above if you want to select and add roster(s) not listed. Proceed to step 7b.

Warning: If you click on the Create Site without a Roster button, you cannot create the site with the site template. Instead, you must build the entire site, as instructed in Building Your Own Course Site. 

b.  Find and add available rosters by selecting from the Subject, Course, and Section dropdown menus. 

Once you have made your selections, the CLE displays additional boxes:

•      In the Authorizer’s username box, enter the name of the person who must authorize the creation of this course site.

•      If you have any comments or special instructions regarding your site creation request, enter them in the Special instructions box.

To add more rosters, click Save and add another section and repeat the above process.

If the appropriate rosters are not displayed and you want to specify your own roster information, click Still cannot find your course/section? Proceed to step 7c.

c.  Specify the appropriate roster information in the Subject, Course, and Section boxes.  

If you want to add more rosters, select the appropriate number of rosters from the Add More Rosters? dropdown menu for a display of additional information boxes.

Once you have specified all appropriate roster information:

•      In the Authorizer’s username box, enter the name of the person who must authorize the creation of this course site.

•      If you have any comments or special instructions regarding your site creation request, enter them in the Special instructions box.

8.  Click Done – go to site. The CLE displays the Site Setup home page, where you’ll see your new course site in the list of sites.

Tip: After creating a site with a template, access the Site Editor tool in the site to edit site information, add/remove tools, manage participants, or publish the site.
