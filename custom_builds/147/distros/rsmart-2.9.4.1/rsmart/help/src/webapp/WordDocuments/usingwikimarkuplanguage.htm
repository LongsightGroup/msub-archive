Using Wiki Markup Language

When you access the Edit screen, the system displays helpful Wiki Tips on the right side of the screen. Using those tips and the icons near the top of the editor, you will be able to add new pages, format text, etc.

Wiki markup language allows you to perform many other activities. To learn more about using this language, click the More Hints on Wiki formatting link in the lower right quadrant of the screen. The system displays the wiki Help Page.

The Help Page contains many helpful tips. The top part of the help display looks like this:

Each row of the table shows the punctuation you need to use in order to accomplish specific results. For example:

•      To enter a major heading, enter h1 and a space before the text of the title.

•      To create a bulleted list, enter a hyphen (–) and a space before the text for each list item.

•      To enter boldface type, enter an underscore (_) before and after the text that is to be displayed in bold.
