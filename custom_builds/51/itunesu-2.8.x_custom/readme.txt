This version of iTunesU tool will integrate with iTunesU web service to automatically 
create iTunesU course page and add proper permission to the page when the tool is firstly
launched.

Those following parameters needs customization for local installation:

iTunesU_site_id=ITUNESU_SITE_ID
iTunesU_site_sharedSecret=SECRET_KEY_VALUE
admin_credential=ADMIN_CREDENTIAL_STRING
site_maintain_credential=SITE_MAINTAINER_CREDENTIAL_STRING
site_member_credential=SITE_MEMBER_CREDENTIAL_STRING
section_handle=SECTION_HANDLE_WHERE_TO_UPLOAD_COURSE_TO
template_handle=TEMPLATE_HANDLE_WHICH_CAN_BE_USED_FOR_COURSE_ITUNESU_PAGE
