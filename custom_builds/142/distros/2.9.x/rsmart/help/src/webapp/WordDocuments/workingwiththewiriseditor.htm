Working with the WIRIS Editor

Depending on how your instance of the CLE has been configured, you may have access to the WIRIS plugin, an image-based math editor that lets you create and insert mathematical equations and formulas.

Let’s say you want to create and insert an equation with the WIRIS editor. To do so, follow these steps:

1.  In the rich text editor, position the cursor where you want the equation to be inserted.

2.  Click the WIRIS icon () in the menu bar. The WIRIS editor window is displayed.

3.  Using the WIRIS editor, create the appropriate equation. Click on the tabs displayed across the top of the editor to select from a variety of symbols and templates for expressions.

When you select a template, the editor displays the appropriate boxes for entering values. To create a fraction, for example, you would enter a value in the box for the numerator and for the denominator:

4.  Once you have created the appropriate equation, click on the Accept button. The equation is inserted in the rich text editor:

  

To edit the equation, select it in the rich text editor and click on the WIRIS icon (). In the WIRIS editor window, the equation will be displayed. Make your changes and then click the Accept button. You’ll see the changes to the equation in the rich text editor.
