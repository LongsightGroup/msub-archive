Adding a Certificate

To add a new certificate, access the Certification tool and follow these steps:

1.  In the tool menu bar, click Add. The CLE displays the following:

2.  In the Name field, enter a name for the certificate. In the Description field, optionally enter a description that students will see when they access the tool.

3.  Click the Browse button and select the appropriate PDF template from your computer. Then, click the Next button. The CLE displays a page similar to the following:

4.  Enter values for the fields. For static text that you want displayed to all students, simply enter the text. For text that will be customized for each student, choose a predefined variable from the displayed list and enter it for the appropriate field. To include a student's full name, for example, you would enter the following variable:  ${recipient.fullname}

5.  Click the Next button. The CLE displays a page for setting rules for printing the certificate. You may set multiple rules for one certificate.

6.  To create a rule, select from the dropdown menu. You have three choices:

•      Students must earn a score greater than or equal to a set amount of points for a gradebook item. Choose a gradebook item from the displayed dropdown menu and then enter the appropriate points in the Score field.

•      Students must earn a score greater than or equal to a set final course grade. Enter the appropriate points in the Score field. (Note: “final course grade” rules are based on the calculated course grade. Since this grade may change during the semester, when all items have not yet been scored, use caution when adding this type of rule.)

•      Due date has passed for a gradebook item. Choose a gradebook item from the displayed dropdown menu. Keep in mind that only items with due dates will be displayed in this menu.

Click Add. The CLE displays the rule you have added. Add more rules as necessary. You may also remove a rule by clicking the Remove link.

Note: If you have set up your gradebook for entering percentages or letter grades, keep in mind the following:

* You should not include a rule for course grade. Even if students earn the appropriate grade, the CLE will indicate the condition has not been met.

* Criteria for item scores is based on maximum points for items. If you set a rule with a score of 70 for an item that has a maximum of 70 points, for example, and the gradebook is percentage-based, students will need to earn 100 percent to meet the condition. Likewise, if the gradebook is letter grade-based, students will only meet the condition if they earn a letter grade equivalent to the points you set.    

7.  Click the Next button. The CLE displays a page similar to the following:

8.  Review all of the information and settings for the certificate. If necessary, click the Back button to make changes. Otherwise, click the Activate Certificate button.

The CLE then lists the new certificate on the tool home page. Copy all of the URL in the Status Url Link column if you want to send the status link to students, or include the link in an announcement or html page.
