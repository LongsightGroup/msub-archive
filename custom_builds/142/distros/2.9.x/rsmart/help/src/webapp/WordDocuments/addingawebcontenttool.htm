Web Content

Adding a Web Content Tool

To add a Web Content tool to a site, access the Site Editor and follow these steps:

1.  Click the Edit Tools button in the menu bar.

2.  When the CLE displays the tools list, check the empty box for Web Content and click Continue. The CLE displays the Customize Tools screen.

3.  Enter a title for the web page in the Title box and the URL for it in the URL box.

4.  To add more Web Content tools, select from the More Web Content Tools? pulldown list and repeat step 3 for each of them.

5.  To proceed, click Continue. Alternatively, to return to the Edit Tools list without adding the web page, click Back; to exit from the Site Editor without adding the web page, click Cancel.

6.  When the CLE displays the confirmation screen, click Continue.

Note: The CLE lists web pages in the order in which they were added to the site. To change the order of the list, you may rename titles and reassign the URLs.
