Exporting a Matrix

To export a matrix to your computer as a compressed file, access the Matrices tool and follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  Find the matrix you want to export.

3.  Click the Export link for it. Your browser displays its standard Opening File window.

4.  Follow the instructions in Exporting a File from an ePortfolio Tool. The CLE creates a compressed file and downloads it to your computer.
