Changing Permissions for a Specific Forum or Topic

Note: Changes that you make to a specific forum's permissions override the default settings for the permissions template for this site. Your new permissions settings for the forum apply to all topics in this forum. Changes that you make to a topic's permissions override the permissions settings in both the template for the site and the forum to which the topic belongs.

To change the default permissions settings for an individual forum (and all topics in it) or for an individual topic, access the Forums tool and follow these steps:

1.  If the Forums home page is not displayed, click the Reset button.

2.  Find the forum or topic for which you want to change permissions.

3.  Depending on whether you are changing permissions for a forum or a topic, click the Forum Settings or Topic Settings link for the appropriate item.

The CLE displays the Forum Settings screen or the Topic Settings screen.

4.  Referring to the permissions table and steps 3–6 in Changing Permissions in the Site Template, change permissions as needed.

5.  Choose one of these options:

•      To save your changes and make the forum or topic available to site participants, click Save. The CLE displays the Forums home page.

•      To save this forum or topic in draft form, click Save Draft. The CLE displays the Forums home page.

•      To save this forum or topic and add a new topic to the same forum, click Save Settings & Add Topic.

•      To return to the Forums home page without changing the default permissions, click Cancel.
