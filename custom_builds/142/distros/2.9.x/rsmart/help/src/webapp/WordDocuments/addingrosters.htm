Adding Rosters

To add a roster, follow these steps:

1.  After accessing the Edit Roster screen, click Add Roster in the Edit Roster menu bar. The CLE displays the Editing screen.

2.  Select the appropriate academic term from the Academic term pulldown list.

3.  Check the box for the appropriate class. Or, if you do not see the appropriate class listed, click Add courses(s) and/or section(s) not listed above.

Note: If your CLE is integrated with a student information system (SIS), the roster list includes all classes you are teaching.

If you selected Add courses(s) and/or section(s) not listed above, the CLE displays this screen:

Either select a subject, course, and section from the pulldown lists or—if you still cannot find the course or section you need—click Still cannot find your course/section. The CLE displays the Editing site… screen.

Enter the subject, course and section, authorizer's username, and any special instructions. After you click Continue, the CLE sends these details to your system administrator, who connects your site with student data only after confirming that you are the instructor for the course.

4.  To continue adding the roster, click Continue. Or, to return to the Edit Roster page without adding the roster, click Cancel.

If you selected a class and clicked Continue, the CLE displays a confirmation screen:

5.  To add the roster, click Add Classes. To return to the previous page without adding the roster, click Back, or to return to the Edit Roster page without saving your changes, click Cancel.

If you clicked Add Classes, the CLE adds the roster and displays the Site Editor screen with the updated information.
