Setting up Your Gradebook

The Gradebook tool allows you to customize your gradebook in these ways:

•      Specify whether grades in your gradebook are to be entered and displayed as points or as percentages.

•      Specify whether graders have the option to release grades to students for a graded item or prevent item-by-item grades from being released.

Note: This tool also allows you to control whether students can see their course grades or not. For information on setting this option, see Setting Grade Display and Conversion Options.

•      If desired, create categories within your gradebook, assign items to them, and assign percentages to the categories, in order to weight them and the items they contain.

To customize your gradebook in any of the above ways, access the Gradebook tool and follow these steps:

1.  If the Gradebook home page is not displayed, click the Reset button.

2.  Click the Gradebook Setup button in the topmost menu bar or click the Change course grade options link in the upper part of the home page. The CLE displays the Gradebook Setup screen.

3.  In the Grade Entry section, click a radio button to indicate whether grades are to be entered and displayed as point values or percentages.

4.  In the Gradebook Items Display section, choose one of these options:

•      To indicate that no gradebook items may be released to students, clear the box for Display released Gradebook Items to students.

•      To indicate that gradebook items may be released to students, check the box for Display released Gradebook Items to students.

Note: Checking this box does not automatically cause any grades to be released. Instead, it presents the option for you to release grades for any given graded item whenever you decide to do so. To release an item, you must edit the individual item as indicated in the table in Creating a Gradebook Item.

5.  In the Categories & Weighting section, specify whether you want to use categories or not and whether you want to weight categories.

•      If you check No Categories, proceed to step 6.

•      If you check Categories Only, the CLE displays the first row of a category table. Enter a category name. To add more categories, click Add a Category and enter another name. After adding all categories, proceed to step 6.

•      If you check Categories & Weighting, the CLE displays the first row of a category and weighting table. Enter a name for the first category in the Category column and specify the relative weight of this category in the % column. To add more weighted categories, click Add a Category and enter another category name and percentage. The CLE displays a running total of the percentages you have added. After adding weighted categories that add up to 100.0%, proceed to step 6.

6.  To complete your work, click Save Changes. The CLE displays a confirmation message on the screen. Or, to exit without making changes, click Cancel. The CLE displays the Gradebook home page.
