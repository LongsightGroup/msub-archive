Creating the File

Input to Feedback must be a spreadsheet in .csv format and saved on your computer. This format produces a comma-delimited file. Your spreadsheet, which may be created in Microsoft Excel, must be set up as follows:

•      The first column contains usernames for the site participants.

•      The first row contains headings. Each column must have a heading.

•      All other data cells may contain text or numbers.

•      The file must be saved in .csv format.
