Using the Matrices Tool for Site Association

In the Matrices tool, you may associate another site with your current site, so that an assignment (in the Assignments tool) in that site can be linked to a matrix cell in your current site. For users who are members of both sites, the submitted assignment is automatically added to the appropriate matrix cell.

To associate another site with your current site, follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  In the tool menu bar, click Manage Site Associations. The CLE displays the Manage Site Associations screen.

3.  Find the appropriate site in the “Sites Available for Association” list. Or, if you do not see the site listed, search for it by entering the site title or ID in the “Search for Additional Sites” field and clicking the Search button.

4.  For the appropriate site, click Add. The CLE displays the site in the list of associated sites.

Tip: If you want to select all available sites listed for association, click the Add All link.

5.  Click Save. The CLE displays the Confirm Site Association Changes screen.

 

6.  To save your changes and associate the site, click Yes. The CLE displays the Matrices home page.

If you access the associated site and select the Assignments tool, you will now have the option to link any assignment to a cell in any matrix you have published in your current site. For more information, see Linking an Assignment to a Matrix Cell.
