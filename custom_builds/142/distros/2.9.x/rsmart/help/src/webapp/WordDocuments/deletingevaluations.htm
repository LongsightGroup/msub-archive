Deleting Evaluations

At any time before the status is set to “complete,” you may delete evaluations you have entered for wizard pages and matrix cells. To do so, access the Evaluations tool and follow these steps:

1.  If the Evaluations home page is not displayed, click the Reset button.

2.  In the list of items awaiting evaluation, find the appropriate matrix or wizard you want and click its name. The CLE displays it.

3.  Use the dropdown list to select the name of the person whose matrix or wizard content you want. The CLE displays a read-only copy of the requested material.

4.  Find the evaluation you want to delete and click the Remove link for it. The CLE deletes the evaluation immediately.

Warning: Be sure you want to delete this feedback before clicking Remove. The CLE does not display a warning message before deleting it.

5.  To return to the matrix display, click Back to the Matrix.
