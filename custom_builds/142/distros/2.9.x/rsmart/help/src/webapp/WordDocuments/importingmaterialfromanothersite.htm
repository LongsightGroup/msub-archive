Site Editor

Importing Material from another Site

The Import from Site feature allows you to selectively reuse material from different sites you own.

Note: When you create a new site through the Duplicate Site function, only content from the following tools is transferred: Announcements, Assignments, Calendar, Chat Room (custom rooms), Discussion Forums (categories and forums, but not topics), Forums, Gradebook, Gradebook2, Lessons, News, Polls, Resources, Site Statistics, Syllabus, Tests & Quizzes, Web Content, and Wiki. Content from other tools is not transferred.

To import materials from a site, access the Site Editor tool and follow these steps:

1.  If you do not see the Site Editor Menu bar, click the Reset button.

2.  Click Import from Site on the menu bar. The CLE displays the Import Data screen.

3.  Select an import method.

•      If you want the imported material to replace existing material, click I would like to replace my data.

•      If you want the imported material to merge with existing material, click I would like to merge my data.

Note: You must choose the “replace my data” method if you want to import Gradebook or Gradebook2 settings from another site.

When you select an import method, the CLE displays a screen like this one:

4.  Check box(es) for site(s) from which you wish to reuse material.

•      If imported data will replace existing data, you may only select material from a single site.

•      If imported data will merge with existing data, you may select material from multiple sites. 

5.  To continue with the importing process, click Continue. Or, to return to the Site Editor page without importing materials from other sites, click Cancel.

If you clicked Continue, the CLE displays a screen like this one:

 

6.  Check the box for each tool containing material you want to reuse.

Note: If you replace existing Wiki material with imported material, the existing material is archived and can be accessed from the History link in the Wiki menu bar.

7.  To proceed, click Continue. To return to the previous page without importing, click Back. Or, to return to the Site Editor page without importing materials, click Cancel.

It may take one minute or more for the Site Editor to import material from other sites. When the process is complete, the CLE displays the Site Editor home page.
