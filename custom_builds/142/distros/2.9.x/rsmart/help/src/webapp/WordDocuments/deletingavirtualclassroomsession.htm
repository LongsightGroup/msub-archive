Deleting a Virtual Classroom Session

Note: You cannot delete a session that is due to start within the next few minutes. Also, note that if you have scheduled a recurring session, you cannot delete it while one of the sessions is ongoing.

Warning: When you delete a recording from the list, the CLE deletes both the recording and its associated session. Likewise, when you delete a session that has been recorded, both the session and the recording are deleted from the list.

To delete a scheduled session or an archived recording, access the Live Virtual Classroom tool and follow these steps:

1.  If the Live Virtual Classroom home page is not displayed, click the Reset button to display it.

2.  Find the session or recording you want to work with and click the small box in the far right column of the row. Or, to delete all sessions and recordings shown on the screen, check the Check All box in the header for the list.

Note: Be sure you have selected the right item before clicking Delete. The CLE does not display a confirmation message before deleting the item you have selected.

3.  In the tool menu near the top of the screen, click Delete. The CLE deletes this session or recording from the list.
