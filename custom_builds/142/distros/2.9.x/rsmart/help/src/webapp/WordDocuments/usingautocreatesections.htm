Using Auto Create Sections

Note: You will only have access to  “auto create sections” if your instance of the CLE has been configured for this feature. 

The “auto create sections” feature lets you add sections to which the CLE randomly assigns students. When using this feature, keep in mind the following:

•      All students in your site will be assigned to the sections.

•      Students are distributed as equally as possible, according to the number of sections you are adding. If, for example, you have ten students and you use “auto create” to add three sections, the CLE assigns four students to one section and three students to the other two sections.

•      As when adding sections manually, you may only use “auto create” to add one section category at a time.

•      Any existing sections of the same category type will be overridden by the auto created sections.

•      If you use “auto create” to add sections and then subsequently use this feature to add the same number of sections of the same type, the distribution of students in the new sections will override the existing distribution.

To add sections using the “auto create sections” feature, access the Sections tool and follow these steps.

1.  If you do not see the Sections menu bar, click the Reset button.

2.  Click Auto Create Sections in the menu bar. The CLE displays the Auto Create Sections screen.    

    

3.  From the Add dropdown menu, select the number of sections you want to add, and from the category dropdown menu, select the category type. Once you have selected from both of these menus, the CLE displays a message similar to the following:

4.  To add the sections, click Add Sections. The CLE displays the Sections home page, with the new sections listed. Or, to return to the home page without making any changes, click Cancel.

Once you have added the sections, you may edit them to add information, as instructed in Editing a Section. You may also change student assignments for the sections, as instructed in Changing Section Assignments: Student Memberships.
