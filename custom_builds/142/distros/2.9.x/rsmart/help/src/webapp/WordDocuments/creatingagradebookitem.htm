Creating a Gradebook Item

To create a gradebook item in this tool, access the Gradebook and follow these steps:

1.  If the Gradebook home page is not displayed, click the Reset button.

2.  In the menu bar in the Gradebook Items section, click Add Gradebook Item(s). The CLE displays the Add Gradebook Item(s) screen.

3.  Referring to the table below, enter information for the item.

Field Name

Entry

Title

Enter a name for the gradebook item; required; maximum length is 255 characters.

Gradebook Item Point Value or

Grade Book Item Relative Weight

You will see one field or the other, depending on the setup you have selected for your gradebook. This entry is required.

If you are entering grades as points, enter a value greater than zero; required; decimals are permitted, with up to two decimal places allowed.

If you are using categories with weighting, enter the point value of this item relative to the other items within this category.

For more information on setting up the gradebook, see Setting up Your Gradebook.

Due Date

Enter a date manually in mm/dd/yy format or click the  calendar icon to select a date; optional.

Category

Displayed only if you have set up your gradebook to include categories. Select a category from the pulldown list. The default entry is Unassigned.

Release this item to Students

Check to display this item to students; clear to hide this item from them.

Include this item in course grade calculations

Check to include; clear to calculate the final grade without considering the grade on this item.

 

4.  To add another item to the gradebook, click Add Another Gradebook Item and repeat step 3. (At this point, you may delete any item by clicking the red X to the right of its title.)

5.  To post this item or items for students, click Add Item(s). The CLE displays the Gradebook home page. The assignments list now includes the new assignment(s) and this message:

Or, to return to the Gradebook home page without creating this assignment, click Cancel.
