Editing and Deleting Feedback on Wizards

If you are a reviewer, you may change or delete feedback you have already left. To do so, access the Wizards tool and follow these steps:

1.  If the Wizards home page is not displayed, click the Reset button.

2.  Find the wizard you want to review and click its name. The CLE displays the wizard.

3.  Use the dropdown list to select the name of the person whose wizard you want. The CLE displays a read-only copy of the wizard for the user you specified.

4.  Find the feedback you want to work with (to find feedback on the wizard as a whole, scroll down to the Feedback section; to find feedback on a specific page, display the contents of the appropriate page by clicking on it).

5.  To change feedback, click the Edit link for it, change as needed, and click Save Changes.

6.  To delete the feedback, click the Remove link for it. The CLE deletes the feedback immediately.

Warning: Be sure you want to delete this feedback before clicking Remove. The CLE does not display a warning message before deleting it.

7.  To return to the wizard, click Back to the Wizard.
