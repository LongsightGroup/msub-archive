Adding the Form

To add a form, access the Forms tool and follow these steps:

1.  If you do not see the Forms menu bar, click the Reset button.

2.  Click Add on the menu bar. The CLE displays the Add Form screen.

3.  In the Name box, enter a meaningful name for the form.

Warning: Choose the name carefully. Be sure to choose a name that indicates the specific use of this form. After the form has been added to the site or published globally, its name will be visible in Resources to all site participants (or if published globally, to all participants of all sites in this CLE instance). Therefore, if the form is tailored to a particular matrix cell or wizard page, a generic name such as Reflection is misleading and makes the form seem appropriate for general use when, in fact, it is not. Use a generic name for a form only if the fields, field names, and instructions in it make the form suitable for use in a wide variety of situations.

4.  Click Select Schema File. The CLE displays the Add Attachment screen.

5.  Follow the instructions for selecting an .xsd file from the Resources tool (see Selecting an Item from Resources for Use in ePortfolios.). After you click Continue, the CLE displays the Add Form screen again. The name of the file you selected is shown in the Schema File (.xsd) box and a root node is identified in the Document Root Node pulldown list.

6.  In the Instruction box, enter appropriate instructions for filling out the form. Use the icons and pulldown lists to format the text as appropriate.

7.  Optionally, in the Advanced Option section:

•      In Alternate Form Creation Renderer box , change the layout for creating form items by specifying another .xsl file that you have created and uploaded to Resources.

•      In Alternate Form View Renderer box , change the layout for viewing form items by specifying another .xsl file that you have created and uploaded to Resources.

8.  Choose from among these options:

•      To preview the form, click Preview. The CLE displays the form as it would display when the form is in use. If desired, enter data and click Validate. After completing your work, click Return and make necessary changes or add the form.

•      To add the form to this site, click Add Form. The CLE creates the form and redisplays the Forms home page. The new form has been added to the list in an unpublished state.

•      To return to the home page without adding this form, click Cancel.
