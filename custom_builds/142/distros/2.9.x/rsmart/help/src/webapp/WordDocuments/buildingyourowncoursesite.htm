Building Your Own Course Site

To build your own course site in the CLE, access your My Workspace site and select the Site Setup tool. Then, follow these steps.

1.  Click on the course site radio button.

2.  Select a term from the Academic term dropdown list.

3.  Click Continue. The CLE displays the Course/Section(s) Selection screen.

If your instance of the CLE is integrated with a student information system (SIS), the CLE also displays a list of available rosters that you may add to your site.

4.  In the Subject, Course, and Section boxes, specify subject, course, and section information for the first roster.

5.  If the site supports multiple sections of the course:

•      Select the number of rosters to add from the Add More Rosters? pulldown list. The CLE displays a set of Subject, Course, and Section boxes for each additional roster or select the rosters from the list of available rosters.

•      Enter subject, course, and section information for each roster.

6.  In the Authorizer’s username box, enter the name of the person who must authorize the addition of this course. If you have any comments or special instructions regarding your site setup request, enter them in the Special instructions box.

7.  Click Continue. The CLE displays the Course Site Information screen: 

8.  Enter information as follows:

•      In the Site URL Alias box, optionally enter a site URL alias. This alias can be a more user-friendly URL than the default, which includes the site ID.

•      In the Description box, enter information of general interest, such as: the scope of the course, its meeting days and times, your contact information and office hours, etc. Use the icons and pulldown lists to format the text, as appropriate. For information on using these options, see Using the Rich Text Editor.

Tip: After you create the site, you may change the description from the Options button on the menu bar of the site's home page.

•      In the Short Description box, enter a shorter description (maximum 80 characters). The short description is displayed on publicly viewable sites.

•      In the Site Contact Name box, enter the first and last name of the person responsible for the site.

•      In the Site Contact Email box, enter the email address of the person responsible for the site.

Click Continue. The CLE displays the Course Site Tools screen:

9.  Check appropriate boxes to select tools to be used on your site. Or, to select all of the tools, check the box for All tools at the top of the list.

10. If you do not plan to reuse materials from other sites you own, select No, thanks and then click Continue. You may skip the next step.   

11. To reuse material from other sites you own (such as a site for this same class for a previous term):

a.  Select Yes, from these sites.

b.  Scroll through the list and select the site(s) from which you wish to reuse material.

c.  Click Continue. The CLE displays another screen.

d.  Click on the appropriate check boxes to choose the tools from which you want to reuse material. Then, click Continue.  

12. If you are not using tools that require more information, the CLE displays the Course Site Access screen. Proceed to step 13.

If the tools you selected do require more information, the CLE displays the Add Multiple Tool Instances screen. On this screen, enter the requested information for each tool.

     The type of information requested will vary, depending on the tools you have   selected. For example:

•      If you have selected the Email Archive tool, you must select a unique archive email address for the site.

•      If you have selected the News or Web Content tool, you must enter a title and a URL.

You may also add additional instances of the News or Web Content tool. To do so, select the number of tools of each type from the appropriate dropdown list and provide all appropriate information for the additional tools.

     To proceed, click Continue. The CLE displays the Course Site Access screen.

13. On the Course Site Access screen, choose how you would like the course to be accessed by users.

     Choose from these options:

     Site Status

•      To publish your site so it is available to site participants, select Publish site.

•      To keep your site unpublished so it is available only to you and other site maintainers (such as CLE Functional Administrators), select Leave as Draft – accessible only to site maintainers.

Note: By default, course site visibility is set to "public." Your course site, as well as any public content in the site, will be visible to all users upon login. 

Global Access

•      To make your site accessible only to site participants, select Limited to whom I add manually, or through automatic roster updates.   

•      To make your site accessible to all users with a CLE login ID and password, select Allow anyone to join the site with valid login id. Then, use the pulldown list to select an appropriate role for users who choose to join the site.

     Next, click Continue. The CLE displays the Confirm Your Site Setup screen.

14. Review the information on the confirmation screen. To finish creating the site, click Request Site. The CLE initially displays the following message.

The CLE then displays the Site Setup home page, with your new course site included in the list of sites.
