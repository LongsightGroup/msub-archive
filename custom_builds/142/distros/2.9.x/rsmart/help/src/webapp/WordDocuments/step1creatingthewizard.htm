Wizards > Building a Wizard

Step 1: Creating the Wizard

1.  If the Wizards home page is not displayed, click the Reset button.

2.  Click the Add button on the menu bar. The CLE displays the Add screen.

3.  Select the type of wizard you are creating by clicking the button for either sequential or hierarchical.

4.  To proceed, click Continue. Or, to return to the Wizards home page without creating a wizard, click Cancel.

If you clicked Continue, the CLE displays the Add Wizard, Step 1, screen.

Note: Although the screen titles differ depending on whether you are creating a sequential or hierarchical wizard, the fields on this screen are identical.

5.  Using the boxes shown, enter a name, a description of the wizard’s purpose, and keywords. Only the Name entry is required.

6.  Choose one of the following options:

•      To apply the CLE’s default style to this wizard, skip this step and proceed directly to step 7.

•      To apply another style to this wizard, click Select Style. The CLE displays the Manage Styles screen.

This screen lists the styles available for use in your wizards. For information on creating and working with styles, see Styles.

Click the Select link for the style you want to apply to this wizard. The CLE redisplays the Add Wizard, Step 1, screen. The name of the style you chose is displayed in the Apply a Style box. Or, to return to that screen without selecting a style, click Go Back.

7.  In the bottom half of the screen, click buttons to specify whether you want reviewer and evaluator access to be able to view wizards for participants of groups they belong to or wizards of all site participants. Also specify whether and how to restrict feedback options at both the wizard and page level and at the level of attached evidence items.

8.  To proceed, click Continue. Or, to return to the Wizards home page without creating a wizard, click Cancel.

If you clicked Continue, the CLE displays the Add Wizard, Step 2, screen.
