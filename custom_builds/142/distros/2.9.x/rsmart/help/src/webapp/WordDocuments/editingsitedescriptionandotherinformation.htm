Site Editor

Editing Site Description and Other Information

You may change the site description that is displayed on the Site Editor home page. You may also change contact information and other basic information that is displayed on the home page. To do so, access the Site Editor tool and follow these steps:

1.  If you do not see the Site Editor menu bar, click the Reset button.

2.  Click Edit Site Information on the menu bar. The CLE displays the Revising Site Information screen.

3.  In the Site URL Alias box, enter or edit a site URL alias. This alias can be a more user-friendly URL than the default, which includes the site ID.

4.  In the Description box, enter or edit a description of your site. Use the icons and pulldown lists to format the text, as appropriate. For information on using these options, see Using the Rich Text Editor.

Note: You may also edit the site description by using the Options button on the menu bar on the site home page.

5.  In the Short Description box, enter or edit a summary description of this site (up to 80 characters long). This information is displayed by the Search Public Courses and Projects function that users access via the CLE gateway page (the pre-login page).

6.  Change the site contact name and email, if appropriate.

7.  To proceed with the changes, click Continue. Or, to return to the Site Editor home page without making changes, click Cancel.

If you clicked Continue, the CLE displays a confirmation screen.

8.  To confirm the changes and return to the Site Editor home page, click Finish. To revise further, click Back. To exit without making changes, click Cancel.

Note: The site information you have entered here is displayed in the home page unless you have entered a URL in the Site Info Display field.
