Requesting that a Form Be Published Globally

If you have created a form that has general applicability, you may want to make it available for use in all sites in your CLE. However, only the CLE system administrator(s) can publish a form globally (that is, across all sites in the CLE).

To request that a form be published globally, access the Forms tool and follow these steps:

1.  If the Forms home page is not displayed, click the Reset button.

2.  Find the form you want to submit to the system administrator for global publication.

3.  Click the Publish Global link for the form. The CLE displays this confirmation screen:

4.  To proceed with the request, click Yes. The CLE displays the Forms home page. The Global State for this form has been changed to “waiting for approval.” Or, to return to the home page without submitting the request, click No.

Note: If you do not see the Suggest Global Publish link, contact your administrator and request permission to perform this action.
