Polls

If you are an instructor or site owner, the Polls tool allows you to conduct anonymous voting. The tool has the following features.

•      Each poll contains a single question. For conducting polls containing multiple questions, consider using the Tests & Quizzes tool instead. For more information on that tool, see Tests & Quizzes.

•      Users may vote once per poll.

•      You may enter any number of answer options for a question and may specify how many or few options each user may choose. You may also allow a user to choose none of the options.

•      You control the display of results to site participants.

•      You may opt to make a poll public, so users who are not site members may access the poll and vote.
