Changing Details about a Form

Using the Forms tool, you may change the name of a form and the instructions for using it. However, you cannot use this tool to change the schema file for the form itself.

To change the name or instructions for a form, access the Forms tool and follow these steps:

1.  If the Forms home page is not displayed, click the Reset button.

2.  Find the form you want to change, and click the Edit link for it. The CLE displays the Edit Form screen. It is nearly identical to the Add Form screen.

3.  Change information as needed. For more information on the fields of this screen, see Adding the Form.

4.  Choose from among these options:

•      To proceed, click Save Changes. The CLE changes the form and redisplays the Forms home page.

•      To preview and validate this form, click Preview. When the CLE displays the form, enter information and click Validate.

•      To return to the Forms home page without changing this form, click Cancel.
