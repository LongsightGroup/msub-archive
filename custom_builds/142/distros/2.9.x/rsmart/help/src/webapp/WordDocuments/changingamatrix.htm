Changing a Matrix

If you have not yet published a matrix, you may change any part of it. After a matrix has been published, however, you may change the following:

•      Description of the matrix

•      Style selected for the matrix

•      Row and column labels

•      Names and colors of rows and columns

•      Colors for ready, pending, completed, and locked cells

•      Number of rows and columns in the matrix

•      Titles and descriptions of individual cells

•      Initial status of individual cells

•      Styles selected for individual cells

•      Instructions, rationale, and examples for individual cells

•      Evaluators for individual cells

You may not revise the following in a published matrix:

•      Title of the matrix

•      Any aspect of forms added to matrix cells

To revise a matrix, access the Matrices tool and follow these steps:

1.  If the Matrices home page is not displayed, click the Reset button.

2.  Find the matrix you want to change and click the Edit link for it. The CLE displays the Edit Matrix screen.

3.  In the Matrices menu near the top of the screen, click Edit Properties. The CLE displays the Edit Matrix Properties screen.

4.  To revise a column or row, click the Edit link for it and proceed as instructed in Creating a Matrix, steps 9–12 (for columns) and steps 15–18 (for rows).

5.  To remove a column or row, click the Remove link for it. The CLE displays a remove confirmation screen.

6.  To proceed, click Continue. The CLE deletes the column (or row) and redisplays the Edit Matrix screen. Otherwise, click Cancel.

7.  To move a row or column up or down in the list, click the associated Up or Down link. The CLE reorders the columns.

8.  To change other selections and entries on this screen, proceed as instructed in Creating a Matrix.

9.  To save your work, click Save Changes. Or, to exit without adding a new matrix, click Cancel.

If you clicked Save Changes, the CLE saves your changes and displays the revised matrix on the Edit Matrix screen.
