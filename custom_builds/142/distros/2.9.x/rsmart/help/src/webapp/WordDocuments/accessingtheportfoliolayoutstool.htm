Portfolio Layouts

Accessing the Portfolio Layouts Tool

To access the tool, click Portfolio Layouts in the tools menu on the left side of the screen. The CLE displays the Portfolio Layouts home page.

This page contains a list of all portfolio layouts on the site, along with their owner and status. If the list is longer than one page, use the buttons under the Portfolio Layouts menu bar to display other pages in the list.

•      To display the first page of the list, click First.

•      To display the previous page of the list, click Previous.

•      To display the next page of the list, click Next.

•      To display the last page of the list, click Last.
