Deleting a Section

To delete a section of a module, access the Lessons tool and follow these steps:

1.  If the Lessons home page is not displayed, click the Reset button.

2.  Find the module that contains the section you want to delete and display the list of sections in it according to the instructions in Displaying the List of Sections in a Module.

3.  Check the box for the section you want to delete and click the Delete button near the top of the screen.

The CLE displays this warning message:

4.  To delete the section, click Continue. To return to the Lessons home page without deleting the module, click Cancel.

If you chose to continue, the CLE displays this message:

5.  To return to the Lessons home page, click Return to Modules.
