Deleting Messages

To delete one or more messages, either display a message you want to delete or access the folder the message(s) are in and follow these steps:

1.  Choose the applicable option:

•      If you are displaying a single message, click the Delete button.

•      If you are displaying the contents of a folder, either select all messages by clicking Check All in the header or select individual messages by checking boxes for them. Then click the Delete link above the header. The CLE moves the message(s) to the Deleted folder and displays a confirmation message.

Warning: Items in the Deleted folder have not been permanently deleted. This folder is simply a holding place for items you intend to delete.

2.  To permanently delete items from the CLE, access the Deleted folder.

3.  Either select all messages by clicking Check All in the header or select individual messages by checking boxes for them.

4.  Click the Delete icon above the header. The CLE displays a confirmation screen. The screen lists each message you have selected for permanent removal.

5.  Click the Delete Message(s) button. The CLE deletes the message(s) and displays the list of messages remaining in the Delete folder again. Or, to return to that display without deleting any messages, click Cancel.
