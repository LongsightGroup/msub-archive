Profile

Working with Messages

In the Profile tool, you may receive messages from, and send messages to, any user with whom you have a connection.

Note: To receive messages in the Profile tool, you must set the appropriate privacy control, as explained in Setting Privacy Controls.

Viewing and Replying to Messages

To view your messages, access the Profile tool. Then, click on Messages in the tool menu bar. The CLE displays a screen similar to the following:


For each item in the list, the CLE displays the subject of the initial message. The CLE also displays either the initial message, if no reply has been sent, or the most recent reply, as well as a profile link for the sender.

When you click on the subject of a message, the CLE displays the message thread, which includes the initial message and all subsequent replies, in chronological order:

 

To reply to a message, enter your reply in the text box. If necessary, put your cursor in the bottom right corner and drag to resize the box. Then, click the Send Message button. The CLE displays your reply in the message thread.

To return to your list of messages, click the My Messages button. In this list, the CLE displays your reply below the message subject, as the most recent in the thread.

Composing a Message

In the Profile tool, you may initiate a message thread by composing and sending a message to another user.

Note: You may only send messages to users with whom you have a connection, and users will only receive your messages if they have set the appropriate privacy control, as explained in Setting Privacy Controls.

To compose and send a message in the Profile tool, access the tool and follow these steps:

1.  Click on Messages in the tool menu bar.

2.  Click on the Compose message button. The CLE displays the following:

3.  Enter the name of the user to whom you want to send a message. You may only enter one user. If the user you enter is not a connection, the CLE will not accept the entry and will display the following message:

     Once you have entered a connection, the CLE displays a red “X” icon next to
     the name. Click on this icon if you want to change the user.

4.  Enter the appropriate subject and message. If necessary, put your cursor in the bottom right corner of the message box and drag to resize the box.

5.  Click the Send Message button. The CLE sends the message and displays the following:

To send another message, click Compose message. To return to your list of messages, click My messages. To return to your profile, click My profile.

Note: In the Profile tool, the CLE does not display the initial messages you send to other users. Once a user replies to a message, however, you will see your initial message as part of the message thread.
