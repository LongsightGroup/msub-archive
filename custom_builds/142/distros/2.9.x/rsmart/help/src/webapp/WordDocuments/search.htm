Performing Common Tasks

Search

The Search tool allows you to search an entire site to find documents and other materials that contain text you specify. The tool searches the content and file names of the following file types in Resources:

•      .doc (MS Word document)

•      .pdf (portable document format)

•      .ppt (MS PowerPoint presentation)

•      .xsl (MS Excel).xsd (XML schema definition for forms)

•      .html, and .xhtml

It also searches:

•      emails in the Email Archive

•      wikis

•      announcements (searches both the subject and content of announcements)

•      chat messages (searches both the header and body of chat messages)

•      titles (but not the content) of .zip (Winzip archive) and .sit (Stuffit archive) files

To access the Search tool, click Search in the Tools menu. The CLE displays the Search screen.
