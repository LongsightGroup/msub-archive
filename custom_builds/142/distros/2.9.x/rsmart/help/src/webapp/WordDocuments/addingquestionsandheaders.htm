Adding Questions and Headers

Tip: You may add questions as you create a template or add questions first and then use them in templates.

Each question or header (for a group of questions) in an evaluation is known as an “item.” To add a question or header, access the Evaluation System tool and follow these steps:

1.  If you are not on the Evaluation System home page, click the Reset button to display it.

2.  In the menu bar at the top of the screen, click My Items. The CLE displays the Items screen.

This screen lists items you have already created in this or any other site. For each item, the system displays two lines of information.

•      The top line identifies the item as a particular type of question or as a header, provides an indication of the layout of answer options (vertical or full; for questions only), provides links for activities you may perform on the item, and displays your name as its creator.

•      Indented on the second line is the text of the question or header.

3.  In the top right quadrant of the screen, select the type of item you want to add from the pulldown list.

Item Type

Description

Rating scale

A question for which all possible answers are determined by the rating scale you select

Multiple choice

A question for which you specify multiple possible answers, only one of which may be selected by each respondent

Multiple answer

A question for which you specify multiple possible answers, any or all of which may be selected by each respondent

Free text/essay question

A question that requests a text response; you specify the number of lines allowed in the response

Text header

Text that introduces a new section of the evaluation

 

4.  Click Add Item. The CLE displays the Add/Edit Item popup screen for the type of item you chose.

The fields of this screen vary somewhat, depending on the type of item you are adding.

5.  Enter the text of the new question or header in the Item Text box. Use the editor’s pulldown lists and buttons to format the text as appropriate. For information on using these options, see Using the Rich Text Editor.

6.  For a rating scale question, select the scale you want to use from the Scale pulldown list.

7.  For a multiple choice or multiple answer question, enter answer options in the Enter choices section of the screen.

•      To delete an existing option, click the Remove button for it.

•      To add a new option, click Add Scale Point. The CLE displays a blank field in which you enter appropriate text.

8.  In the Display Setting Hints section, select options to control the display of the question and its answers.

•      For a rating scale, multiple choice, or multiple answer question only, choose the display you prefer from the Scale Display Setting pulldown list.

•      For a free text/essay question only, select the maximum number of lines allowed from the Response size pulldown list.

•      To add the answer option N/A (not applicable) to the possible answers for any question, click the N/A (not applicable) box.

•      To allow respondents to enter explanatory text in response to a rating scale, multiple choice, or multiple answer question, check Include comment box.

•      Choose an appropriate category for the question or header. If this question rates the evaluate(s) or instructor(s), click Evaluatee/Instructor; otherwise, be sure that the default setting of Group/Course is selected.

Note: When an evaluation pertains to multiple instructors, the instrument contains a subhead for each evaluatee. All Evaluatee/Instructor questions are repeated under each instructor’s subhead.

9.  To save your work, click Save item. The CLE displays the Items screen, which now includes your new item. Or, to return to the Items screen without saving your work, click Cancel.
