Adding a Podcast

To add a podcast to your site, access the Podcasts tool and follow these steps:

1.  If the Podcasts home page is not displayed, click the Reset button.

2.  Click the Add button in the menu bar near the top of the screen. The CLE displays the Add Podcast screen.

 

For the Choose a file entry, click Browse and select the file you want to add. For information on uploading files, see Uploading a Local File. This entry is required.

3.  Specify a Publish Date/Time by clicking the calendar icon () and selecting the date and time at which you want this podcast to be available. An entry in this field is required.

4.  Enter a title (also required) for the podcast in the Title box. If desired, provide a description as well.

5.  To add the podcast to your site, click Add. The CLE adds the podcast and displays the Podcasts home page. To return to that page without adding the podcast, click Cancel.
