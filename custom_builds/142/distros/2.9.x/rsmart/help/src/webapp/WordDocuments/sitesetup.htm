Site Setup

The Site Setup tool allows you to create, edit and delete sites. This tool is normally available only in My Workspace. To access it, click Site Setup in the tool menu on the left side of the screen. The CLE displays the Site Setup home page.

The home page lists all the sites you belong to. You have several options for filtering, sorting, paging through, and displaying the list and searching for a particular site. For information about using these options, see Working with Lists.
