Setting Notification Preferences for the Clog Tool

To set your preference for notification of new activity in the Clog tool, first access the Preferences tool, which is typically available in My Workspace. Then, follow these steps:

1.  Click on Notifications in the Preferences tool menu bar. The CLE displays the Notifications page.

2.  Scroll down to the Blog section, where you’ll see notification options for the Clog tool.

Choose from three options:

•      To receive an email whenever a new blog post or comment is added, select Send me each notification separately.

•      To receive one email each day for all new activity in the Clog tool, select Send me one email per day summarizing all notifications.

•      To receive no email notifications at all, select Do not send me notifications.

3.  Click the Update Preferences button. The CLE displays the Preferences home page again, with a message that your changes have been updated successfully. Your preference has been applied globally to all sites with the Clog tool in which you are a participant.

If you want to override this global preference and apply a different preference for a particular site or sites, click the Add a site link. You’ll see a list of all sites with the Clog tool in which you are a participant.

Check the box for the appropriate site(s), and then click the Add Site(s) button.  

On the Notifications page, the CLE displays a separate notification dropdown menu for each selected site:

Select the appropriate notification option from the dropdown menu(s) and then click the Update Preferences button. The CLE displays the Preferences home page, with a message that your changes have been updated successfully.

To remove the notification override for a site, click the remove icon () next to the dropdown menu for that site.
