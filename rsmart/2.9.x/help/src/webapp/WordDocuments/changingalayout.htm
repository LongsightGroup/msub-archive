Portfolio Layouts

Changing a Layout

You may change the title and description for a layout. You may also attach different files. To change a layout, access the Portfolio Layouts tool and follow these steps:

1.  If the Portfolio Layouts home page is not displayed, click the Reset button.

2.  Click the Edit link for the layout you want to change. The CLE displays the Edit Layout screen. It is nearly identical to the Add Layout screen.

3.  Change entries in the Display Name and Description boxes, as needed.

4.  To change the layout or preview file, click the Select File link for the appropriate box and proceed as described in Adding an Attachment. After you click Continue, the CLE displays the Add Layout screen again. The name of the file you selected is shown in the box.

5.  Click the Save Changes button. The CLE makes the changes and displays the list on the Portfolio Layouts home page. Or, to return to the home page without changing the layout, click Cancel.
