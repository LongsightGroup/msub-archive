Site Editor

Managing Access

The Site Editor allows you to control access to your site. To manage access to the site, go to the Site Editor tool and follow these steps:

1.  If you do not see the Site Editor menu bar, click the Reset button.

2.  Click Manage Access on the menu bar. The CLE displays the Change Access screen

3.  Under “Site Status,” choose one of the following options:

•      To publish your site so it is available to students and other site participants, select Publish site.

•      To keep your site unpublished so it is available only to you and other site maintainers (such as CLE Functional Administrators), select Leave as Draft – accessible only to site maintainers. Other site participants will not have access to the site.

4.  If you are managing access for a project site, under “Site Visibility,” choose one of the following options:

•      To keep your site visible only to site participants, select Private.

•      To display your site in a list of public sites, so users may search for it on the login page and also access any content on the site you have made public, select Display in public site list.  

Note: By default, course and portfolio site visibility is set to "public." Your course or portfolio site, as well as any public content in the site, will be visible to all users upon login.

5.  Under “Global Access,” choose one of the following options:

•      To make your site accessible only to site participants, select Limited to whom I add manually, or through automatic roster updates.

•      To make your site accessible to all users with a CLE login ID and password, select Allow anyone to join the site with valid login id. Then, use the dropdown menu to select an appropriate role for users who choose to join the site.

If the default role is not appropriate for a user, you may change that user’s role at any time, as instructed in Changing a Participant's Role and Status.

Warning: It is not advisable to make a course site joinable.

6.  To save your changes, click Update. The CLE displays the Site Editor home page again. Or, to return to the Site Editor home page without making the changes, click Back.
