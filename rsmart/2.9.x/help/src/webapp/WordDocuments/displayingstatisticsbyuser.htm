Displaying Statistics by User

To access participation statistics by user for all forums and topics, follow these steps:

1.  From the Forums home page, click on Statistics & Grading in the tool menu bar. The CLE displays the Statistics & Grading by User page.

On this page, you’ll see a list of all site participants. For each student, you’ll see the total number of messages authored, read, and unread, as well as the percentage of total messages that have been read. You may sort the information by clicking on any column header.  

2.  To access more information about a particular student, click on the student’s user name. The CLE displays all messages authored and read by the student.

For each message, you'll see the topic and forum to which it belongs, as well as its date, subject, and word count. You will also see a Grade link for each message. For more information, see Grading Participation in Discussions.

3.  To view an authored message in isolation, click on its subject link.

4.  To view an authored message highlighted within its conversation, click on the display in conversation link for that message. 

5.  For a display of all authored messages at once, as well as a Grade link for each message, click Show Full Text for All Authored Messages.

When viewing all authored messages or a specific authored message, you can navigate to the list of all authored and read messages for the student by clicking the Return to Statistics button for that student.

You can also navigate to the complete list of messages for other students by clicking on the <Previous Participant and Next Participant> buttons, or by clicking Statistics & Grading in the navigation breadcrumb trail.
