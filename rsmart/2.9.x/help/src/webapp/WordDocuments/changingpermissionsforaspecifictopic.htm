Changing Permissions for a Specific Topic

Note: Changes to a specific topic's permissions override the permissions of the forum to which it belongs, as well as the permissions in the settings template.

To change the permissions for a specific topic, access the Forums tool and follow these steps:

1.  On the Forums home page, Click the Topic Settings link for the appropriate topic. The CLE displays the Topic Settings page for the topic.

2.  Referring to the instructions and permissions table in step 2 of Changing Permissions in the Settings Template, change permissions as needed.

3.  Choose one of these options:

•      To save your changes and make the topic available to site participants, click Save. The CLE displays the Forums home page.

•      To save the topic in draft form, click Save Draft. The CLE displays the Forums home page.

•      To save the topic and add a new topic to the same forum, click Save Settings & Add Topic.

•      To return to the Forums home page without changing the default permissions, click Cancel.
