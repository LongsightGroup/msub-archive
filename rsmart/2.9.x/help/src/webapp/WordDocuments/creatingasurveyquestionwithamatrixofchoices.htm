Creating a Survey Question with a Matrix of Choices

Note: You can add a variety of formatting to all text you provide in a question, by clicking on the Show/Hide Rich-Text Editor link for the appropriate text box and using the rich text editor that is displayed.

To create a survey question with a matrix of choices, so a single set of responses displays for multiple statements/queries, first select Survey – Matrix of Choices from the Insert New Question dropdown menu on the Question page. Then, follow these steps:

1.  If desired, enter a point value for the question in the Answer Point Value box. Or, leave the zero ("0") that is entered by default. In the Question Text box, provide instructions or other information as appropriate. To add one or more attachments, click Add Attachments and follow the instructions in Adding an Attachment.

2.  In the Row Choices box, enter the statements or queries for which users will select from a set of responses. Be sure to press "return" on your keyboard after each line, so the CLE recognizes each line as a separate statement/query.

3.  In the Column Choices box, enter a set of responses from which users can choose for the statements/queries. Be sure to press "return" on your keyboard after each line, so the CLE recognizes each line as a separate response.

4.  To allow users to select only one response for each statement/query, check the box for Allow Only One Response per Column. Leave this box unchecked if you want to let users select multiple responses for each statement/query.

5.  To include a comment box for users, check the box for Add Comment Field. In the displayed box, you can enter descriptive text for the comment field. (Note: Comment boxes have a limit of 200 characters.)

6.  To set the column width for statements/queries in relation to column width for responses, select from the Relative Widths of Columns dropdown menu.

7.  If you want to assign the question to another part in the assessment, select the appropriate part from the Assign to Part dropdown menu.

8.  To assign this question to a question pool, select the name of the desired pool from the Assign to question pool dropdown menu.

9.  In the Feedback box, enter appropriate feedback to participants, if desired.

10. Click Save. The CLE once again displays the Questions page, where you’ll find the new question in the appropriate part and in the appropriate order.

To return to the Questions page without saving the question, click Cancel.
