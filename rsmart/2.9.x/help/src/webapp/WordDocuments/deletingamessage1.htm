Deleting a Message

Depending on your permissions, you may be allowed to delete any message in the Forums tool or only your own messages, or you may not be allowed to delete any posted message.

To delete a message, follow these steps:

1.  Access the message. You can do so by viewing it in the list of messages in a conversation, or by clicking the subject of the message so the CLE displays only that message.

2.  If you’re viewing the message in the list of conversation messages, click the Delete Message link for the message. If the CLE is displaying only that message, click Delete in the menu bar.

In either case, the CLE displays an alert message to confirm you want to delete the message.

3.  To delete the message, click the Delete button. The message is deleted and the list of all messages in the conversation is displayed.

Note: When you delete a message, participants will see “this message has been deleted” in place of the message when viewing messages in a conversation.
