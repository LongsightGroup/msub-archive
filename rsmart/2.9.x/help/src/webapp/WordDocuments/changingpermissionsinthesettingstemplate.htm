Changing Permissions in the Settings Template

To view and change permissions in the settings template, access the Forums tool and follow these steps:

1.  On the Forums home page, click on Template Settings in the menu bar.

The CLE displays the Default Settings Template screen.

2.  In the Permissions section, change permissions for a site role or group/section as appropriate. You can do so in one of two ways:

•      To assign a preconfigured set of permissions, click on the Permissions Level dropdown menu for the site role or group/section and select a permission level.

•      To assign a custom set of permissions, click on the Customize button for the site role or group/section. The CLE displays the permission level and specific permissions currently assigned to that role or group/section.

For the displayed permissions, check boxes and radio buttons as appropriate. The table below describes these settings.

Permission

Description

New Forum

Allows users in this role or group/section to create new forums in this site.

 

New Topic

Allows users in this role or group/section to create new topics within forums.

 

New Conversation

Allows users in this role or group/section to start new conversations.

 

Reply

Allows users in this role or group/section to post replies to messages.

 

Post to Gradebook

Allows users in this role or group/section to assign point scores to discussions, grade participation, and post grades to the Gradebook or Gradebook2 tool.

 

Change Settings

Allows users in this role or group/section to change the settings for forums and topics.

 

Read

Allows users in this role or group/section to read messages.

 

Mark as Read

Allows users in this role or group/section to mark messages as having been read.

 

Moderate Messages

Allows users in this role or group/section to moderate messages in topics by approving or denying postings. Only approved messages can be viewed by other participants. Note: This permission is only active if the moderate topic setting for the forum template or a particular forum/topic has been selected and saved.

 

Edit Messages (None, Own, All)

Determines whether users in this role or group/section can change messages that have been posted.

•      None: Users are not permitted to change any posted messages.

•      Own: Users are permitted to change only their own posted messages.

•      All: Users can change any posted messages.

 

Delete Messages (None, Own, All)

Determines whether users in this role or group/section can delete messages that have been posted.

•      None: Users are not permitted to delete any posted messages.

•      Own: Users are permitted to delete only their own posted messages.

•      All: Users can delete any posted messages.

 

Once you customize permission settings, the CLE displays Custom in the Permission Level dropdown menu for that role or section/group.

Tip: If you want to see the permission settings for a particular permission level, select that level from the Permission Level dropdown menu and then click on the Customize button. 

3.  To save your changes, click the Save Settings button. The CLE saves your settings and permissions and the Forums home page is once again displayed.
