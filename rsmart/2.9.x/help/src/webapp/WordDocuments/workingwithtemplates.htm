Evaluation System

Working with Templates

In the Evaluation System tool, each evaluation you create is based on an evaluation template. Having templates helps you save time because you can create many evaluations from a single template and then change each individual evaluation as needed without affecting the “master” template. You may create as many different “master” templates as you like. Depending on how your system has been set up, you may be allowed work with templates you’ve created and/or with templates created by others for your use.
