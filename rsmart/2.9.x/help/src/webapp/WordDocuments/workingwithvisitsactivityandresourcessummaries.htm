Site Statistics

Working with Visits, Activity, and Resources Summaries

When you click Show more for a summary on the Overview screen, the CLE expands that section and displays a table and chart, with several options for grouping and filtering information. If you expand the Visits summary, for example, your screen might look similar to this:

To group information, select the appropriate tab. By default, the CLE displays information for each summary by date. It displays "by date" information as bar graphs and displays all other information as pie charts.

If you select the By user tab in the Visits summary, for example, your screen might look similar to this:

To filter information, use the dropdown menus to make the appropriate selections. The table below explains the filter menus available in the Site Statistics tool.

 

 Filter Menu

Description

Time Period

Lets you select the time period for which statistics are displayed. You may select Last 7 days (default), Last 30 days, Last 365 days, or Since site creation.

 

User Role

Lets you select the user role for which statistics are displayed. You may select All roles (default) or a specific user role in the CLE.

 

Tool

Lets you select the tool for which statistics are displayed. You may select All tools (default) or a specific tool in the CLE for which Site Statistics provides information.

 

File Action

Lets you select the file action for which statistics are displayed. You may select All actions (default), New, Read, Revise, or Delete.

The CLE also displays a View complete report link for each summary. Click this link for a detailed report defined by the group/filter options you have chosen.
