Creating a Survey Question

Note: You can add a variety of formatting to all text you provide in a question, by clicking on the Show/Hide Rich-Text Editor link for the appropriate text box and using the rich text editor that is displayed. 

To create a survey question, select Survey from the Insert New Question dropdown menu on the Question page.

Follow the instructions for creating a multiple choice question (see Creating a Multiple Choice Question), except for the following differences:

•      The Answer Point Value entry is set by default to 0 (zero).

•      For possible answers, choose the scale you want to use. To select a scale, click its corresponding radio button.

•      You do not have the option of randomizing answers or requiring a rationale.

•      Be sure to select anonymous grading.
