Setting Permissions for the Sign-up Tool

In the Sign-up tool, you can set permissions on the site level and for specific groups and sections.

Note: To set permissions, your site role must have the realm.upd permission enabled, on the site level and also for any group or section for which you want to set permissions. If you need this permission granted, contact your Sakai CLE Functional Administrator.

To set permissions for the Sign-up tool, first access the Sign-up tool home page. Then, click on Permissions in the menu bar. The CLE displays the Permissions Update Page:

From this page, you can set permissions for the site, as well as for specific groups and sections—including groups created automatically for meeting time slots, as explained in Managing Groups Created for a Meeting.

Note: In order to set permissions for a group or section, you must be a member of that group or section. Permissions buttons for groups or sections to which you do not belong will be inactive.

To set permissions on the site level, click the Edit Permissions button for the site. The CLE displays the Permissions page for the site.

Referring to the table below, check appropriate permissions for each role. Keep in mind that users must be granted either the signup.view or signup.view.all permission to see the Sign-up tool displayed in a site’s tools menu.  

 

Permission

Description

signup.view

View meetings available to all site members and to groups/sections to which you belong.

 

signup.view.all

View meetings available to all site members and to any groups/sections in the site.

 

signup.attend

Sign up for meetings available to all site members and to groups/sections to which you belong.

 

signup.attend.all

Sign up for meetings available to all site members and to any groups/sections in the site.

 

signup.create.site

Create meetings available to all site participants.

 

signup.create.group

Create meetings for groups/sections to which you belong.

 

signup.create.group.all

Create meetings for all groups/sections in the site.

 

signup.delete.site

Delete meetings available to all site participants.

 

signup.delete.group

Delete meetings for groups/sections to which you belong.

 

signup.delete.group.all

Delete meetings for all groups/sections in the site.

 

signup.update.site

Edit meetings available to all site participants.

 

signup.update.group

Edit meetings for groups/sections to which you belong.

 

signup.update.group.all

Edit meetings for all groups/sections in the site.

 

After making your changes, click Save. The CLE once again displays the Permissions Update page.

To set permissions for a specific group or section, click the appropriate Edit Permissions button on the Permissions Update page. The CLE displays the Permissions page for that group or section.

Referring to the table above, check appropriate permissions for each role. Keep in mind that when setting group permissions, only changes to the following permissions will have an effect:

•      signup.view

•      signup.attend

•      signup.create.group

•      signup.delete.group

•      signup.update.group

After making your changes, click Save. The CLE once again displays the Permissions Update page. To return to the Sign-up home page, click the refresh icon () or Back button.
