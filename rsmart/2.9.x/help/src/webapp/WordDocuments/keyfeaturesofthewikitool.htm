Wiki

Key Features of the Wiki Tool

•      Multiple versions: Each time you or another user changes content on a wiki page, the CLE saves the previous version of the page. You may compare versions and revert to any previous version at any time.

•      Track changes: You can easily see who changed what when.

•      Help with text formatting: When you edit a wiki page, the CLE displays tips for using punctuation in wiki markup language to format text. It also provides a link to a more comprehensive list of tips for using wiki markup language.

•      Formatting icons: Formatting icons and pulldown lists simplify text formatting, adding attachments, etc.

•      Mathematics: The jsMath plugin lets you create mathematical notation for your wiki by using the TeX markup language.

•      Email notifications: You can put a watch on a page to receive email notification each time the page is changed.

•      Permissions at the page level and the tool level: You can set permissions across the entire wiki while specifying different permissions for particular pages.
