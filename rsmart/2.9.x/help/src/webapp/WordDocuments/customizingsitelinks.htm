Customizing Site Links

The Preferences tool allows you to customize site links in the following ways:

•      Change the number of site links that display across the top of the screen.

•      Change the order of site links, from left to right.

•      Hide sites, so they can’t display as site links.

•      Activate hidden sites, so they can be displayed as site links.

To customize site links, access the Preferences tool. By default, the Customize Tabs page displays. This page will also display if you click Customize Tabs in the menu bar.

You may then change the number or order of site links displayed, as instructed in Changing Your Site Links. You may also hide sites, as instructed in Hiding Sites, and activate sites, as instructed in Activating Hidden Sites.
