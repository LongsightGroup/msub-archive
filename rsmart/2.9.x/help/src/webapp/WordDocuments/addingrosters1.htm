Site Editor

Adding Rosters

To add a roster or rosters to a course site, select the Site Editor tool and follow these steps:

1.  From the Site Editor home page, click on Edit Class Roster(s) in the menu bar. The CLE displays the Edit Roster Access page.

2.  On this page, click Add Roster(s). The CLE displays the Editing Course/Section Information page.

3.  Select from the Academic term dropdown menu. Depending on the term selection and how your CLE instance has been configured, you may have three roster options:

•      To select rosters provided by your institution, proceed to step 3a.

•      To find and select other available rosters from your institution, proceed to step 3b.

•      To specify your own roster information, proceed to step 3c.

a.  Select from a list of provided rosters by checking the appropriate box(es). Then click Continue. The CLE displays the Request Site Access page. Proceed to step 4. 

b.  If you want to add available roster(s) that are not listed, click on the Add course(s) and/or section(s) not listed above link. The CLE displays the Course/Section Information page. Find and add available rosters by selecting from the Subject, Course, and Section dropdown menus.   

Once you have made your selections, the CLE displays additional boxes:

•      In the Authorizer’s username box, enter the name of the person who must authorize site access for the additional roster(s).

•      If you have any comments or special instructions regarding your roster access request, enter them in the Special instructions box.

To add more rosters, click Save and add another section and repeat the above process. When you have added all rosters, click Continue. The CLE displays the Request Site Access page. Proceed to step 4.

c.  If no menus are displayed on the Course/Section Information page, or if you would like to provide your own roster information, click on the Still cannot find your course/section? link. Then, specify the appropriate roster information in the Subject, Course, and Section boxes.  

If you want to add more rosters, select the appropriate number of rosters from the Add More Rosters? dropdown menu for a display of additional information boxes.

Once you have specified all appropriate roster information:

•      In the Authorizer’s username box, enter the name of the person who must authorize site access for the additional roster(s).

•      If you have any comments or special instructions regarding your roster access request, enter them in the Special instructions box.

Click Continue. The CLE displays the Request Site Access page.

4.  On the Request Site Access page, confirm the roster(s) that you want to add to the site. Then, click Add Class(es).

The CLE displays the Site Editor home page, where you’ll see the additional roster(s) listed.
