Changing Permissions for the Entire Wiki

To edit permissions for the whole wiki, access the Wiki tool and follow these steps:

1.  Click the Info button on the menu bar. The CLE displays the Info screen for the Home page.

2.  Beneath the Permissions section, click the edit site permissions link.

The CLE displays the permissions for the entire wiki.

3.  Referring to the table below, check and un-check boxes as appropriate to switch permissions off and on for each role.

Permission

Description

Create

Users are allowed to create new wiki pages. (Update permission must also be turned on for this permission to be in effect.)

Read

Users are allowed to view wiki pages.

Edit

Users are allowed to change wiki pages.

Admin

Users are allowed to change site permissions.

Super Admin

Users are allowed to create, read, and edit wiki pages, and to change site permissions.

 

4.  To save your changes, click save. Or, to exit without saving changes, click cancel.
