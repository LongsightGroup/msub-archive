Changing Permissions for Forums and Topics

If you are an instructor or site organizer, you can control user permissions for your site's forums and topics. You can change permissions at the following levels.

•      Template: A template defines the default permissions for users in each site role, as well as for any groups or sections in your site. The permissions you set in the template apply to all newly created forums and topics, unless you change permissions for a specific forum or topic.

•      Forum: You can override the template permissions for any forum. The permissions you set for a forum will apply to all newly created topics in the forum, unless you override the permissions for a particular topic.

•      Topic: You can override both template and forum permissions for any topic.
