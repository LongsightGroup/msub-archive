Site Setup > Creating a Course Site

Building Your Own Course Site

To build your own course site in the CLE, access My Workspace and select the Site Setup tool. Then, follow these steps.

1.  From the Site Setup home page, click New in the menu bar. The CLE displays the Site Type page.

2.  On this page, click on the course site radio button and select a term from the Academic term dropdown list.

Note: If you do not see a New button in the Site Setup menu bar, you do not have permission to create new sites. If you do not see a course site radio button, you do not have permission to create course sites.

3.  Click Continue. The CLE displays the Course/Section Information page. Depending on how your CLE instance has been configured, you may have three roster options:

•      To select rosters provided by your institution, proceed to step 3a.

•      To find and select other available rosters from your institution, proceed to step 3b.

•      To specify your own roster information, proceed to step 3c.

a.  Select from a list of provided rosters by checking the appropriate box(es). Then click Continue. The CLE displays the Course Site Information page. Proceed to step 4. 

Alternatively, you may:

•      Click the Add course(s) and/or section(s) not listed above link if you want to select and add roster(s) not listed. Proceed to step 3b.

•      Click Continue with no roster if you want to create the site without any rosters. The CLE displays the Course Site Information page. Proceed to step 4.

Tip: If you create a course site without a roster, you can use the Site Editor tool to add a roster at a later date.   

b.  Find and add available rosters by selecting from the Subject, Course, and Section dropdown menus.  

Once you have made your selections, the CLE displays additional boxes:

•      In the Authorizer’s username box, enter the name of the person who must authorize the creation of this course site.

•      If you have any comments or special instructions regarding your site creation request, enter them in the Special instructions box.

To add more rosters, click Save and add another section and repeat the above process. When you have added all rosters, click Continue. The CLE displays the Course Site Information page. Proceed to step 4.

If the appropriate rosters are not displayed and you want to specify your own roster information, click Still cannot find your course/section? Proceed to step 3c.

c.  Specify the appropriate roster information in the Subject, Course, and Section boxes.   

If you want to add more rosters, select the appropriate number of rosters from the Add More Rosters? dropdown menu for a display of additional information boxes.

Once you have specified all appropriate roster information:

•      In the Authorizer’s username box, enter the name of the person who must authorize the creation of this course site.

•      If you have any comments or special instructions regarding your site creation request, enter them in the Special instructions box.

Click Continue. The CLE displays the Course Site Information page.

4.  On the Course Site Information page, provide settings and information. 

•      Depending on how your CLE instance has been configured, you may be able to edit your site title. Edit or enter text as appropriate in the Site Title box. 

•      Depending on how your CLE instance has been configured, you may be able to provide a site URL. If so, optionally enter the URL in the Site URL box. This URL can be a more user-friendly URL than the default, which includes the site ID.

•      From the Site Language menu, optionally select a different language for your site. By default, the site will use the language for which your CLE instance has been configured.

•      Under the Description header, use the rich text editor to enter course information that will be displayed on the site’s home page.

Tip: After you create a site, you may change the description by clicking the Edit icon () in the Site Information area on the site's home page.

•      In the Short Description box, enter a shorter description (maximum 80 characters). This description is displayed on publicly viewable sites.

•      From the Appearance (Theme) menu, you may optionally select a different “theme,” or graphics, for your site. (Note: the selections you see in this menu will depend on how your CLE instance has been configured.)   

•      In the Site Contact Name box, enter the first and last name of the person responsible for the site.

•      In the Site Contact Email box, enter the email address of the person responsible for the site.

Once you have provided all information and settings, click Continue. The CLE displays the Course Site Tools page.

5.  On the Course Site Tools page, check appropriate boxes to select tools to be used on your site. Or, to select all tools, check the box for All tools at the top of the list.

6.  If you don’t want to reuse materials from other sites you own, select No, thanks and then click Continue. If you do want to reuse material from other sites you own:

a.  Select Yes, from these sites.

b.  Scroll through the list and select the site(s) from which you wish to reuse material.

c.  Click Continue. The CLE displays another screen listing tools for which you may reuse material. Check boxes to choose the appropriate tools. Then, click Continue again.  

Your next step will depend on whether any tools in your site require further configuration:

•      If one or more tools require further configuration, the CLE displays the Add Multiple Tool Instances page. Proceed to step 7.

•      If no tools require configuration, the CLE displays the Course Site Access page. Proceed to step 8.    

7.  On the Add Multiple Tool Instances page, enter the appropriate information for each tool.

The type of information you provide will vary, depending on the tools you have selected:

•      If you have selected the Email Archive tool, you must select a unique archive email address for the site.

•      If you have selected the Basic LTI or Lesson Builder tool, you may enter a different tool title.

•      If you have selected the News or Web Content tool, you may enter a different tool title as well as enter a URL.

You may also add additional instances of the Basic LTI, Lesson Builder, News and Web Content tools to your site. To do so, select the number of instances for a tool from the dropdown list and provide the appropriate information.

     Click Continue. The CLE displays the Course Site Access page.

8.  On the Course Site Access page, choose how you would like the course to be accessed by users.

     Choose from these options:

     Site Status

•      To publish your site so it is available to site participants, select Publish site.

•      To keep your site unpublished so it is available only to you and other site maintainers (such as CLE Functional Administrators), select Leave as Draft – accessible only to site maintainers.

Note: By default, course site visibility is set to "public." Your course site, as well as any public content in the site, will be visible to all users upon login. 

Global Access

•      To make your site accessible only to site participants, select Limited to whom I add manually, or through automatic roster updates.   

•      To make your site accessible to all users with a CLE login ID and password, select Allow anyone to join the site with valid login id. Then, use the pulldown list to select an appropriate role for users who choose to join the site.

9.  Click Continue. The CLE displays the Confirm Your Course Site Setup page.

10. Review the information on this page. To finish creating the site, click Request Site.

After processing your site request, the CLE displays the Site Setup home page, where you’ll see your new course site included in the list of sites.
