Exporting an Assessment

To export an assessment, access the Tests & Quizzes tool in the site that contains the assessment and follow these steps:

1.  On the Tests & Quizzes home page, find the appropriate assessment in the list from the Working Copies tab.

2.  Select Export from the Action dropdown menu next to the assessment. The CLE displays the Export Assessment screen.

3.  Click to select either QTI v 1.2 or Content Packaging.

Note: If this assessment contains images and attachments, choose Content Packaging. This option creates a zip file that includes all attachments. If you have text only, you can use the QTI option to create an .xml export. Both formats import easily into other courses.

4.  To export the assessment, click Export. If you are exporting with the Content Packaging option, a zip file downloads immediately to your computer. If you are exporting with the QTI option, your browser displays a window with downloading instructions:

In this window, click Download. In your browser, access the File menu and click either Save Page As or Save As (depending on your browser) to save the assessment as an XML file on your computer.
