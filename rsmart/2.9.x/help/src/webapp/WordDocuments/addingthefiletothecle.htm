Adding the File to the CLE

After you have created a .csv file, upload it from your computer to the site as follows:

1.  If you do not see the Feedback menu bar, click the Reset button.

2.  Click Add on the menu bar. The CLE displays this screen:

3.  Enter a title for the posting in the Title box.

4.  Specify a file by browsing and selecting the appropriate .csv file from your computer.

5.  To make feedback available to participants, check the Release feedback to participants? box.

Warning: If you do not check this box, your file will be added, but the contents will not be displayed to site participants.

6.  Click Post to add the file, or click Cancel to exit without adding it.
