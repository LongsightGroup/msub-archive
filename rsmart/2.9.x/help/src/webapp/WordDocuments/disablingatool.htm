Disabling a Tool

To disable a tool using Page Order, follow these steps:

1.  On the Site Editor home page, click on Page Order in the menu bar.

2.  Find the appropriate tool in the Page Order list.

3.  Click on the “lock” icon () for the tool. The icon changes from unlocked to locked (), and a message is displayed confirming that the tool is now disabled, so users won’t see it displayed in the tools menu and won’t have access to its content.

A site owner or other user with appropriate site maintenance permission will still have access and will see the tool in the menu, displayed in gray italics.

Clicking “Save” is not necessary, but clicking this button or the “Cancel” button will return you to the Site Editor home page. To enable a tool you have disabled, simply click on the “locked” icon for that tool.    

Note: If you don’t want a tool to display in the tools menu, but you do want to provide access to its content, you should hide the tool instead, as instructed in Hiding a Tool.
