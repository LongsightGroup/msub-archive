Creating a Portfolio Based on Your Own Design

After you have completed the steps in Creating a Portfolio, the CLE displays this screen.

Tip: You may return to the portfolios hub screen from this screen at any time. To do so, click the Summary tab near the top of the screen.

To proceed, follow these steps:

1.  Begin to create a page in the portfolio by clicking the Add Page link. The CLE displays the Edit Page screen.

The screen contains two sections: Page Information and Page Content.

•      A right-facing arrow ( ) in the header for either section indicates that the section is closed. To open it so you can see the boxes in that section, click the arrow.

•      A down-facing arrow ( ) in the header for either section indicates that the section is open (that is, its boxes are displayed). To close it, click the arrow.

2.  In the Page Information section, enter a title for the portfolio page in the Title box. The Title entry is required.

3.  If desired, enter a description of the page in the Description box and enter keywords in the Keywords box.

4.  Click Select Layout link. The CLE displays a list of layouts.

5.  Select a layout by following the instructions in Selecting an Item from Resources for Use in ePortfolios. After you select the layout, the CLE displays the Edit Page screen again. The name of the layout you have chosen is displayed in the Layout box, and a preview of this layout is displayed below the box.

6.  To use your institution's default style for fonts, colors, and graphics, leave the Style box blank. To select another style for this portfolio, click the Select Style link. The CLE displays a list of styles for you to choose from.

7.  To select a style for this portfolio, follow the instructions in Selecting an Item from Resources for Use in ePortfolios. After you select a style, the CLE displays the Edit Page screen again. The name of your chosen style is displayed in the Style box. Unless you select another style for one or more specific pages, this style applies to the entire portfolio.

8.  If the Page Content section is closed, click the arrow in the section header to open it. The CLE displays boxes for the layout you have chosen.

9.  Enter text for each part of the layout via the rich text editor.  For information on using the formatting icons and pulldown lists in the text box, see Using the Rick Text Editor. To add blocks of text in sequence, click the Add Row button near the bottom of the screen. The CLE displays a box.

Choose among these options:

•      To add a chunk of text, type or paste it into the box. Although each box is small, you may enter as much text as you need.

•      To remove a text box, click the Remove Row button.

•      To add the next chunk of text, click Add Row again.

10. To add this page to your portfolio, click Add Page. The CLE displays the preceding screen again. The screen now shows an icon for the page you have added.

11. Choose any of these options:

•      To add another page, repeat steps 1 – 10.

•      To edit a page, click the  icon under the name of the page.

•      To remove a page, click the  icon under the name of the page. When the CLE displays a confirmation screen, click Continue.

12. To include links that allow users to move from page to page within the portfolio, check the Page Navigation box.

13. After finishing your work, click Save Changes. Or to undo all work done during this editing session, click Undo.

14. To preview your portfolio at any time, click the Preview the Portfolio link in the top right quadrant of the screen. After previewing it, close the window.

15. To return to the Portfolio home page at any time, click the Return to List link in the top right quadrant of the screen.
