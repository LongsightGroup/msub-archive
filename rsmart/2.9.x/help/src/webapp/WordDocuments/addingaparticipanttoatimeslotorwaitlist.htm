Adding a participant to a time slot or wait list

On the Meeting Details page for a meeting, you can add a participant to a time slot by clicking on the appropriate Add Participant link in the Participants column:

From the dropdown menu, select the appropriate user. Then, click OK. The user will be displayed for that time slot.

If you want to add a participant to the wait list for a slot, click on the appropriate Add Participant link in the Wait List column. (Keep in mind that the link will only be active if the slot has reached its maximum number of participants.)

 

From the dropdown menu, select the appropriate user. To choose whether the user is added to the bottom or top of the wait list, click on the Bottom or Top radio button. Last, click OK. The user will be displayed in that wait list.
