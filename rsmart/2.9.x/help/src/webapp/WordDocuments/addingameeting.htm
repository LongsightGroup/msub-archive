Adding a Meeting

To add a new meeting in the Sign-up tool, go to the tool home page. Then, follow these steps.

1.  In the menu bar, click Add. The CLE displays the Create New Meeting: Basic Information page.

2.  Provide meeting information, organizer, location, and time.

a.  In the Title field, enter a title for the meeting. This information is required.

b.  From the Organizer menu, optionally select an organizer for the meeting. If you’re the site owner, you will be the meeting organizer by default, but you may also choose another user as an organizer. (Note: only users with the appropriate permission will be displayed in this menu.)

c.  Provide a location for the meeting. If you have already added meetings, you may use the dropdown menu to select from previously entered locations. Or, you may enter a new location. This information is required.

d.  Include additional information for the meeting, by entering a name in the Category field, using the rich text editor, and adding attachments.

e.  Using the displayed fields, dropdown menus, and calendar icons, select a start/end time and date for the meeting. This information is required.

Note: Even if your meeting will be recurring, the start/end dates should be identical. You should only set a different end date if the meeting will span multiple days.

3.  Determine frequency, sign-up period, attendance tracking, and availability.

a.  From the Meeting Frequency dropdown menu, select how many times the meeting will occur. For every selection except Once Only, the CLE displays additional settings.

         

You can set frequency in one of two ways:

•      Click the occurrences radio button and enter the appropriate number in the field.

•      Click the End After radio button and provide a date.

Meeting frequency is a required setting.

b.  Determine the meeting’s sign-up period with the Sign-up begins and Sign-up ends dropdown menus and fields. The dropdown menus let you set the time period by days, hours, or minutes. You can also set the sign-up period to start immediately by selecting Start Now from the menu.

c.  If you want to record attendance for the meeting, check the box for Attendance will be taken. For more information, see Tracking Attendance for a Meeting.

d.  In the Available To section, set availability for the meeting. This setting is required. By default, the current site is selected, so all participants in the site have access. But you may also set availability for specific groups or sections, as well as click Other Sites to make the meeting available to participants in other sites you own.

4.  In the Meeting Type section, select the meeting type, which is a required setting. If you don’t want to require sign-up for the meeting, select Open meeting and proceed to the next step. Otherwise, select one of the other two meeting types. The CLE displays additional settings.

•      Single slot meeting type:

Select Max # of participants and enter a number in the field. Or, select Unlimited number of participants. Then proceed to the next step.

•      Multiple slots meeting type:

If you want all slots to be of the same duration, enter the number of available slots, as well as the number of participants per slot. The CLE calculates the estimated duration of each slot, in minutes, according to these two settings. Once you have entered the number of slots/participants, proceed to the next step.

If you want to customize any or all of the slots, select Advanced user-defined timeslots. Then, click on the Create Timeslot(s) link. The CLE displays the Define Custom Timeslots page.

On this page, you’ll see the number of time slots and participants per slot you have already set for the meeting, as well as the time period for each slot based on those settings. You may customize each individual slot, and you may also create additional slots by clicking Add a new timeslot.

Note: You may overlap time slots, but keep in mind that a slot’s start and end times must be within the time period you original set for the meeting. By default, any time slots that are more than two hours apart will be published as separate events in the Calendar tool, but you may de-select this setting.

After customizing the time slots, click the Continue button. You will be returned to the Create New Meeting: Basic Information page. Proceed to the next step.

5.  Click the Next button. The CLE displays the Meeting Summary page, with a summary of information and settings for the meeting. If you have created custom time slots, click on Show custom-defined timeslots for a display of the slots.

6.  Select display and notification options.

a.  Check the box for Display Participant Names if you want site participants to see the names of other users in the meeting.

b.  Check the box for Receive Notification if you want to be notified by email when a user signs up for the meeting or cancels the appointment.

c.  Change the Announce Availability setting. By default, an email notification of availability will be sent to all potential participants, but you may select Only Organizers to send email notification only to meeting organizers and coordinators. 

d.  Change the Default Notification setting. Uncheck the box for this setting if you do not want it selected by default for users.

e.  Add meeting coordinators. Click on Add/Edit for a list of users with organizer permissions and check the appropriate boxes. These users will receive email notification of attendee activity.

7.  Change or select additional settings by clicking Show the other default settings.

a.  Uncheck the Allow Wait List box if you don't want users to have the option of joining a wait list for the meeting.

b.  Uncheck the Allow Adding Comment box if you don't want users to have the option of adding a comment when signing up for the meeting.

c.  Check the box for User ID Input Mode box if you want site participants to enter their user ID when signing up for the meeting.

d.  Uncheck the Auto Reminder box if you don't want an email reminder sent to attendees one day prior to the meeting.

e.  Uncheck the Publish to Calendar setting if you don’t want the meeting included as an event in the Calendar tool.

f.   Check the Create groups for timeslots box if you want the CLE to automatically create a site group for each time slot. For more information, see Managing Groups Created for a Meeting.

g.  To allow site participants to sign up for more than one time slot, select from the dropdown menu for Max # of time slots per participant.

8.  Publish the meeting.

•      If you have created an open or single-slot meeting--or if you have created a multiple-slot meeting and you want participants to sign up for it--click Publish.

•      If you have created a multiple-slot meeting and you want to assign participants to it, click Assign Participants & Publish. The CLE displays the Assign Participants page:

For each time slot, click Add Participant. Select a site participant from the displayed dropdown menu and then click OK. (You may add more participants than the maximum you selected). Once you have added all participants, click Publish.

Note: When you assign a time slot to a participant, keep in mind that the participant may still cancel the slot, and may then join any other available slot.

Once you publish the meeting, you will be returned to the tool home page, and the new meeting will be displayed in the list of meetings.
