Changing a Message

Depending on your permissions, you may be allowed to change any message on the site or only your own messages, or you may not be allowed to change any message after it has been posted.

To change a message, access the Forums tool and follow these steps:

2.  Access the Edit Forum Message screen. You can do so in one of two ways:

•      From the list of messages in a conversation, find the appropriate message and click the Edit link for it.

•      Click on the subject of the message so the CLE only displays this message. Then, click Edit in the menu bar.

In either case, the CLE displays the Edit Forum Message screen.

2.  If desired, change the Reply Title and/or Message entries.

3.  To add one or more attachments, click Add Attachment and follow the instructions in Adding an Attachment.

4.  To post the changed message, click Post Edited Message. The CLE displays the view of all messages in the conversation again. 

Note: If participants read this message before you revise it, or if they mark it as read before that time, the message continues to be marked as read. When they view the message, they’ll see a note indicating the date and time of your revision.
