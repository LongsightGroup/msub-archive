Working with the fMath Editor

The rich text editor includes fMath, a math editor that lets you create mathematical equations and formulas and insert them as images in your text.

To add math in the rich text editor using fMath, first position your cursor where you want the math to be inserted. Then, follow these steps:

1.  Click the fMath icon () in the menu bar. The fMath editor window opens.

 

2.  In the properties window, which displays automatically, select or enter properties as appropriate. Then, click the Update button. Close the window by clicking on the “x” icon in the window’s upper right corner. You may reopen this window at any time by clicking on the properties icon ().

3.  Create math. You can do so by selecting from the different menus displayed horizontally on top and vertically at the sides. With these menus, you can add letters, numbers, symbols, functions, and more.  

Tip: To move a menu, click on the icon () at the end of the menu and drag the menu to the appropriate location. For more information about menu functions, see Overview of fMath Menus.

Many menus let you add templates with entry boxes for creating the appropriate math. You could, for example, add a template for creating a simple fraction:

After clicking first on the menu icon and then on the appropriate item in the menu window that opens, enter a value for the numerator and denominator in the two entry boxes:

Click on an ellipse to see a highlighted box for that entry. You can enter letters and numbers either with your keyboard or by selecting from a menu.

Alternatively, you can click on the ellipse at the cursor location and hit the return key on your keyboard. A window for adding new elements displays. From this window, you can select from a limited number of items to add, such as a fraction:

After selecting, click the Add New Element button. The window closes and the item is added. For a fraction, you would see the template with entry boxes.

Once you’ve finished adding items, the math you create might look like this:

When creating math, you can use the arrow, backspace, and delete keys on your keyboard for navigating to different entry boxes, making changes, and moving the cursor to a new position to add more items.

4.  Click the OK button to insert the math. The fMath window closes and the math is displayed as an image in the rich text editor:

To edit the math, click on the image and then once again click on the fMath icon () in the menu bar. The fMath window will open. Once you’ve made all necessary changes, click OK. The fMath window closes and you’ll see the edited math in the rich text editor.

Important Notes
You may only edit an image created with fMath before you save your work in the rich text editor. If, for example, you publish an assignment or assessment that includes fMath images, you cannot edit the math in an image when editing the assignment or assessment. Instead, you must delete the image and create new math.

Images you create with fMath are saved as attachments. They will not be stored as items in Resources in your site, and only CLE administrators will have access to them. Links to fMath images will be maintained upon site import or duplication.
