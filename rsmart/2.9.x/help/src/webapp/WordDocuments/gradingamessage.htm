Grading a Message

To grade a particular message authored by a student, follow these steps:

1.  From the Forums home page, access the Grade Forum Message window or page for the appropriate message. You can do so in one of the following ways:

•      Access the appropriate topic and select the conversation thread to which the message belongs. For a display of the Grade Forum Message popup window, click the Grade link.

•      Or, open the message and click Grade in the menu bar to be taken to the Grade Forum Message page.

•      You can also access the Grade Forum Message popup window by clicking Statistics & Grading in the tool menu bar and then clicking Statistics & Grading by User. Click on the appropriate user name for a list of all authored messages and click the Grade link for the appropriate message.

Tip: From the Statistics & Grading page, another way to access a message is by clicking Statistics & Grading by Topic, selecting the appropriate topic, and then selecting the appropriate user.

Whether the CLE displays the Grade Forum Message popup window or page, you’ll see a display of the message, a dropdown menu for associating the message with a gradebook item, and fields for entering a grade and comments.

    

2.  If the message has not yet been associated with a gradebook item, or you want to change the association, choose the appropriate item from the dropdown menu.      

3.  Enter a grade and comments. You may only enter grades as points, and only to the maximum allowed for the associated gradebook item.

4.  Click Submit Grade. A confirmation message is displayed. (If you are in the Grade Forum Message popup window, the window closes.)  

To return to the Forums tool home page, click the reset button (). Depending on how you have graded, you may also access another forum, by clicking the appropriate link in the breadcrumb trail; navigate to the authored messages of another user; or access grading for another message.
