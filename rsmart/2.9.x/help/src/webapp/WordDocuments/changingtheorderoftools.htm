Changing the Order of Tools

To change the order of tools displayed in the tools menu using Page Order, follow these steps:

1.  On the Site Editor home page, click on Page Order in the menu bar.

2.  Drag and drop tools in the Page Order list as appropriate.

3.  Click the Save button. The Site Editor home page is once again displayed, and you will see the new order of tools in the tools menu.

If you want to undo your saved changes to the tool order, click the Reset button on the Page Order screen. In the confirmation window that displays, click the OK button. Once you navigate away from Site Editor, you’ll see the previous tool order.
