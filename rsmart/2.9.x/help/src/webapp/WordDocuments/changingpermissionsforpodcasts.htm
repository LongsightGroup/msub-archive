Changing Permissions for Podcasts

Note: Permission options for the Podcasts tool are the same as for the Resources tool. Note that in the Resources tool, you may set permissions as desired for the Podcasts folder.

To view and change permissions for using this tool, access the Podcasts tool and follow these steps:

1.  If the Podcasts home page is not displayed, click the Reset button.

2.  Click Permissions in the menu bar near the top of the screen. The CLE displays the Permissions screen.

3.  Click appropriate check boxes to set permissions. You have the following options for each role:

Role

Description

content.new

Users in this role are allowed to add new podcasts to the site.

content.read

Users in this role are allowed to view and read podcasts.

content.revise.any

Users in this role are allowed to revise any podcast on the site.

content.revise.own

Users in this role are allowed to revise only those podcasts they have added.

content.delete.any

Users in this role are allowed to delete any podcast on the site.

content.delete.own

Users in this role are allowed to delete only those podcasts they have added.

content.hidden

Users in this role are allowed to view and work with podcasts that are hidden from users in other roles.

 

4.  After you have changed the permissions, click Save. Or, to exit and revert back to your current settings, click Cancel.

Note: The permission changes you have entered take effect immediately.
