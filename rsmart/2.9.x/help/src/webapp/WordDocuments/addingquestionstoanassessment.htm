Adding Questions to an Assessment

With the Tests & Quizzes tool, you may add a question to a pending (unpublished) assessment when you first create it, or at any other time. You may create a new question to add, or you may add an existing question from a Question Pool. You may also choose the part to which the question will be added.

Note: You may not add questions to a published assessment.

To add a question from a Question Pool to an unpublished assessment, proceed as instructed in Copying a Question from a Question Pool to an Assessment.

To add a newly created question to an unpublished assessment, follow these basic steps:

1.  Access the Questions page for the assessment. You’ll see this page when you initially create an assessment, as instructed in Creating an Assessment with Assessment Builder. On the Tests & Quizzes home page, you can access the Questions page for any unpublished assessment by selecting Edit from the Action dropdown menu for that assessment.

2.  Decide where in the assessment you want to add the question.

•      If your assessment includes multiple parts, find the part to which you would like to assign the question.

•      If this part has existing questions, decide where you want the new question inserted (questions are automatically re-ordered once the question is added).

3.  From the appropriate Insert New Question dropdown menu, select the question type.

Tip: When creating a question, you can also use the Add Question dropdown menu displayed at the top of the Question page, and select the appropriate part before adding the question.  

For each question type, the CLE displays an Edit Question page. What you see on this page will depend on the type of question you’re creating.

4.  Create the question by entering and saving all necessary information and/or content, as appropriate for the question type:

•      Multiple Choice

•      Survey

•      Survey – Matrix of Choices

•      Short Answer/Essay

•      Fill in the Blank

•      Numeric Response

•      Matching

•      True/False

•      Audio Recording

•      File Upload

•      Calculated Question

Once you’ve added the question, you’ll see it displayed on the Questions page, in the appropriate part and in the appropriate order.
