Hiding a Tool

To hide a tool in the tools menu using Page Order, follow these steps:

1.  On the Site Editor home page, click on Page Order in the menu bar.

2.  Find the appropriate tool in the Page Order list.

3.  Click on the “hide” icon () for the tool. The icon changes color (), and a message is displayed confirming that the tool is now hidden from users, so it won’t display in the tools menu. 

A site owner or other user with appropriate site maintenance permission will still see the tool in the menu, displayed in gray italics.

Clicking “Save” is not necessary, but clicking this button or the “Cancel” button will return you to the Site Editor home page. To display a hidden tool to users, simply click on the “hide” icon for that tool.      

Note: Although a hidden tool is not displayed to other users, you may still provide users with access to content for the tool. You may, for example, provide a URL for an item in Resources or for a published assessment in Tests & Quizzes. You may also add a link to a hidden tool in Lesson Builder.
