Creating a Blog Post

To create a new blog post, access the Clog tool and follow these steps:

1.  Click on Create new post in the menu bar. The Post Editor page is displayed.

2.  In the Title field, enter a title for the post (the title must be at least 4 characters).

3.  Enter your post in the rich text editor. Use the tool menu in the editor to format text and insert content or links, as appropriate.

4.  If you do not want site participants to comment on your post, uncheck the Allow comments box (by default, the box will be checked).

5.  Decide on visibility for the post. If you want the post to be visible only to yourself and the instructor (or another user with appropriate permission), select Only site administrators and I can see this post. If you want the post to be visible to all users in the site, select All members of this site can see this post.     

6.  Click Publish if you want to publish the blog post and make it visible according to your setting selection in the previous step. Click Save as Draft if you want to save the post but keep it visible only to yourself.

Whether your post is published or saved as a draft, you’ll see it displayed on the Clog home page, as the first post in the list:

You’ll also see it in your list of posts if you click View my blog in the tool menu bar.

The post will include an icon for adding a comment () if comments are allowed, as well as icons for deleting () and editing () the post. You’ll see an additional “draft” icon () displayed if the post has been saved as a draft.
