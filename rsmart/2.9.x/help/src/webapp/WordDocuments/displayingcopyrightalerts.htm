Displaying Copyright Alerts

The CLE can insert a copyright statement for any uploaded material that has a copyright. Whenever you upload items into Resources or another tool, the CLE displays this set of fields:

To manage copyrighted material, follow these steps:

1.  Select the appropriate status from the Copyright Status pulldown list.

Tip: For more information about copyright status, click the (more info) link. The CLE displays detailed information in a separate window.

2.  To display a copyright alert when a site participant accesses this document, click Display copyright alert and require acknowledgement when accessed by others. When you choose this option, site participants must acknowledge this warning every time they access the document.
