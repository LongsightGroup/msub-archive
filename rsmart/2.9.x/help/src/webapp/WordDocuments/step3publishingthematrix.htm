Step 3: Publishing the Matrix

Publishing a matrix makes it available for use by site participants.

Warning: After a matrix has been published, you cannot add, move, reorder, or delete any of the columns or rows in it. For this reason, we strongly recommend that you publish a matrix only after previewing it and verifying that it contains the components are exactly you want them.

To preview the matrix, access the Matrices tool and follow these steps:

1.  If the CLE is not displaying the Matrices home page, click the Reset button.

2.  Find the matrix you want to change, and click Preview. The CLE changes the entry in the Status column to Preview. It also changes the Preview link to Publish.

3.  Click the name of this matrix. The CLE displays the matrix in preview mode.

4.  Review the material on this screen and click all links to review materials in all cells.

5.  Click Return to List in the menu bar near the top of the screen. The CLE displays the Matrices home page.

6.  When you have determined that the matrix is exactly as you want it, publish it by clicking the Publish link. The CLE displays a confirmation screen.

7.  Click Continue. The CLE publishes the matrix and redisplays the home page. Or to return to the home page without publishing this matrix, click Cancel.
