Adding the Evaluation

Note: Evaluations are based on templates, so at least one evaluation template must be available for your use (created by you or someone else) before you can create an evaluation.

To create a new evaluation, access the Evaluation System tool and follow these steps:

1.  If you are not on the Evaluation System home page, click the Reset button to display it.

2.  In the menu bar at the top of the screen, click My Evaluations. The CLE displays the My Evaluations screen.

This screen contains lists of your evaluations in all sites you own. Each list includes evaluations in a different state.

Evaluation State


Explanation

Partially Complete

Evaluations you began creating but did not complete, because you exited either step 2 (Specifying Settings and Dates) or step 3 (Assigning Groups of Evaluators) before finishing. Warning: If you do not complete a partial evaluation within 5 calendar days, the system automatically deletes it.

In-Queue

Evaluations that are scheduled to begin but have not yet started.

Active

Evaluations that have been released to evaluators; evaluations are not yet due. Users can take these evaluations now.

Closed

Evaluations whose due date has passed. Users cannot take these evaluations any longer.

 

3.  In the upper right quadrant of the screen, click Add Evaluation. The CLE displays the Add Evaluation screen.

4.  In the Title field, enter a name for this evaluation.

5.  In the Instructions box, provide instructions to the users. Use the editor’s pulldown lists and buttons to format the instructions as appropriate. For information on using these options, see Using the Rich Text Editor.

6.  In the lower portion of the screen, click to choose the template you want to use.

7.  To proceed, click Continue to Settings. The CLE creates a “partial” evaluation and displays the Evaluation Settings screen for it. (Or, to return to the preceding screen without creating the evaluation, click Cancel.)

8.  To go on to the next step in the overall process, see Specifying Settings and Dates. (Otherwise, return to the My Evaluations screen by clicking Cancel. The saved evaluation is displayed in the list of partially complete evaluations.)
