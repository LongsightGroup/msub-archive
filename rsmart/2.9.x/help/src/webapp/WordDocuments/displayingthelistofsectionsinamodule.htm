Displaying the List of Sections in a Module

Each module on the home page is made up of one or more content sections.

To see the sections in a module, click the triangle icon next to the module name. The CLE displays the sections beneath that name.

To return to the list of modules, click the triangle icon again. The CLE displays the usual view of the home page.

Another option is to display all sections in all modules. To do so, simply click the  icon to the left of the Title column header above the list itself.
