Duplicating a Site

Using Site Editor, you can duplicate an existing site to create a new site.

Note: To protect participants' privacy, when a new site is created by duplication, participant lists and student data are not duplicated

To duplicate a site, access the Site Editor tool and follow these steps:

1.  From the Site Editor home page, click Duplicate Site in the menu bar. The CLE displays the Duplicate Site page.

2.  Enter a title for the new site (up to 20 characters). An entry in this field is required.

3.  Select the appropriate term from the Academic term dropdown menu. An entry in this field is required.

4.  To create a copy of your site, click Duplicate. The CLE duplicates the site with the new name and displays the Site Editor home page.

Depending upon how much material you have on your site, it may take some time to duplicate the site. When the CLE displays the Site Editor home page, the duplication is complete.

To confirm the new site has been created, find the site via displayed navigation link or by accessing the home page of the Site Setup tool in My Workspace.
