List of Events

The List of Events view allows you to view calendar information in list format and print it. Above the list the CLE displays several fields that allow you to customize the list.

•      Select the date range in the Show list and click Filter Events.

•      To limit the list of events to a date range other than the pre-defined range, select Custom Date Range in the Show list, select Start and End dates, and click Filter Events.

•      To view today’s schedule in the Calendar by Day view, click Go to Today.

•      To see events for a particular group, click the radio button for Display group events and then select the appropriate group or groups. Otherwise, leave Display site events selected or click to select it.

•      To print the list, click Printable Version in the menu bar above the list and then proceed as usual for printing from your computer system.

•      To make the list your default view, click Set as Default View in the menu bar above the list.
