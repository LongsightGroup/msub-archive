Performing Common Tasks

Accessing Public Site Content

Site owners in the CLE have the option of making resources, announcements, syllabus items, and other site content publicly available on the gateway, so no CLE login is required for access.

Note: Depending on how your institution has configured the CLE and the gateway, you may not have access to public content.

 To access public site content, follow these steps:

1.  Navigate to the gateway page for your institution’s instance of the CLE.

2.  On the gateway, access the Site Search tool. Depending on how your gateway is configured, you may access the tool by clicking on a link indicating public sites, or by selecting from a tools menu on the left side of the gateway.

In either case, the CLE displays the Site Search home page.

3.  If desired, use the radio buttons to select a particular type of site. For course sites, you may also select an academic term from the pulldown list.

4.  If desired, enter part of a site name in the Search for box.

5.  Click Search for Sites. The CLE displays a list of sites that match the criteria you entered.

6.  Click the name of the site for which you want to view public information. The CLE displays all publicly available information, files, links, etc., for that site.

7.  To return to the CLE gateway, use the Back button on your browser.
