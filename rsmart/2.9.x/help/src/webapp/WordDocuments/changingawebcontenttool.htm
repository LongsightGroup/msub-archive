Changing a Web Content Tool

To change a Web Content tool you have already added to your site, access the appropriate tool and follow these steps:

1.  Click the “Edit” icon () in the upper right corner. The CLE displays the Customize Web Content screen:

2.  Change the tool titles:

•      To change the name displayed on the tool home page, enter the appropriate name in the Tool Title box.

•      To change the name displayed in the site’s navigation (tool) menu on the left side of the screen, enter the appropriate name in the Page Title box.

Keep in mind that entries are required for both tool titles.

3.  To determine the height of the window in which the external web page displays in the CLE (in pixels), make the appropriate selection from the Frame Height dropdown menu.

4.  To provide a link to a different web page, enter the appropriate URL in the URL box.

5.  To display the external web page in a separate tab/window, check the Open in new window? box.

6.  To have the tool open automatically to the full width of the CLE window, with the site tool menu on the left side of the screen minimized, check the Launch Maximum Width box. (Note: depending on how your CLE instance has been configured, this option may not be applicable.)   

7.  Click Update Options. Or, to exit without making any changes, click Cancel.
