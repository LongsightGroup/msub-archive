Importing an Assessment

To import an assessment, access the Tests & Quizzes tool and follow these steps:

1.  On the Tests & Quizzes home page, click the Import button. The CLE displays the Import Assessment screen.

2.  Click the Browse button and select the appropriate file from your computer.

3.  Choose your file type. If the file you’re importing was originally exported in the CLE, select Export from this (or other Sakai) system. If the file is an export from Respondus, select Export from Respondus.

4.  To import the assessment, click Import. The CLE displays the Tests & Quizzes home page, and the imported assessment is listed. Or, to return to that screen without importing the assessment, click Cancel.
