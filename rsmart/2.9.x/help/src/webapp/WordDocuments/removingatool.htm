Removing a Tool

To remove a tool from your site using Page Order, follow these steps:

1.  On the Site Editor home page, click on Page Order in the menu bar. Find the appropriate tool in the list of tools that is displayed.

2.  Click on the “remove” icon () for the tool. The CLE displays a window for removing the tool.

3.  In this window, click the OK button. A confirmation message is displayed.

You’ll see that the tool is immediately removed from both the Page Order list and the tools menu for the site. Clicking “Save” is not necessary, but clicking this button or the “Cancel” button will return you to the Site Editor home page.
