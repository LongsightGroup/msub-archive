Site Editor

Adding a Link to a Parent Site

Note: Depending on how your CLE instance is configured, the following functionality may not be available. 

In Site Editor, you can create a link between two sites you own, so users who are members of both sites can easily navigate between the sites.

When you create this navigation link, the site in which you've selected Site Editor is considered the "child" site. In this site, you can link to another "parent" site, which is any site you own or in which you have the appropriate site maintenance permissions.

Keep in mind that you will only be creating a navigation link between the sites. You can link multiple child sites to a parent site, but you may only select one parent site from a child site. The navigation links that a user sees  depend on whether the user is in a child or parent site:

•      A child site will include a breadcrumb trail with a link to the parent site.

•      A parent site will include all child site links in the tools menu.

To add a link to a parent site, follow these steps:

1.  From the Site Editor home page in the child site, click on Link to Parent Site in the menu bar.

2.  From the dropdown menu, select the appropriate parent site:

3.  Click the Set Link button. The CLE displays the Site Editor home page.

On the site's home page, users will see a breadcrumb trail displayed in the upper left corner of the screen, above the tools menu:

Clicking on the parent site link will take users directly to that site. In the parent site, the child site will be displayed as a link in the tools menu.

To remove the link to a parent site, select Site Editor again and click on Link to Parent Site in the menu bar. Then, click the Remove Link button.
