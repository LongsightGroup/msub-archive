Setting up Your Gradebook

The Gradebook tool allows you to customize your gradebook in these ways:

•      Specify whether grades in your gradebook are to be entered and displayed as points or as percentages.

•      Specify whether graders have the option to release grades to students for a graded item or prevent item-by-item grades from being released.

Note: This tool also allows you to control whether students can see their course grades or not. For information on setting this option, see Setting Grade Display and Conversion Options.

•      If desired, create categories for gradebook items; assign percentages to the categories in order to weight them and the items they contain; opt to drop a specified number of items with either the highest or lowest scores in a category, so the items are not included in course grade calculation; or opt to keep a specified number of items with the highest scores in a category, so only those items are included in course grade calculation.  

To customize your gradebook in any of the above ways, access the Gradebook tool and follow these steps:

1.  From the Gradebook home page, click Gradebook Setup in the menu bar. The CLE displays the Gradebook Setup screen.

2.  In the Grade Entry section, click a radio button to indicate whether grades are to be entered and displayed as point values or percentages.

3.  In the Gradebook Items Display section, choose one of these options:

•      To indicate that no gradebook items may be released to students, clear the box for Display released Gradebook Items to students.

•      To indicate that gradebook items may be released to students, check the box for Display released Gradebook Items to students.

Note: Checking this box does not automatically cause any grades to be released. Instead, it presents the option for you to release grades for any given graded item whenever you decide to do so. To release an item, you must edit the individual item as indicated in the table in Creating a Gradebook Item.

4.  In the Categories & Weighting section, specify whether you want to use categories or not and whether you want to weight categories.

•      If you check No Categories, proceed to step 6.

•      If you check Categories Only, the CLE displays the first row of a category table.

Enter a category name. To add more categories, click Add a Category and enter another name. If you want to drop items with the highest or lowest scores, or keep items with the highest scores, proceed to the next step. Otherwise, proceed to step 6.

•      If you check Categories & Weighting, the CLE displays the first row of a category and weighting table.

Enter a name for the first category in the Category column and specify the relative weight of this category in the % column. To add more weighted categories, click Add a Category and enter another category name and percentage. The CLE displays a running total of the percentages you have added. Once you have added all categories, be sure the percentages total 100%. If you want to drop items with the highest or lowest scores, or keep items with the highest scores, proceed to the next step. Otherwise, proceed to step 6.

5.  If you have set up your gradebook for categories, you’ll have the option to drop items with the highest or lowest scores, or keep items with the highest scores, in your categories. Check these settings as appropriate. For each setting selected, the CLE displays an additional column.

 

In each column, you’ll see a box where you can specific the number of items you want to drop or keep. For each category, enter a value in the appropriate box. Depending on your settings, a certain number of items with the highest or lowest scores in a category will not be included in the course grade calculation, or only a certain number of items with the highest scores will be included in this calculation.

Note: You can opt to drop items with the highest and lowest scores from a category, but you cannot drop items and keep items simultaneously. If you enter a “drop” value for a category, the “keep” field will be grayed out, and vice versa. Also, be sure that you assign the same maximum points to all items in a category if you want to enable the “drop” and “keep” options.    

6.  To complete your work, click Save Changes. The CLE displays a confirmation message on the screen. Or, to exit without making changes, click Cancel. The CLE displays the Gradebook home page.
