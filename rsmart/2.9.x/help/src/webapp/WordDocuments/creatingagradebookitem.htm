Creating a Gradebook Item

To create a gradebook item in this tool, access the Gradebook and follow these steps:

1.  On the Gradebook home page, click Add Gradebook Item(s). The CLE displays the Add Gradebook Item(s) screen.

2.  Referring to the table below, enter information for the item.

Field Name

Entry

Title

Required. Enter a name for the gradebook item.

Gradebook Item Point Value

Required. If you are entering grades as points, enter a value greater than zero; decimals are permitted, with up to two decimal places allowed.

Due Date

Enter a date manually in mm/dd/yy format or click the  calendar icon to select a date; optional.

Category

A dropdown menu is displayed only if you have set up your gradebook for categories. Select a category from the menu. The default selection is Unassigned .

Release this item to Students

Check to display this item to students; clear to hide this item from them.

Include this item in course grade calculations

Check to include; do not check if you don’t want the course grade calculation to include this item.

 

3.  To add another item to the gradebook, click Add Another Gradebook Item and repeat step 3. (At this point, you may delete any item by clicking the red X to the right of its title.)

4.  To post this item or items for students, click Add Item(s). The CLE displays the Gradebook home page. The assignments list now includes the new assignment(s) and a confirmation message is displayed:

Or, to return to the Gradebook home page without creating this assignment, click Cancel.
