Mailtool

Setting Mailtool Permissions

To view and set permissions for the Mailtool, access the tool and follow these steps:

1.  If the Mailtool home page is not displayed, click the Reset button to display it.

2.  In the menu bar near the top of the screen, click Permissions. The CLE displays the Permissions screen.

3.  Select the appropriate permissions for each user role. You have the following options for each role:

 

Permission

Description

mailtool.admin

Users are allowed to choose default settings and set permissions. Typically granted to the instructor and teaching assistant roles. Note: if you grant this permission to students, they will not have access to permissions when using this tool.

mailtool.send

Users are allowed to send emails. Typically granted to all user roles.

To clear all permissions, click the Clear all link. To undo any changes you’ve made and remain on the Permissions page, click the Undo changes link.

4.  If you want to save your changes, click Save. If you do not want to save your changes, click Cancel. In either case, the CLE displays the home page.
