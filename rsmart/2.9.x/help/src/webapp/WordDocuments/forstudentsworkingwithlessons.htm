For Students: Working with Lessons

If you are a student, when you access the Lessons tool, the CLE displays the Lessons home page.

This screen lists the modules in the course, along with the start and end date for each. Each module is made up of one or more sections, each of which also has a start and end date. If your instructor allows students to print lessons, the CLE also displays a print icon for each module.

Note: You cannot see modules that have not started yet.
