Creating a Project Site from a Template

To create a project site using a template, access My Workspace and select the Site Setup tool. Then, follow these steps:

1.  From the Site Setup home page, click New in the menu bar. The CLE displays the Site Type page.

Note: If you do not see New in the Site Setup menu bar, you do not have permission to create new sites.

2.  On this page, click on the Create site from template radio button. The CLE displays the site titles of available templates and, if included in a template, a site description.

3.  From the dropdown menu, select the appropriate project site template. In the Site title box that displays, enter a title for the project site.

4.  To copy the list of site participants from the site template, check the box for  Copy users. By default, the Copy content box is checked. If you do not want to copy tool content from the site template, uncheck this box.

5.  If you want the site to be published once you finish creating it, check the Publish Site box. If you don’t check this box, the site will be unpublished once you create it.

6.  Click Continue.

The CLE then displays the Site Setup home page, with your new project site included in the list of sites.
