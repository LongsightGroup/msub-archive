Inserting a Link to an Item in Resources

To insert a link to a content item in Resources, follow these steps:

1.  In the rich text editor, highlight the text where you want to insert the link.

2.  Click the Link icon (). The Link window displays.

3.  Click the Browse Server button. The Server Browser window opens.

  

In this window, you can select a content item in your current site or in any other site you own.

Tip: From the Server Browser window, you may also add a new folder or upload a new file to Resources by clicking the New Folder icon () or the Upload File icon ().

4.  To select an item, click on its name. The Server Browser window closes.

5.  In the Link window, optionally click on the Target tab to change how the link opens, or click on the Advanced tab for additional link configuration options. (Note: most of these options require advanced HTML experience.) 

6.  Click OK to insert the link.
