Navigating with the User Menu

Depending on how your CLE instance is configured, you may see a user icon in the upper right corner of the screen.

If you click on this icon, the user menu will display. This menu provides access to certain tools in My Workspace.

From this menu, you can:

•      Click on your user name to access the Profile tool.

•      Click on Preferences to access the Preferences tool.

•      Click on +Add New Site to access the Site Setup tool (note: you will need the appropriate permission to create a site.)

You can also click on Tutorial in this menu. In the popup window that displays, you’ll find a brief tutorial about the basic features and layout of the CLE.

Note: Your institution may have configured the CLE so that some or all user menu selections do not display.
