Viewing Grades, by Student 

For any student, you can view the grade for each item to date and the student's cumulative points. To do so, access the Gradebook tool and follow these steps:

1.  If the Gradebook home page is not displayed, click the Reset button.

2.  In the menu bar near the top of the screen, click the All Grades button. The CLE displays the All Grades screen.

On this screen, you’ll see a list of all students in the course, along with a column for each item and, depending on how you’ve set up your gradebook, a column for each category. If your gradebook has been set up for weighted categories, the headers for category columns include category weight. You may need to scroll right to see all items and scroll down to see all students.

In the Course Grade column, you’ll see each student’s calculated course grade to date. If your gradebook is set up for categories, a category column displays each student’s calculated grade to date for that category.

Tip: If you want to specify which columns you see, click the Hide/Show Columns link. In the window that displays, check the box for the columns you want hidden. Then, click the Save button in the window.  

3.  To display greater detail for a student, as well as enter or edit grades for that student, click the student’s name. The CLE displays the Student Summary screen.

4.  Choose any of these options:

•      To see the student’s view of his or her grades, click the View of Grades link for the student in the upper right. To return to the Student Summary screen for that student, click the Return to Instructor’s View of Grades link.

•      To save any changes to grades for the student, click Save Changes. Or, to clear any changes, click Clear Changes.

•      To save any changes to grades for the current student and then move to the Student Summary screen for the previous student, click the Save and View Previous Student button.

•      To save any changes to grades for the current student and then move to the Student Summary screen for the next student, click the Save and View Next Student button.

•      To return to the All Grades screen, click the Return to All Grades button.
