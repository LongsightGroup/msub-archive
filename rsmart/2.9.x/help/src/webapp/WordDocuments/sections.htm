Sections

Note: If your CLE is integrated with a student information system that keeps track of officially defined sections, sections may be created for your site automatically.

The Sections tool provides a way for instructors to manage multiple sections of a course in a single course site. If you are an instructor, this tool can help you manage a course that includes lectures, labs, discussions, studio work, and/or recitations.

Using this tool, you may set up the various sections of a course. For each section, you may:

•      Specify basic information, including day, time, room, current enrollment, available slots, and maximum enrollment, as well as the category (lecture, lab, recitation, etc.) for this section. If desired you may specify multiple meeting places and times for a section.

•      Assign students to the section or allow students to choose the section they prefer. Each student can belong to only one section in each section category.

•      Assign one or more teaching assistants.

•      Manage content (and/or allow TAs to manage content) in all section-aware tools (Announcements, Calendar, Assignments, Resources, Gradebook, Message Center, and Tests & Quizzes). This capability allows you to send an announcement to participants in a particular section, post a calendar entry for the section, grade online assignments for it, etc.

In addition, you may “close” the Sections tool until a certain date, so students won’t see the tool displayed in the site navigation menu before this date. Depending on how your instance of the CLE has been configured, you may also have access to an “auto create sections” feature, which lets you add sections to which the CLE randomly assigns students.
