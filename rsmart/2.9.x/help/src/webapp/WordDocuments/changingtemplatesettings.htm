Changing Template Settings

In the Forums tool, a template determines the default settings for each new forum and topic that is created. You can always override these settings when creating or editing a forum or topic. But if most of your forums and topics will have identical settings, you may want to select the settings in the template instead.

To view and change template settings, access the Forums tool and follow these steps:

1.  On the Forums home page, click on Template Settings in the menu bar.

The CLE displays the Default Settings Template screen.

2.  Select the appropriate Forum Posting settings:

a.  Click the Yes radio button for Moderate topics in forum if you want this setting selected by default for all newly created forums and topics. In the Permissions section for a forum or topic, you may then specify which site roles have permission to moderate messages (the permission will be inactive if you do not select this setting). Note that you may uncheck the setting for any forum or topic.

b.  Click the Yes radio button for Require users to post before reading if you want this setting selected by default for all newly created forums and topics. Note that you may uncheck the setting for any forum/topic. If this setting is selected for a topic, users must post a message to view other posts for the topic. 3.  Select the appropriate Availability setting:

•      Click on the radio button for Show Immediately if you want all forums/topics that are open to be available to users.

•      If you want to set a specific date for when all existing open forums/topics can become available, and/or a date for then whey are no longer available, click on the radio button for Specify dates to open (show) and/or close (hide) and provide the appropriate dates. When you set a date that makes open forums/topics unavailable, the CLE displays an icon in the tool menu bar, indicating restricted availability mode:

Note: If you set the template to be in restricted availability mode, students will not have access to any forums/topics, even if you set some forums/topics to be open.          

4.  Check the Yes radio button for Automatically Mark All Messages in a Thread as Read if you want this setting selected by default for all newly created forums and topics. If this setting is selected for a topic, all messages in a thread will be automatically marked as read when a user displays them. If the setting is not selected, a user must manually mark each message as read.

5.  In the Permissions section, change permissions for a site role or group/section, as instructed in Changing Permissions in the Settings Template.

6.  To save your changes, click the Save Settings button. The CLE saves your settings and permissions and the Forums home page is once again displayed.
