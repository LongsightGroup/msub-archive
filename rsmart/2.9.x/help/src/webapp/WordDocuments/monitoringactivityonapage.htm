Monitoring Activity on a Page

To monitor changes to a page, you view its history. To view page history, access the wiki page you want to monitor and click the History button on the menu bar.

The CLE displays the History screen for this page.

This screen provides a list of each version of the page, along with the name of the participant who changed it and the date it was changed.

•      To see any version of the page, click on its version number.

•      To compare the contents of any version with the current version, click the  Current link in the “Compare to” column .

•      To compare the contents of any version with the version previous to it, click the Previous link in the “Compare to” column.

•      To revert to a previous version of the page, click the Revert to this version link in the “Revert” column.

Tip: If you have accidentally deleted content from your page, you can use the “revert to this version” option to revert to an earlier, correct version.
