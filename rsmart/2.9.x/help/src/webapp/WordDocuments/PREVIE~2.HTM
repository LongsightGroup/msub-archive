Previewing and Publishing a Wizard

Publishing a wizard makes it available to site participants for use.

Warning: After a wizard has been published, you cannot add, move, reorder, or delete any of the categories or forms in it. For this reason, we strongly recommend that you publish a wizard only after previewing it and verifying that it contains the components you want in the order you want them.

To preview the wizard, access the Wizards tool and follow these steps:

1.  If the CLE is not displaying the Wizards home page, click the Reset button.

2.  Find the wizard you want to change, and click Preview. The CLE changes the entry in the Published? column to Preview. It also changes the Preview link to Publish.

3.  Click the name of this wizard. The CLE displays the wizard in preview mode.

4.  Review the material on this screen and click all links to review materials in all categories and pages.

5.  Click Return to Wizards. The CLE displays the Wizards home page.

6.  When you have determined that the wizard is exactly as you want it, publish it by clicking the Publish link for it. The CLE publishes the wizard and redisplays the home page.
