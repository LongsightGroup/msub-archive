Performing Common Tasks

Using the Help Tool

The Help tool allows you to access comprehensive documentation on using the CLE, either by searching for keywords or by selecting topics from a table of contents. The tool is typically available in all sites in the CLE.

You can access Help in one of two ways:

•      Select the Help tool in a site’s tool menu.

•      When using any tool in the CLE, click on the contextual Help icon () that is displayed to the right of the tool name.

In either case, the CLE displays the Help tool in a separate window:

This window consists of three panes, all of which can be resized:

•      Search

•      Table of contents

•      Content area

If you select the Help tool from a site’s tool menu, a “Welcome to Help” page displays in the content area. If you click on the contextual Help icon () while using a particular tool, you’ll see the overview for that tool in the content area.

To search for information in Help, enter one or more keywords in the Search Help box and click Search. You’ll see a list of topics that contain your keyword(s):

To view information on a topic, click its name in the results list. The CLE displays content for the topic in the content area. It also displays the location of this topic in the table of contents.

The table of contents displays Help topics in alphabetical order. Except for the first two topics, which provide general information about the CLE, topics are organized by tool. You have two options for using the table of contents:

•      To expand the list of subtopics for a topic in the table of contents, click the topic name.

•      To view content for a subtopic in the content area, click the subtopic name.
