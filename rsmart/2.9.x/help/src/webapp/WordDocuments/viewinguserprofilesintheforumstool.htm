Viewing User Profiles in the Forums Tool

Depending on how your instance of the CLE has been configured, you may see user profile images displayed with posted messages in the Forums tool:

You’ll see profile images in the list of messages in a conversation, as well as when you click on the subject of a message to display only that message. (For instructions on accessing messages, see Reading Messages in a Conversation.) 

If you click on a user’s name or profile image, the CLE displays a popup window that may contain more profile information:

In this window, click on View full profile to access the user’s complete profile in the Profile tool. You can also request a connection with the user by clicking on Add as a connection. (If you have a connection, you’ll see a Remove connection link.)

To close this window, click the “x” in the upper right corner.

Note: The profile information you see for a user will depend on the privacy and preferences settings the user has selected.
