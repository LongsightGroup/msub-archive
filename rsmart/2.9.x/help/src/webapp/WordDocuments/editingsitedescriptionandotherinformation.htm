Site Editor

Editing Site Description and Other Information

With Site Editor, you may change the site description and other information that is displayed on the site home page, as well as change other settings and information for the site. To do so, access the Site Editor tool and follow these steps:

1.  On the Site Editor home page, click Edit Site Information in the menu bar. The CLE displays the Revising Site Information page.

a.  Depending on how your CLE instance has been configured and the type of site you’re editing, you may be able to edit your site title. Edit or enter text as appropriate in the Site Title box. 

b.  Depending on how your CLE instance has been configured, you may be able to provide or edit a site URL. Enter the URL in the Site URL box. This URL can be a more user-friendly URL than the default, which includes the site ID.

c.  From the Site Language menu, optionally select a different language for your site. By default, the site will use the language for which your CLE instance has been configured.

d.  Under the Description header, use the rich text editor to add or edit course information that will be displayed on the site’s home page.

Tip: You may also change a site description by clicking the “edit” icon () in the Site Information section of the site's home page.

e.  In the Short Description box, add or edit a shorter site description (maximum 80 characters), which users will see in the list of public sites.

f.   Depending on the type of site you’re editing, you may provide optional settings for customizing the appearance of your site:

•      From the Appearance (Theme) menu in a course site, you may optionally select a different “theme,” or graphics, for your site. (Note: the selections you see in this menu depend on how your CLE instance has been configured.)

•      In the Icon URL box in a project or portfolio site, you may optionally enter a URL for an image that will display above your site’s tool menu.   

g.  In the Site Contact Name box, edit the first and last name of the person responsible for the site.

h.  In the Site Contact Email box, edit the email address of the person responsible for the site

2.  To proceed with the changes, click Continue. The CLE displays a confirmation page.

3.  To confirm the changes and return to the Site Editor home page, click Finish.

The CLE once again displays the Site Editor home page.
